Events

    • List of jQuery Events
    • Click (click) - runs on a single click
    • Double Click (dblclick) - runs on a double click
    • mouseenter and mouseleave - runs when the mouse is hovered and removed
    • mousedown and mouseup - runs when any mouse button is clicked or released on an element
    • hover() - runs when the mouse is hovered and removed from an element
    • focus() - changes the input field when clicked into (focused on)
    • blur() - changes the input field when it is clicked out of (loses focus)
    • on() - attaches one or more event handlers for the selected elements.

Effects

    • List of jQuery Effects
    • hide() - hides an element
    • fade() - fades in and out of visibility
    • show() - displays a hidden element
    • toggle() - Toggles between the hide() and show() methods
    • slideDown() - Slides (and displays) the selected hidden element down on the page
    • slideUp() - Slides (and hides) the selected displayed element up on the page
    • slideToggle() - Toggles between slideUp() and slideDown() for the selected elements
    • animate() - create custom animations
    • stop() - stopping animations
    • callback functions
