hover()

Takes two functions and is a combination of the mouseenter() and mouseleave() methods.

The first function is executed when the mouse enters the HTML element, and the second function is executed when the mouse leaves the HTML element:

<p id="p1">This is a sentence</p>

$("#p1").hover(function(){
  $(this).css('color', 'yellow');
},
function(){
  $(this).css('color', 'white');
});

Example

Hover over me to change my color to yellow. Remove the mouse away to change my color to white.
