The 'mouseenter' event

When the user hovers their mouse over an element, the function is run.

<p id="p1">This is a sentence</p>

$('p#p1').on("mouseenter", function() {
  $('p#p1').hide();
});

mouseleave()

This executes when the mouse leaves the HTML element.

<p id="p2">This is a sentence</p>

$('p#p2').mouseenter(function() {
  $(this).css('color', 'yellow');
});

$("p#p2").mouseleave(function() {
  $(this).css('color', 'red');
});
