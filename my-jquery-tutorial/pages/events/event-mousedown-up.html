mousedown

This fires when any mouse button is pressed down (clicked) on an element.

<p id="p1">This is a sentence</p>

$('p#p1').on("mousedown", function() {
  $('p#p1').css();
});

Example

Press the mouse down on me to change my color.

mouseup

This fires when any mouse button is released on an element.

<p id="p2">This is a sentence</p>

$("#p2").mousedown(function() {
  $(this).css('color', 'red');
})
$("#p2").mouseup(function() {
  $(this).css('color', 'yellow');
})

Example

Press the mouse down on me to change my color to red. Release to change my color to yellow.
