The 'dblclick' Event

When the user Double clicks the button, the paragraph disappears.

The #firstBtn element has an event happen 'on' it.

The 'click' event is the event that happens 'on' the #firstBtn element.

When the 'click' event occurs, the function() is run.

The function looks for the paragraph with an ID of 'statement' and uses hide() to hide it.

<input type="button" value="submit" id="firstBtn"/>
<p id="statement">This is a sentence</p>

$('#firstBtn').on("dblclick", function() {
  //code to be executed when the button is clicked
  $('p#statement').hide();
});
