Retrieve HTML of an element

<p id="test">This is some <b>bold</b> text in a paragraph.</p>
<button id="btn2">Show HTML</button>

$("#btn2").click(function(){
  alert("HTML: " + $("#test").html());
});

Set HTML of an element

<p id="test2">This is another paragraph.</p>
<button id="btn2">Set HTML</button>

  $("#btn2").click(function(){
    $("#test2").html("Hello world!");
  });

Changing HTML using the html() method

The html() method changes the HTML of a specified element

It also adds a CSS class to the element.

<p class="dope"> You'll learn how to replace me</p>
<input type="button" value="Replace" id="btn_main" />

$('#btn_main').on('click', function() {
  $('.dope').html('<strong class="html-change-class">l; HI! The HTML just changed. </strong>');
});

A Callback Function for html()

html() also comes with a callback function. The callback function has two parameters:

  • the index of the current element in the list of elements selected
  • and the original (old) value

You then return the string you wish to use as the new value from the function.

<p id="testcallback2">This is another <b>bold</b> paragraph.</p>
    <button id="btncallback2">Show Old/New HTML</button>

    $("#btncallback2").click(function(){
       $("#testcallback2").html(function(i, origText){
         return "Old html: " + origText + " New html: Hello world! (index: " + i + ")";
       });
     });
