Retrieve the text of an element

This retrieves the text of a paragraph

<p id="test">This is some <b>bold</b> text in a paragraph.</p>
<button id="btn1">Show Text</button>

$("#btn1").click(function(){
  alert("Text: " + $("#test").text());
});

Set the text of an element

This sets the text of a paragraph

<p id="testset1">This is a paragraph.</p>
<button id="btnset1">Set Text</button>

$("#btnset1").click(function(){
  $("#testset1").text("Hello world!");
});

Replacing text using the text() method

The text() method changes the text of a specified element

<h2 class="dope"> You'll learn how to replace me
<input type="button" value="Replace" id="btn_main" />

$('#btn_main').on('click', function() {
  //use the text() method to replace the existing text
  $('.dope').text('This is new text.')
});

A Callback Function for text()

text() also comes with a callback function. The callback function has two parameters:

  • the index of the current element in the list of elements selected
  • and the original (old) value

You then return the string you wish to use as the new value from the function.

<p id="testcallback1">This is a <b>bold</b> paragraph.</p>
<button id="btncallback1">Show Old/New Text</button>

$("#btncallback1").click(function(){
  $("#testcallback1").text(function(i, origText){
    return "Old text: " + origText + " New text: Hello world! (index: " + i + ")";
  });
});
