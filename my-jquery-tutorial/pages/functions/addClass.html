addClass()

The addClass function adds a CSS class to an element.

This targets all paragraph tags.

It then assigns those tags the action of addClass().

Finally, specify the CSS class name you want to add. This example adds 2 classes.

.timestwo {
  font-size: 2em;
}

.colorful {
  color: cyan;
}

<div>
  <p>This is a sentence</p>
  <p>Here's another one </p>
  <h4> h4 tag not changed </h4>
</div>

<script src="js/jquery-3.1.1.min.js"></script>

  $("p").addClass("colorful timestwo");
</script>
