removeClass()

The addClass function adds a CSS class to an element.

This targets all paragraph tags.

It then assigns those tags the action of addClass().

Specify the CSS class name you want to add. This example adds 2 classes.

The removeClass() is then run which removes the 'timestwo' class.

The result is only the color changing.

.timestwo {
  font-size: 2em;
}

.colorful {
  color: white;
}

<div>
  <p>This is a sentence</p>
  <p class="super"> Here's another one </p>
  <h4 class="super"> h4 tag not changed </h4>
</div>

<script src="js/jquery-3.1.1.min.js"></script>

<script>
  $("p").addClass("colorful timestwo");
  $("p").removeClass("timestwo");
</script>
