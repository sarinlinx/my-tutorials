The 'hide' effect

This example shows two paragraphs.

The jQuery code targets all paragraphs in this document, then assigns them the action of 'hide'.

the hide() function automatically hides the elements, so you will not see them when the page loads.

<div>
  <p class="p1">This is a sentence</p>
  <p class="super p1"> Here's another one </p>
  <h4 class="super"> Only the h4 tag shows </h4>
</div>


<script>
  $("p.p1").hide();
</script>

Click button to hide

Clicking the button hides the paragraph with an ID

<p>If you click on the "Hide" button, I will disappear.</p>
<button id="hide">Hide</button>

$("#hide").click(function(){
  $("p").hide();
});

Example

Adding a time parameter

The optional speed parameter specifies the speed of the hiding/showing, and can take the following values: "slow", "fast", or milliseconds.

<p>If you click on the "Hide" button, I will disappear.</p>
        <button id="hide">Hide</button>

        $("#hide").click(function() {
            $("p#p2").hide(1000);
        });
        $("#show").click(function() {
            $("p#p2").show("slow");
        });
