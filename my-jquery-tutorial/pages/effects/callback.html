Callback Functions

A callback function is executed after the current effect is finished.

This is important because JavaScript statements are executed line by line. However, with effects, the next line of code can be run even though the effect is not finished. This can create errors.

Example without a callback

Since there is no callback, the alert box is triggered before the paragraph has time to finish hiding

<button id="btn1">Hide</button>
<p id="p1">This is a paragraph with little content.</p>

$("button#btn1").click(function(){
  $("p#p1").hide(3000);
  alert("The paragraph is now hidden");
});

Example with a callback

Since a callback function is used, the paragraph is allowed to completely hide itself first. Only then does the alert box display.

<button id="btn2">Hide</button>
<p id="p2">This is a paragraph with little content.</p>

$("button#btn2").click(function(){
 $("p#p2").hide("slow", function(){
   alert("The paragraph is now hidden");
 });
});
