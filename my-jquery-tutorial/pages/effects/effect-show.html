The 'show' effect

When the user clicks the link, the paragraph disappears and the yellow rectangle appears.

The #target element has an event happen 'on' it.

The 'click' event is the event that happens 'on' the #target element.

When the 'click' event occurs, the function() is run.

The function looks for the paragraph with an ID of 'statement' and uses hide() to hide it.

It then finds the ID of 'box' and uses show() to display it.

<a id="target" href="#"> Click Here </a>
<p id="statement"> This is a sentence </p>
<div id="box" style="width:500px; height:100px; border:1px solid #000; background-color:yellow; display:none;">
  This is a rectangle
  </div>

$('#target').on("click", function() {
  //code to be executed when the button is clicked
  $('p#statement').hide();
  $('div#box').show();
});

Adding a time parameter

The optional speed parameter specifies the speed of the hiding/showing, and can take the following values: "slow", "fast", or milliseconds.

<p>If you click on the "Hide" button, I will disappear.</p>
<button id="hide">Hide</button>

$("#hide").click(function() {
    $("p#p2").hide(1000);
});
$("#show").click(function() {
    $("p#p2").show("slow");
});
