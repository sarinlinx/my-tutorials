fadeIn()

fade in a hidden element

The optional callback parameter is a function to be executed after the fading completes.

<div id="div1" style="width:80px;height:80px;display:none;background-color:red;"></div><br>
<div id="div2" style="width:80px;height:80px;display:none;background-color:green;"></div><br>
<div id="div3" style="width:80px;height:80px;display:none;background-color:yellow;"></div>

$("button#btn1").click(function(){
  $("#div1").fadeIn();
  $("#div2").fadeIn("slow");
  $("#div3").fadeIn(3000);
});

Example

fadeToggle()

toggles between the fadeIn() and fadeOut()

The optional callback parameter is a function to be executed after the fading completes.

<div id="div1toggle" style="width:80px;height:80px;display:none;background-color:red;"></div><br>
<div id="divtoggle" style="width:80px;height:80px;display:none;background-color:green;"></div><br>
<div id="div3toggle" style="width:80px;height:80px;display:none;background-color:yellow;"></div>

$("button#btn3").click(function() {
  $("#div1toggle").fadeToggle();
  $("#div2toggle").fadeToggle("slow");
  $("#div3toggle").fadeToggle(3000);
});

Example
