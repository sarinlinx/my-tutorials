Using the ajax method

Executed after current event has completed

Wait for an element to load before doing anything else

To tell JS to wait, use a callback function

<div class="main5">
</div>

<button id="btn5"> Load Messages </button>

$("button#btn5").on("click", function() {
    var url = "https://raw.githubusercontent.com/imtiazahmad007/resources/master/sample_table.html";
//first var is the URL
  $.ajax(url, {
    dataType: "html",//next add the data type
    method: 'GET',//method
    //only if call is successful load response from the URL
    success: function(response){
      $("div.main5").html(response);
    }//no comma needed since it's last
  })
});

If there was an error during the Ajax call

Since the URL is broken, an alert box displays notifying you that the ajax call failed.

<div class="main6">
</div>

<button id="btn6"> Load Messages </button>

$("button#btn6").on("click", function() {
  //load method
  var url = "https://raw.githubusercontent.com/imtiazahmad007/resources/master/sample_table_BROKEN.html";
  //first var is the URL

  $.ajax(url, {
    dataType: "html",
    method: 'GET',
    success: function(response){
      $("div.main6").html(response);
    },
    //if there's an error
    //the param order matters, but you can name them anything
    error: function(request, errorType, errorMsg){
      alert("There was an issue with the Ajax call: " + errorMsg);
    }
  });
});

$.ajax(root, {
  dataType: "json",
  method: 'GET',
  success: function(response){
    //get specific data from a specific Object Array
    //use array notation to specify

    var postId = response[0].postId;
    var email = response[0].email;
    //define html node
    var $info = $("
");
    //populate paragraph tags
    $info.text("post id is: " + postId + " email is: " + email);
    //print to web page
    $("div.main9").append($info);
    console.log(response);

  },
  error: function(request, errorType, errorMsg){
    alert("There was an issue with the Ajax call: " + errorMsg);
  },
  //data function. This passes in the param to the URL
  data: {"postId": 1}

Passing data in the code

Add a variable to store and object.

The object contains the value of the HTML field

Add the variable name to the .get() function

When you click the link, the page updates with GET hihihi since that's the value of the input field.

<div id="output">Output</div>
<div class="one"><a href="#" id="myClicker">Click me</a>Some Content</div>
<input type="text" value="hihihi" id="myA">

<script>
$(document).ready(function() {
  $("#myClicker2").click(function(e) {
      e.preventDefault(); //prevent default link action
      // use object oriented method to pick up data   
      // create var and assign to object with data to pass over
      var newdata = {
        //get the value
        name: $("#myA").val()
      }
      // add newdata var as content your're passing
      $.get("get-method-php.php", newdata, function(data) {
          //output any returned data
          $("#output2").html(data);
      })
  })
})
</script>

AJAX method

Allows you to control content more than post.

Default method is get, but you can change with type

<div id="output">Output</div>
              <div class="one"><a href="#" id="myClicker">Click me</a>Some Content</div>
              <input type="text" value="hihihi" id="myA">
              
<script>
$(document).ready(function() {
  $("#myClicker2").click(function(e) {
      e.preventDefault(); //prevent default link action  
      var newdata = {                       
        name: $("#myA").val()
      }
    
    $.ajax({
      url:"ajax.php",
      type:"POST",
      data: newdata,
      //if success is successful, fire function
      success:function(data,status,xhr){
      },
      //otherwise run error function
      error: function(data,status,xhr){
      console.log(xhr); }

  })
})
</script>
