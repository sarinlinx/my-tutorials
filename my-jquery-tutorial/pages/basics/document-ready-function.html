Document Ready Function

You should put your calls to jQuery at the bottom of your HTML. However, if you don't you can still use the Document Ready Function to only run it once the document has loaded.

Wrapping your jquery code in the Document Ready Function makes sure that jQuery is loaded before any of your code runs.

Target the document

Call the ready() function on it

Then pass in the function

Long version

$(document).ready(function(){
  //any code in here runs after the document has been completely loaded

});

Short version

$(function(){
  //any code in here runs after the document has been completely loaded

});
