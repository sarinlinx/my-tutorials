Basic jQuery statement

$ sign

This tells the browser interpreter that the code that follows is jQuery.

The $ sign is a shorthand for writing 'jQuery'. For example, the following examples are exactly the same.

jQuery("img").hide();
window.$("img").hide();
window.jQuery("img").hide();

The target

The CSS element/s to be acted on.

Always in parenthesis and quotes.

View the targeting page for more examples.

  • targeting elements

The dot separator

This separates the CSS element from the action this statement will take.

The Action (function)

The action is what will happen to the CSS element selected.

It is written as the name of the action followed by ().

In this case, all img tags will be hidden.

Closing semicolon

Close the statement with a semicolon.
