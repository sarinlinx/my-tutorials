Manually Installing

Download the files

Download the jQuery files at:

  • jquery.com/download

On that page, download the compressed version link.

  • code.jquery.com/jquery-3.4.1.min.js

Add the script to your website

Add a line to link the jQuery file just above the closing <body> tag.

<script src="../_js/jquery-3.4.1.min.js"></script>

Order matters! Make sure the call to the jquery file is before any other js files that require jQuery to function.

Linking to a CDN

You can also enable jQuery by linking to a CDN. One big advantage of using the hosted jQuery from Google or Microsoft is that many users already have downloaded jQuery from Google or Microsoft when visiting another site. As a result, it will be loaded from cache when they visit your site, which leads to faster loading time. Also, most CDN's will make sure that once a user requests a file from it, it will be served from the server closest to them, which also leads to faster loading time.

Linking from Google

<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
</head>

Linking from Microsoft

<head>
<script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.4.1.min.js"></script>
</head>
