The 'this' keyword

The 'this' keyword allows you to pinpoint the exact element the user is interacting with.

For example, if you have two buttons, the 'this' keyword will only apply to the button that is clicked on my the user, not the other button.

This example shows two buttons and an event inside another event.

<button type="button">Button 1</button>
<button type="button">Button 2</button>


//target the button element
$('button').on('click', function() {
  //Since you've already targeted the button element, you can use
  //the 'this' keyword in the function to refer to it
  $(this).addClass('beautify');

  //create another event within this event
  //This event only runs when the parent event is run
  //The 'this' keyword applies to the single element you're interacting with
  //If there's more than 1 button, only the one you click is changed
  $(this).on('dblclick', function() {
    $(this).addClass('blue_border');
  });
});

Example

Click a button once to change its CSS.

Double click it to change it again.
