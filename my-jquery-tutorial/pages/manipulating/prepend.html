prepend()

This prepends (adds before) text to the beginning of an existing paragraph

<p>This is a paragraph.</p>
<button id="btn1">Prepend text</button>

$("#btn1").click(function(){
  $("p#p1").prepend("<b>Prepended text</b>. ");
});

prepend()

This prepends a list item to the beginning of a list

<ol id="ol1">
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ol>

<button id="btn2">Prepend list item</button>

$("#btn2").click(function(){
  $("ol#ol1").prepend("<li>Prepended item</li>");
});

Example

  1. List item 1
  2. List item 2
  3. List item 3
