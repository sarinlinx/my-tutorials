Using css() to add CSS styles

Adding inline styles to a single element

This changes the text in a paragraph to red.

<p class="boot"> Computers are cool </p>

$('p.boot').css('color', 'red');

Adding inline styles to a single element

This changes multiple CSS valaues use a map of key value pairs.

<p class="boot"> Computers are cool </p>

$('p.boot2').css({
    'color': 'white',
    'background-color': 'grey',
    'font-family': 'arial, helvetica, sans-serif'
});
