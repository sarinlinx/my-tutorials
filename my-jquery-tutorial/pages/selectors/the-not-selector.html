The NOT != selector

Use the != NOT selector to check if a given value does NOT belong

This selects all paragraphs that do NOT contain the class named 'className'

<h3>Example</h3>
<p class="className">This paragraph has a class named 'className'</p>
<p>This one does not</p>

<button id="btn1">Click me</button>

$("#btn1").click(function() {
  $("p[class!='className']").css({
      'border': '1px solid red'
  });
});
