Remove element using remove()

Target with attr(), then add the second class in the same parenthesis.

<p id="title">Click once to add CSS. Double click to remove</p>

        $("p#title").on("click", function() {
              $(this).attr("class", "pretty border");
          });

          $("p#remove_css").on("dblclick", function(){
            $(this).remove();
        });

Remove attribute using removeAttr()

You can also remove a specific attribute using removeAttr()

Double Click to remove the src attribute.

<img id="the_img"  src="img/man_changing_tire.jpg" width="20%" height="20%">

        $('img#the_img').on('dblclick', function(){
          $(this).removeAttr("src");
        });
