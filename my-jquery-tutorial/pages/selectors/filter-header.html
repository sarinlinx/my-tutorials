:header

Target all headers on the page

<h1>Header 1</h1>
<h2>Header 2</h2>
<h3>Header 3</h3>
<p>Paragraph</p>
<h4>Header 4</h4>

$("#btn1").click(function() {
  $(":header").css({
    'border': '1px solid red'
  });
});

:has(:header)

Target only divs with a class of myDiv that contain a header

<h1>Header 1</h1>
<div class="myDiv">
  <h2>Header 2</h2>
  <h3>Header 3</h3>
</div>
<p>Paragraph</p>
<h4>Header 4</h4>

$("#btn1").click(function() {
$("div.myDiv:has(:header)").css({
'border': '1px solid red'
});
});
