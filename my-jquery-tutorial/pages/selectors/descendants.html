Descendants

A descendant is a child, grandchild, great-grandchild, and so on.

Two useful jQuery methods for traversing down the DOM tree are:

  • children()
  • find()

children()

children()

The children() method returns all direct children of the selected element.

This method only traverses a single level down the DOM tree.

The following example returns all paragraphs that are direct children of each the div element with an ID of children1:

<div id="children1" style="width:500px;"><div> (current element)
  <p> (child)
    <span>(grandchild)</span>
  </p>
  <p> (child)
    <span> (grandchild)</span>
  </p>
</div>
<p> Outside the div</p>

 $("#children1").children().css({"color": "red", "border": "2px solid red"});

children(param)

You can also use an optional parameter to filter the search for children.

The following example returns all p elements with the class name "first", that are direct children of div:

<div id="children2" style="width:500px;">div (current element)
<p class="first">p (child)
  <span>span (grandchild)</span>
</p>
<p class="second">p (child)
  <span>span (grandchild)</span>
</p>
<p class="first">p (child)
  <span>span (grandchild)</span>
</p>
</div>

$("#children2").children("p.first").css({"color": "red", "border": "2px solid red"});

find()

The find() method returns descendant elements of the selected element, all the way down to the last descendant.

The following example returns all span elements that are descendants of an element with the ID of 'children3'

<div id="children3" style="width:500px;">div (current element)
  <p>p (child)
    <span>span (grandchild)</span>
  </p>
  <p>p (child)
    <span>span (grandchild)</span>
  </p>
</div>

<span>outside of div</span>

$("#children3").find("span").css({"color": "red", "border": "2px solid red"});

Target a nested list (direct descendants)

Target the direct descendents using the > sign.

This ensures only the immediate descendants of the element with an ID of steps are targeted.

<ul id="steps">
    <li>Step 1</li>
    <li>Step 2</li>
      <ul id="nested_steps">
        <li>nested item 1</li>
        <li>nested item 1</li>
      </ul>
</ul>

$('#steps > li').css('color', 'yellow');
