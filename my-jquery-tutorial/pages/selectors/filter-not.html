:not filter

This negates anything inside the selector.

This example targets the list items that do NOT have a class of 'important'.

<ul>
<li class="important">has a class</li>
<li>Does not have a class</li>
</ul>
<button id="btn1">Click me</button>

$("#btn1").click(function() {
  $("li:not(.important)").css({
    'border': '1px solid red'
  });
});

Target divs that do not contain a header

This reads as: target the divs that do not have a header

<div>
  <h4>h4 tag here</h4>
  <p>paragraph</p>
</div>
<div>
  <p>No headers</p>
</div>
<button id="btn2">Click me</button>

$("#btn2").click(function() {
  $("div:not(:has(:header))").css({
    'border': '1px solid red'
  });
});
