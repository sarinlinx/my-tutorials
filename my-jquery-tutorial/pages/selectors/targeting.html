Basic jQuery statement

The target

The CSS element/s to be acted on.

Always in parenthesis and quotes.

Targeting an element with a class

Use the element, followed by a period, followed by the class name.

<p class="myClass">stuff...</p>

$("p.myClass").addClass("colorful");

Targeting an element with an ID

Use the element, followed by a #, followed by the ID name.

<p id="myID">stuff...</p>

$("p#myID").addClass("colorful");

Targeting 2 elements with a Class

Use the element, followed by a ., followed by the class name.

Next, add a comma to separate the next element

Add the new element, followed by a ., followed by the class name.

<p class="myClass">stuff...</p>
<h4 id="myClass">stuff...</p>

$("p.myClass,h4.myClass").addClass("colorful");

Target all elements on a page with a class

Use the ., followed by the class name.

<p class="myClass">stuff...</p>

$(".myClass").addClass("colorful");

Target only the list elements in a list

Target the ID of the list, add a space, then select the li items.

<ul id="steps">
<li>Step 1</li>
<li>Step 2</li>
</ul>

$('#steps li').css('color', 'yellow');
