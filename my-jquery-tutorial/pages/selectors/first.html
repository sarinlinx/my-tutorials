Select the first type of element on a page

Use :first to select the first type of element on a page.

This example locates the first div on a page and hides it.

<h3>Example</h3>

<h2>This is a heading</h2>

<p>This is a paragraph.</p>
<div>
  <p>This is a paragraph in a div.</p>
</div>

<button id="btn1">Click me</button>

 $("button").click(function(){
  $("p:first").hide();
});

Select the first list item

This selects the first list item in a list.

<p>List 1:</p>
<ul>
  <li>Coffee</li>
  <li>Milk</li>
  <li>Tea</li>
</ul>

<p>List 2:</p>
<ul>
  <li>Coffee</li>
  <li>Milk</li>
  <li>Tea</li>
</ul>

$("#btn2").click(function(){
  $("ul li:first").css('color', 'red');
});

Select every first list item

Use :first-child to select every first list item on the page.

<h3>Example</h3>
<p>List 1:</p>
<ul>
    <li>Coffee</li>
    <li>Milk</li>
    <li>Tea</li>
</ul>

<p>List 2:</p>
<ul>
    <li>Coffee</li>
    <li>Milk</li>
    <li>Tea</li>
</ul>

<button id="btn3">Click me</button>

$("#btn3").click(function() {
  $("ul li:first-child").css('color', 'red');
});

Example

List 1:

  • Coffee
  • Milk
  • Tea

List 2:
