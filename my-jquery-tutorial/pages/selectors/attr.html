$("button#changeurl").click(function(){
    $("#w3schangeurl").attr("href", "https://www.w3schools.com/jquery/");
  });

Assign attribute/value pair using attr()

Use the attr() method to assign new attribute/value pairs to an element.

Hover over the list items with a class of 'important' to assign them the id=big.

Hover your mouse away to remove the class.


<ul id="steps">
    <li class="important">Park Vehicle Safely</li>
    <li>Retrieve Jack and attach to vehicle frame</li>
    <li>untighten tire bolts</li>
    <li class="important">Raise Vehicle by cranking jack</li>
    <li>Unscrew Bolts and Attach Replacement tire</li>
    <li class="important">Tighten the bolts</li>
    <li>Lower vehicle and tighten bolts completely</li>
</ul>

$("li.important").on("mouseover", function(){
    $(this).attr("id", "big");

    $(this).on("mouseleave", function(){
        $(this).attr("id", "small");
    });
});

Example

  • Park Vehicle Safely
  • Retrieve Jack and attach to vehicle frame
  • untighten tire bolts
  • Raise Vehicle by cranking jack
  • Unscrew Bolts and Attach Replacement tire
  • Tighten the bolts
  • Lower vehicle and tighten bolts completely

Add 2 classes to an element

Target with attr(), then add the second class in the same parenthesis.

<p id="title">Add 2 classes to me when clicked</p>

$("h1#title").on("click", function(){
    $(this).attr("class", "pretty border");
});

Assign CSS attributes in JS

This assigns CSS key values within the jQuery directly.

The multiple CSS values are chained together into a single attr() call.

NOTE: This only works on an image. Why?

<img id="the_img"  src="img/man_changing_tire.jpg" width="20%" height="20%">

$("img#the_img").on("click", function(){
    $(this).attr({"width": "40%",
                  "length": "40%"
                });
});
