Target all elements that contain a specific attribute

You can target all elements that contain a specific attribute using []

For example, to target all paragraphs that have a class assigned to them use p[class].

<p class="class1">I'm a paragraph with a class</p>
      <p class="class2">Me too!</p>
      <p>I don't have a class</p>

      $('p[class]').css({'border': '1px solid red'});

Target all elements that contain a specific attribute and name

This targets a specific attribute with a specific name

Notice how the paragraph with a class of 'class4' does not show.

You MUST USE double quotes to surround the element and class names. For example

    Yes
  • $("p[class='class4']").hide();
    • No
  • $('p[class='class4']').hide();

It will not work if you use single quotes.

<p class="class3">I'm a paragraph with a class</p>
      <p class="class4">Me too!</p>
      <p>I don't have a class</p>

      $("p[class='class4']").hide();

Target all elements that contain an attribute of target='_blank'

Selects all elements with a target attribute value equal to "_blank"

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
<p><a href="#" target="_blank">This link has a target="_blank" attribute</a></p>
<p><a href="#">This is a normal link</a></p>

<button id="btn1">Click me</button>

$("#btn1").click(function(){
  $("a[target='_blank']").css('color', 'red');
});
