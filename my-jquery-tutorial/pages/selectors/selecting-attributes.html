Selecting multiple attribute/value pairs in a single call

The attribute would be id= or class= and the value would be it's ID or Class name.

This single line selects multiple attributes/values and changes their CSS.

<p> This is not changed </p>
<p class="instruct_title">Incase of flat tire.</p>

<div id="closing_statement">
    <p>Make sure to put away the tools</p>
</div>

//Select multiple attributes in the same call
$('p.instruct_title, div#closing_statement').css('color', 'green');

Example

This is not changed

This class is changed to green.

This ID is changed to green.

Selecting multiple attribute/value pairs using .add

Using the .add method to select multiple key/values is faster for the computer to process.

<p> This is not changed </p>
<p class="instruct_title">Incase of flat tire.</p>

<div id="closing_statement">
    <p>Make sure to put away the tools</p>
</div>

//Select multiple attributes split us by .add
$("p.instruct_title").add("div#closing_statement").css("color", "green");

Example

This is not changed

This class is changed to green.

This ID is changed to green.
