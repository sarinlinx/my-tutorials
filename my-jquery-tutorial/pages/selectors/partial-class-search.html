Starts with

Partial class search using ^=

^= means 'starts with'

This locates all classes that start with a few letters.

<p class="testClass">p with class named 'testClass'</p>
<p class="testingClass">p with class named 'testingClass'</p>
<p class="classTest">p with class named 'classTest'</p>
<button id="btn1">Click me</button>

$("#btn1").click(function() {
  $("[class^='test']").css({
      'border': '1px solid red'
  });
});

Ends with

Partial class search using $=

$= means 'ends with'

This locates all classes that end the letters 'lass'.

<p class="testClass">p with class named 'testClass'</p>
<p class="testingClass">p with class named 'testingClass'</p>
<p class="classTest">p with class named 'classTest'</p>
<button id="btn2">Click me</button>

$("#btn1").click(function() {
$("[class$='lass']").css({
'border': '1px solid red'
});
});

Contains

Find the element that contains a specific class using ~=

~= means 'contains full name'

This locates all elements with the class of 'classTest'.

THE FULL class name must be searched for. This is not a partial search

<p class="testClass">p with class named 'testClass'</p>
<p class="testingClass">p with class named 'testingClass'</p>
<p class="classTest">p with class named 'classTest'</p>
<button id="btn3">Click me</button>

$("#btn1").click(function() {
$("[class~='classTest']").css({
'border': '1px solid red'
});
});

Locate any sequence of characters using *=

Find the element that contain a string of letters using *=

*= means 'contains string'

This locates all elements with the string of 'est'.

<p class="testClass">p with class named 'testClass'</p>
<p class="testingClass">p with class named 'testingClass'</p>
<p class="classTest">p with class named 'classTest'</p>
<button id="btn4">Click me</button>

$("#btn1").click(function() {
$("[class*='classTest']").css({
'border': '1px solid red'
});
});
