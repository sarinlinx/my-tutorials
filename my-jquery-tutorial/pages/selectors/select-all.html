The * symbol selects all elements on a page

The * symbol selects all elements on page.

If you use it to hide elements, the page goes to a blank white page.

<h3>Example</h3>

<h2>This is a heading</h2>

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

<button>Click me</button>

$("button").click(function(){
  $("*").hide();
});
