MongoDB

MongoDB is a NoSQL database.

Each databse can contain multiple collections (tables).

Each collection can contain multiple documents (rows).

Documents

This compares a document to a row in a relational database.

MongoDB uses a data type called BSON. This looks like JSON.

It allows multiple values in a field (unlike a relational database).

Also allows 'Embedded documents'.

Each document contains a unique ID which acts as a primary key of that document. It's automatically generated with the object ID data type each time there is a new document, and so we don't have to worry about it.
