Installing MongoDB

Windows

  1. Visit the downloads page. Download the community version.
  2. Download msi file.
  3. Run it to install
  4. Open PowerShell
  5. Navigate to C:\Program Files\MongoDB\Server\5.0\bin
  6. Run the server: .\mongod.exe
  7. This starts the server
  8. Open 2nd Powershell window in the same directory
  9. Run the file: .\mongo.exe. This opens the mongo shell.

    Type in db to view the test database

    > db
    test
  10. CTRL + C to exit

Run MongoDB from another directory

  1. Open SETTINGS
  2. Type in env
  3. Click 'Edit the system environment variables'.
  4. In popup, click 'Environment Variables...' button
  5. Next popup: In 'System Variables' list, click Path. Then EDIT
  6. Next popup:
  7. Click NEW button
  8. Paste in path: C:\Program Files\MongoDB\Server\5.0\bin
  9. Click OK to save
  10. Open new PowerShell
  11. Type on mongod.exe
  12. The server will start
