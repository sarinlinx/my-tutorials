Darkened header image with centered text

This example explains the following.

  • How to center a header image
  • Darken the image using a gradient
  • Layer and center text over the background image

Click here to view the example page.

  • Example

#1: Add a header section with text

The header should be in header tags.

Place the text content in a div with a class.

<header>
    <div class="hero-text-box">
        <h1>Text to be centered over image.</h1>
    </div>
</header>

#2: Add CSS to center entire page

Add and center the image in CSS.

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

#3: Add the background image, center it, and add a gradient

Add and center the image in CSS.

linear-gradient darkens the image. Using the same values means there is no gradient.

100vh means 100% of the viewport. This spans the entire page. This can zoom the image so it doesn't look good. To fix, add background-size: cover and background-position: center;

background-image: -webkit-linear-gradient(rgba(0, 0, 0, 0.7), rgba(0, 0, 0, 0.7)), url(img/hero.jpg);
  background-image: linear-gradient(rgba(0, 0, 0, 0.7), rgba(0, 0, 0, 0.7)), url(img/hero.jpg);
  background-size: cover;
  background-position: center;
  height: 100vh;
  background-attachment: fixed;
}

#4: Center text over image

Add CSS to position.

The position must be absolute so you can vertically and horizontally center it.

Using top and left to position should center section on page. If not, use transform to finalize the position.

-50% transforms moves the section half of it's height and width.

.hero-text-box {
  position: absolute;
  width: 1140px;
  top: 50%;
  left: 50%;
  /*-webkit-transform: translate(-50%, -50%);
  -ms-transform: translate(-50%, -50%);
  transform: translate(-50%, -50%);*/
}
