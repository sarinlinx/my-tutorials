Hover over image to expand and change opacity

#1: Position images

(absolute) Position images inside their (relative) container.

.composition {
  position: relative
}

.composition__photo {
  width: 55%;
  box-shadow: 0 1.5rem 4rem rgba(240, 9, 9, 0.966);
  border-radius: 2px;
  position: absolute;                
}

.composition__photo--p1 {
  left: 0;
  top: -2rem
}

.composition__photo--p2 {
  right: 0;
  top: 2rem
}

.composition__photo--p3 {
  left: 20%;
  top: 10rem
}

#2: Add hover effect

Hover over any image and it moves on top of other images.

  1. Add a z-index on the 'composition__photo' class each image shares.
  2. Give the hover property a higher z-index.
  3. Add a transition on the 'composition__photo' class to ease the change.
.composition {
  position: relative
}

.composition__photo {
  width: 55%;
  box-shadow: 0 1.5rem 4rem rgba(240, 9, 9, 0.966);
  border-radius: 2px;
  position: absolute;    
  z-index: 10;
  transition: all 0.2s;            
}

.composition__photo--p1 {
  left: 0;
  top: -2rem
}

.composition__photo--p2 {
  right: 0;
  top: 2rem
}

.composition__photo--p3 {
  left: 20%;
  top: 10rem
}

.composition__photo-x2:hover {
  transform: scale(1.05);
  box-shadow: 0 2.5rem 4rem rgba(240, 9, 9, 0.966);
  z-index: 20;
}
                
                
<div class="composition">
  <img src="nat-1-large.jpg" alt="photo 1" class="composition__photo composition__photo--p1">
  <img src="nat-2-large.jpg" alt="photo 2" class="composition__photo composition__photo--p2">
  <img src="nat-3-large.jpg" alt="photo 3" class="composition__photo composition__photo--p3">
</div>

#3: Add hover border

Add border when hovering image using 'outline' and 'outline-offset'.

Also scale the images that are NOT being hovered down when one is hovered.

.composition {
  position: relative;
}

.composition__photo {
  width: 55%;
  box-shadow: 0 1.5rem 4rem rgba(240, 9, 9, 0.966);
  position: absolute;
  z-index: 10;
  transition: all 0.2s;
  outline-offset: 2rem;
}

.composition__photo--p1 {
  left: 0;
  top: -2rem;
}

.composition__photo--p2 {
  right: 0;
  top: 2rem;
}

.composition__photo--p3 {
  left: 20%;
  top: 10rem;
}

.composition__photo:hover {
  outline: 1.5rem solid rgb(103, 255, 1);
  transform: scale(1.05) translateY(-0.5rem);
  box-shadow: 0 2.5rem 4rem rgba(240, 9, 9, 0.966);
  z-index: 20;
}
/* when you hover '.composition',  select '.composition__photo's that are NOT hovered */
.composition:hover .composition__photo:not(:hover) {
  transform: scale(0.95)
}
  
                
<div class="composition">
  <img src="nat-1-large.jpg" alt="photo 1" class="composition__photo composition__photo--p1">
  <img src="nat-2-large.jpg" alt="photo 2" class="composition__photo composition__photo--p2">
  <img src="nat-3-large.jpg" alt="photo 3" class="composition__photo composition__photo--p3">
</div>
