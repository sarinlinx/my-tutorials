Hover over image to exand and change opacity

This example explains the following.

  • Set opacity of images.
  • Hover effect to enlarge them.
  • Opacity changes on hover.

#1: Add images

Add 4 images as a horizontal list.

.meals-showcase {
  list-style: none;
  width: 100%;
}

.meals-showcase li {
  display: block;
  float: left;
  width: 25%;
}

.meal-photo {
  /* this means it has the width of its parent */
  width: 100%;
  margin: 0;
  /* cuts off excess */
  overflow: hidden;
  background-color: #000;
}
<section class="section-meals">
    <ul class="meals-showcase clearfix">
        <li>
            <figure class="meal-photo">
                <img src="1.jpg" alt="Korean bibimbap with egg and vegetables">
            </figure>
        </li>
        <li>
            <figure class="meal-photo">
                <img src="2.jpg" alt="Simple italian pizza with cherry tomatoes">
            </figure>
        </li>
        <li>
            <figure class="meal-photo">
                <img src="3.jpg" alt="Chicken breast steak with vegetables">
            </figure>
        </li>
        <li>
            <figure class="meal-photo">
                <img src="4.jpg" alt="Autumn pumpkin soup">

    </ul>
</section>

#2: Add CSSimages

Set opacity, width and height.

Scale to expand size of images.

Set a transition property to transition time.

/* adds opacity/transform/transition */
.meal-photo img {
  /* opacity color is based off the background-color of container */
  opacity: 0.7;
  width: 100%;
  height: auto;

  -webkit-transform: scale(1.15);
  -ms-transform: scale(1.15);
  transform: scale(1.15);

  -webkit-transition: -webkit-transform 0.5s, opacity 0.5s;
  transition: transform 0.5s, opacity 0.5s;
}

#3: Add CSS images

Set opacity to return to normal.

Scale to expand size of image.

.meal-photo img:hover {
  opacity: 1;

  /* scaling to 1 will leave blank space. Add .03 to fill */
  -webkit-transform: scale(1.03);
  -ms-transform: scale(1.03);
  transform: scale(1.03);
}
