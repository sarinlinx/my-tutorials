Step 2 - Curve text around first section, then change its shape into a cirlce

Example

Force the text to curve around the first section as if it were a circle using shape-outside.

Then, change the first section into the shape of a circle using clip-path.

.story {
  width: 75%;
  margin: 0 auto;
  box-shadow: 0 3rem 6rem rgba(0, 0, 0, 0.1);
  background-color: rgba(255, 255, 255, 0.6);
  border-radius: 3px;
  padding: 6rem;
  padding-left: 9rem;
  font-size: 1.6rem;
}

.story__shape {
  width: 15rem;
  height: 15rem;
  float: left;
  background-color: orangered;
  /* use shape-outside to define WHERE content should flow around element */
  /* this doesn't actually change the shape of the element */
  /* make text float around shape (whatever the shape may be) */
  /* element must be floated and have width and height */
  /* 1st # is radius of circle
  2nd and 3rd values are vertical and horizontal */
  -webkit-shape-outside: circle(50% at 50% 50%);
  shape-outside: circle(50% at 50% 50%);
  /* clip content into a circle */
  /* this is what changes the shape of the element */
  -webkit-clip-path: circle(50% at 50% 50%);
  clip-path: circle(50% at 50% 50%);
  /* push circle to the left instead of using margin */
  transform: translateX(-3rem);
}

Step 3 - Add image to circle section

Example 3

Remove color from shape.

Using transform to position it within that container.

  width: 15rem;
  height: 15rem;

  -webkit-shape-outside: circle(50% at 50% 50%);

 -webkit-clip-path: circle(50% at 50% 50%);
  clip-path: circle(50% at 50% 50%);

  /* must give it a height of 100% of parent so it is scaled and stays within the parent

  /* scale makes it bigger at first. You can then set it back to 1 when hovered */

}

Step 4 - (Move to its own section since this doesn't involve floating text: Skew section, but skew subsection back to normal

Example 4

Skew the entire section -12deg.

Skew individual sections back 12deg so they appear normal within the main section.

.story {
  width: 75%;
  margin: 0 auto;
  box-shadow: 0 3rem 6rem rgba(0, 0, 0, 0.1);
  background-color: rgba(255, 255, 255, 0.6);
  border-radius: 3px;
  padding: 6rem;
  padding-left: 9rem;
  font-size: 1.6rem;
  /* skew the entire section */
  transform: skewX(-12deg)
}

.story__shape {
  width: 15rem;
  height: 15rem;
  float: left;
  -webkit-shape-outside: circle(50% at 50% 50%);
  shape-outside: circle(50% at 50% 50%);
  -webkit-clip-path: circle(50% at 50% 50%);
  clip-path: circle(50% at 50% 50%);
  /* skew back 12deg since section is skewed -12deg */
  transform: translateX(-3rem) skewX(12deg);
}

.story__img {
    height: 100%;
    transform: translateX(-4rem) scale(1.4);
}

.story__text {
  /* this only skews the text */
  /* the __shape already has a transform used */
  transform: skewX(12deg);
}

Step 6 - (Move to its own section since this doesn't involve floating text: Move the text on hover

Step 7 - (Move to its own section since this doesn't involve floating text: Blur image on hover

Example 7

story__img scale is originally set to 1.4.

When hovered, the scale goes back to 1.

A filter is also added to blur the image

*,
*::after,
*::before {
  margin: 0;
  padding: 0;
  box-sizing: inherit
}

html {
  font-size: 62.5%
}

body {
  box-sizing: border-box;
  padding: 3rem
}

.row {
  max-width: 114rem;
  margin: 0 auto
}

.story {
  width: 75%;
  margin: 0 auto;
  box-shadow: 0 3rem 6rem rgba(0, 0, 0, 0.1);
  background-color: rgba(255, 255, 255, 0.6);
  border-radius: 3px;
  padding: 6rem;
  padding-left: 9rem;
  font-size: 1.6rem;
  transform: skewX(-12deg)
}

.story__shape {
  width: 15rem;
  height: 15rem;
  float: left;
  -webkit-shape-outside: circle(50% at 50% 50%);
  shape-outside: circle(50% at 50% 50%);
  -webkit-clip-path: circle(50% at 50% 50%);
  clip-path: circle(50% at 50% 50%);
  transform: translateX(-3rem) skewX(12deg);
  position: relative;
}

.story__img {
  height: 100%;
  /* original scale of image is 1.4 */
  transform: translateX(-4rem) scale(1.4);
  /* This fixes shape of image if it was change when transform: translateX(-4rem) was added */
  /* Doesn't occur in chrome though */
  backface-visibility: hidden;
  transition: all .5s;

}

.story__text {
  transform: skewX(12deg);
}

.story__caption {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -20%);
  color: #fff;
  text-transform: uppercase;
  font-size: 1.7rem;
  text-align: center;
  opacity: 0;
  transition: all .5s;
  backface-visibility: hidden;
}

.story:hover .story__caption {
  opacity: 1;
  transform: translate(-50%, -50%);
}

.story:hover .story__img {
  /* scales img back to 1 from 1.4 */
  transform: translateX(-4rem) scale(1);
  /* blur 3px is very subtle */
  /* change image 80% makes it darker from 100% */
  filter: blur(3px) brightness(80%)
}
