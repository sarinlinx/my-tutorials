Step #2 - Add CSS for background

Create circle with background radial-gradient.

This will be fixed to the top right of page.

 .navigation__background {
  height: 6rem;
  width: 6rem;
  border-radius: 50%;
  /* Take element out of page flow */
  position: fixed;
  top: 6.5rem;
  right: 6.5rem;
  background-image: radial-gradient(#7ed56f, #28b485);
  /* so it is always on top */
  z-index: 1000;
}

@media only screen and (max-width: 56.25em) {
  .navigation__background {
      top: 4.5rem;
      right: 4.5rem
  }
}

@media only screen and (max-width: 37.5em) {
  .navigation__background {
      top: 3.5rem;
      right: 3.5rem
  }
}

Step #4 - Add CSS for navigation links

When the white button is clicked, the page fully becomes this navigation__nav section.

It is centered using the navigation__list inside it.

.navigation__nav {
    width: 100%;
    /* top right of viewport and covers entire viewport */
    height: 100vh;
    /* starts on top right of viewport */
    top: 0;
    left: 0;
    position: fixed;
    /* this should be on top of the background so it's in between 1000 and 2000 */
    z-index: 1500;
}

.navigation__list {
  /* center list items */
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    list-style: none;
    text-align: center;
    width: 100%
}

Step #7 - Remove/hide navigation

Remove the navigation links using opacity and width.

Remove original width: 100% since it's no longer needed.

When checkbox is checked:

  • Add a transform on the checkbxo to make the background fill screen
  • Add opacity to navigation__nav so it shows.
.navigation__nav {
  height: 100vh;
  top: 0;
  left: 0;
  position: fixed;
  z-index: 1500;
  /* hide menu items */
  opacity: 0;
  /* removes them from page so they can't be clicked even though they are invisible */
  width: 0;
}

.navigation__checkbox:checked~.navigation__background {
  /* Make it huge so it covers the entire screen */
  transform: scale(80)
}

.navigation__checkbox:checked~.navigation__nav {
  /* show navigation */
  opacity: 1;
  width: 100%
}

.navigation__icon::before,

  /* so you can set their position */
  position: absolute;

}

/* spacing of icon line one at top and one at bottom */

Step #11 - Rotate hamburger lines when clicked

When button is clicked, hide center line

Rotate top and bottom lines to become an X

/* when checkbox is checked select sibling's icon
this is the center horizontal line in the icon */
.navigation__checkbox:checked+.navigation__button .navigation__icon {
  /* hide */
  background-color: transparent
}

/* rotate the top and botom icons */
.navigation__checkbox:checked+.navigation__button .navigation__icon::before {
  /* rotate 180deg - 45deg so it flips around */
  transform: rotate(135deg);
  /* set to zero so the top and bottom icons form an X */
  top: 0
}

.navigation__checkbox:checked+.navigation__button .navigation__icon::after {
  /* rotate */
  transform: rotate(-135deg);
  /* set to zero so the top and bottom icons form an X */
  top: 0
}
