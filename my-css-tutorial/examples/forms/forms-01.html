This example creates a form that appears on the left half of the section.

Step #1 - Create section containers

Create a section to contain the form.

Create a row for the form.

Add interior container with background image.

Give it padding an a gradient background.

NOTE: height will not be added to the .book container. It's only used here so you can see the background image. The content will set the height.

.section-book {
  padding: 15rem 0;
  background-image: linear-gradient(to right bottom, #7ed56f, #28b485);
}        

@media only screen and (max-width: 56.25em) {
  .section-book {
      padding: 10rem 0
  }
}

.book {
  /* Do not add height. The content should set the height */
  height: 50rem;
  /* Add gradient over background image */
  /* Use % to set where gradient should appear */
  /* At 0% to 50% it's white. After 50% it's transparent */
  background-image: linear-gradient(105deg, rgba(255, 255, 255, 0.9) 0%, rgba(255, 255, 255, 0.9) 50%, transparent 50%), url(nat-10.jpg);
  /*  fits container in x direction, 100% is the same as cover. If it was a lower % it would start to repeat */
  background-size: 100%;
  border-radius: 3px;
  box-shadow: 0 1.5rem 4rem rgba(0, 0, 0, 0.25);
}

@media only screen and (max-width: 75em) {
  .book {
      background-image: linear-gradient(105deg, rgba(255, 255, 255, 0.9) 0%, rgba(255, 255, 255, 0.9) 65%, transparent 65%), url(../img/nat-10.jpg);
      background-size: cover
  }
}

@media only screen and (max-width: 56.25em) {
  .book {
      background-image: linear-gradient(to right, rgba(255, 255, 255, 0.9) 0%, rgba(255, 255, 255, 0.9) 100%), url(../img/nat-10.jpg)
  }
}

Step 3 - Add CSS to the form container and elements.

Add style to format form__group except last child

Style input element.

Remove default form focus and customize.

Add text validation to the fields.

Style placeholder text. (Safari and Chrome only)

.form__group:not(:last-child) {
  margin-bottom: 2rem;
}

.form__input {
/* input elements do not inherit fonts and color by default
change default browser font for forms to font-family of page */
  font-family: inherit;
  color: inherit;
  font-size: 1.5rem;
  /* Looks better when more padding is on sides */
  padding: 1.5rem 2rem;
  border-radius: 2px;
  background-color: rgba(255, 255, 255, 0.5);
  border: none;
  /*set initial border to transparent
this prevents the 3px (from .form__input:focus) from pushing down the fields below the first box */
  border-bottom: 3px solid transparent;
  width: 90%;
  display: block;
}

.form__input:focus {
  /* remove browser default outline */
  outline: none;
  /* add box shadow for users not using a mouse so they can see what field is focused */
  box-shadow: 0 1rem 2rem rgba(0, 0, 0, 0.1);
  /* This 3px will cause the field to shift down when focused
  To fix, add border-bottom: 3px solid transparent; to the primary input selector */
  border-bottom: 3px solid #55c57a;
}

.form__input:focus:invalid {
  /* invalid reference is built into the browser */
  /* if form field text is invalid when focused*/
  border-bottom: 3px solid #ff7730;
}

.form__input::-webkit-input-placeholder {
  /* Changes color of text when you type into field (safari and chrome)*/
  color: #999;
}

Step 6 - Style the radio button

Add a 'checked' pseudo selector to the radio button.

When it's clicked, the sibling's child will display.

Add a transition on the radiou button::after so it fades in

Hide original radio input

.form__radio-button::after {
  content: "";
  display: block;
  height: 1.3rem;
  width: 1.3rem;
  border-radius: 50%;
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  background-color: #55c57a;
  opacity: 0;
  transition: opacity .2s
}

/* Check all radio buttons
that are siblings of radio-input, then select radio-label, then select child element of it */
/* when you click the radio button, the input becomes CHECKED using the checked pseudo class
then select its sibling, and then its child since this is what is changed*/
  
 form__radio-input:checked~.form__radio-label .form__radio-button::after {
  opacity: 1
}

.form__radio-input {
  display: none
}
