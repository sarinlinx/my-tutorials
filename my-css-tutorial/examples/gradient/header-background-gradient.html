Header Background Gradient

This creates a background gradient under the header text

It fades out on both sides.

It's best when the color you choose blends into the background.

.heading-primary--container__homepage { 
  text-align: center;
  background: linear-gradient(to right, transparent 15%, rgba(111, 111, 111, .4) 50%, transparent 85%);
}

.heading-primary__homepage {
  margin-top: 3rem;
  font-size: 5.5rem;
  font-family: 'Cormorant SC', cursive;
  padding-bottom: 1rem;
  letter-spacing: 2px;
  display: inline;
  background-image: linear-gradient(to top, #8ca9ff, #f6e4ad);
  -webkit-background-clip: text;
  color: transparent;
}

<div class="heading-primary--container__homepage">
  <div class="heading-primary__homepage">Knowledge Base Tools</div>
</div>
