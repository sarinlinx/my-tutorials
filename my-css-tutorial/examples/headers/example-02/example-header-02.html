Header example #2

This example shows how to create a header with an image that fills the page.

A gradient is then laid overtop.

  • Header example

#1: Add a header section

Add a header section in HTML

<header class="header">

</header>

#2: Add CSS to add image and gradient

This adds an image with a gradient.

It then covers the entire page.

.header {
  /* This box should be 95% of viewport */
  height: 95vh;
  /* gradient, comma, then image */
  background-image: linear-gradient(to right bottom, rgba(126, 213, 111, 0.8), rgba(40, 180, 133, 0.8)), url(hero.jpg);
  /* takes up full width of viewport */
  background-size: cover;
  /* top of image will always stay at top of screen when resizing */
  background-position: top;
  /* clip-path changes shape of image by adjusting 4 corners using x y values */
  -webkit-clip-path: polygon(0 0, 100% 0, 100% 75vh, 0 100%);
  clip-path: polygon(0 0, 100% 0, 100% 75vh, 0 100%);
  /* must set to relative so you can position logo as absolute */
  position: relative;
}

#3: Adding text

Add span tags with your text.

Set them as block elments so they won't break apart when the browser is resized.

<header class="header">
  <span class="heading-primary--main">Outdoors</span>
</header>


.heading-primary--main {
  display: block;
  font-size: 6rem;
  font-weight: 400;
  letter-spacing: 3.5rem;
}

#3: Center text

Surround text in a div

Set element to absolute

Add percentages to center. This won't work though since the elements top left corner will be what's centered, so the element will appear low and to the right.

To center by the ELEMENT and not the exact position of where it begins, use 'transform'. This will center the ELEMENT itself.

<header class="header">
  <div class="header__text-box">
    <span class="heading-primary--main">Outdoors</span>
  </div>
</header>

.header__text-box {
  /* set absolute position */
  position: absolute;
  /* the top and left value position from the parent element, 
  so they are not visually centered since the elment will START 50% down and 50% to the right
  this means it will appear low to the right because it's top left point is centered on the page */
  top: 50%;
  left: 50%;
  /* this centers the element visually by adjusting 50% of the element, not the parent*/
  transform: translate(-50%, -50%);
  text-align: center;
}
