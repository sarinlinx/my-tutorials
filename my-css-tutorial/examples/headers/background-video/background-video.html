Background Video

This example shows how to add a background video to the page.

A gradient is then laid overtop.

  • Background video

Step #1 - Add HTML

Add a <div> with a class.

Inside, add a <video> tag. Make sure to set it to autoplay, muted, and loop

In that, add the <source> attributes.

<section class="section-stories">

<!-- video background -->
<div class="bg-video">

  <video class="bg-video__content" autoplay muted loop>
    <!-- dont' use src in the video tag. Use <source> tag instead-->
    <source src="img/video.mp4" type="video/mp4">
    <source src="img/video.webm" type="video/mp4">
    Your browser is not supported!
  </video>

</div>

<!-- Text block -->
<div class="row">
  <div class="story">
    <div class="story__text">
      <h3>Set Opacity on parent</h3>
        <p>If you want to add a background to a text area that is darker than the video background, 
        you must set that opacity on the parent of the text block.</p>
        <p>This block sets the parent div to background-color: rgba(255, 255, 255, 0.6);</p>
    </div>
  </div>
</div>

</section>

Step #2 - Add CSS to container

Set the container section to relative so you can position the image.

.section-stories {
  position: relative;
  padding: 15rem 0
}

Step #3 - Add CSS to Video section

Position the video section absolute.

Set z-index below everything else.

Add overflow: hidden so the large video doesn't spill out of the parent container.

.bg-video {
  position: absolute;
  top: 0;
  left: 0;
  /* height and width of parent */
  height: 100%;
  width: 100%;
  opacity: .15;
  /* set z-index so all elements are above this layer */
  z-index: -1;
  /* prevent video size from overflowing parent */
  overflow: hidden
}

Step #4 - Add CSS to video itself

Add a height and width to ensure it covers the entire section it's in. Otherwise, parts of the section will not contain the video.

Add object-fit to resize it within its container.

.bg-video__content {
  /* prevent video overflowing section */
  height: 100%;
  width: 100%;
  /* allows vide to fill parent while maintaining aspect ratio */
  -o-object-fit: cover;
  object-fit: cover
}

Step #5 - Additional CSS for text section opacity

This code creates a text section with a background opacity.

The opacity is set on the parent, otherwise it would be the same opacity as the video background.

.story {
  width: 75%;
  margin: 0 auto;
  box-shadow: 0 3rem 6rem rgba(0, 0, 0, 0.1);
  background-color: rgba(255, 255, 255, 0.6);
  border-radius: 3px;
  padding: 6rem;
  padding-left: 9rem;
  font-size: 1.6rem;
  transform: skewX(-12deg)
}

.story__text {
  transform: skewX(12deg);
}
