Set Opacity on parent

If you want to add a background to a text area that is darker than the video background, you must set that opacity on the parent of the text block.

This block sets the parent div to background-color: rgba(255, 255, 255, 0.6);

This section did not set an opacity for it's parent background, so it blends in with the background video.

You can only see the outline since there is a box-shadow added.
