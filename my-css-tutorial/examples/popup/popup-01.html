Step #2 - Style container and popup

Give the background a dark transparent color. This covers the entire page.

Give the popup container a white background.

.popup {
  /* cover entire page in dark background */
  height: 100vh;
  /* not really necessary */
  width: 100%;
  position: fixed;
  top: 0;
  left: 0;
  background-color: rgba(0, 0, 0, 0.8);
  z-index: 9999;               
}

.popup__content {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 75%;
  /* only use height now since there's no content */
  height: 50rem;
  background-color: #fff;
  box-shadow: 0 2rem 4rem rgab(#000, 0.2);
  border-radius: 3px;                 
}

Step #4 - Style images and align both sides

Style images first since they are so large.

Use display: table-cell so both sides have the same height. This isn't possible with float.

Remove height: 50rem; from .popup__content since there's now content to display.

Add display: table to .popup__content.

Add overflow: hidden so images do not overflow their container.

.popup__content {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 75%;              
  background-color: #fff;
  box-shadow: 0 2rem 4rem rgab(#000, 0.2);
  border-radius: 3px;
  /* used to make left and right sides have same height */
  display: table;
  overflow: hidden;               
}
                
.popup__img {
  display: block;
  /* 100% of parent container's width */
  width: 100%
}

.popup__left {
  width: 33.33333%;
  /* used to make left and right sides have same height */
  display: table-cell
}

.popup__right {
  width: 66.66666%;
  /* used to make left and right sides have same height */
  display: table-cell;
  /* align vertically */
  vertical-align: middle;
  padding: 3rem 5rem
}

Step #6 - Hide popup and add anchor link

Set opacity and visibility to hide popup container.

Add popup:target CSS which is what is called when the button link is clicked.

Add button on homepage to click.

Add transition to container so it fades in

.popup {
  height: 100vh;
  width: 100%;
  position: fixed;
  top: 0;
  left: 0;
  background-color: rgba(0, 0, 0, 0.8);
  z-index: 9999;
  opacity: 0;
  visibility: hidden;
  transition: all 1s;
}

/* targert when a button is clicked to anchor tag and URL changes to #popup */
.popup:target {
opacity: 1;
visibility: visible
}


  <a href="#popup" class="btn btn--white">Book now!

Step #8 - Add zoom effect

Add a second transform on popup__content with scale to make it smaller.

Then scale back to 1 on the target popup.

Add small delay by .2s

.popup__content {
      position: absolute;
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
      width: 75%;
      background-color: #fff;
      box-shadow: 0 2rem 4rem rgab(#000, 0.2);
      border-radius: 3px;
      display: table;
      overflow: hidden;
      opacity: 0;
      /* there is already a transform in the @include, so you cannot add a 2nd in the same declaration. To fix add the original from the @include, add the new one in the same declaration */
      transform: translate(-50%, -50%) scale(0.5);
      transition: all .4s .2s;
  }

/* scale originally downscaled to .5 on popup__content
This sets it back to full size 1 */
.popup:target .popup__content {
  opacity: 1;
  transform: translate(-50%, -50%) scale(1);
}
