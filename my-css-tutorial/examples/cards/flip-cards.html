Step 1 - rotate the card

Basic styles to flip the card

.card1 {
  background-color: orangered;
  width: 30%;
  /* height of entire card */
  height: 10rem;
  transition: all .8s;
}

.card1:hover {
  transform: rotateY(180deg);
}
        
<div class="card1">
    <h4>The Sea Explorer</h4>      
</div>

Step 2 - Add perspective

Add the perspective property so the flip looks 3d.

The lower the value, the more dramatic the perspective.

You need to keep adjusting the value to get the effect you want. 

.card2 {
  /* add perspective so transform looks 3d */
  perspective: 150rem;
  -moz-perspective: 150rem;      
}

.card2__side {
  background-color: orangered;
  /* height of entire card */
  width: 30%;
  height: 10rem;
  transition: all 2s;
}

.card2:hover .card2__side {
  transform: rotateY(180deg);
}
        
<div class="card1">
  <div class="card2__side">
    <h4>The Sea Explorer</h4>      
  </div>
</div>

Step 3 - Add back side of card

Add the back side of the card so you view something different when flipped.

The front side starts out at zero deg. When hovered, it flips to 180deg.

The back side starts out flipped at 180deg. When hovered, it returns to zero.

The problem is the side are not on top of each other.

.card3 {
  perspective: 150rem;
  -moz-perspective: 150rem;
}

.card3__side {
  background-color: orangered;
  width: 30%;
  height: 10rem;
  transition: all 2s;
}

.card3__side--front {
  background-color: orangered;
}

.card3__side--back {
  background-color: green;
  /* back side starts out flipped at 180deg */
  transform: rotateY(180deg);
}

.card3:hover .card3__side--front {
  /* rotate the front side 180deg to flip */
  transform: rotateY(180deg);
}

.card3:hover .card3__side--back {
  /* rotate back side from 180deg back to zero to flip */
  transform: rotateY(0deg);
}
        
<div class="card3">

  <div class="card3__side card3__side--front">  
    <h4>FRONT</h4>
  </div>

  <div class="card3__side card3__side--back">
    <h4>BACK</h4>
  </div>

</div>

Step 4 - Stack cards on top of each other

Add absolute positioning to stack the cards.

.card4 {
  perspective: 150rem;
  -moz-perspective: 150rem;
}

.card4__side {
  background-color: orangered;
  height: 10rem;
  transition: all 2s;
  /*  use absolute position to stack --front and --back on top of each other */
  /* this also collapses parent height since these elements were removed from natural flow */
  /* to fix give same height as parent */
  position: absolute;
  top: 0;
  left: 0;
}

.card4__side--front {
  background-color: orangered;
}

.card4__side--back {
  background-color: green;
  transform: rotateY(180deg);
}

.card4:hover .card4__side--front {
  transform: rotateY(180deg);
}

.card4:hover .card4__side--back {
  transform: rotateY(0deg);
}

  <div class="card4__side card4__side--front">  
    <h4>FRONT</h4>
  </div>

Step 6 - Add backface visibility to hide back side and set height of container

Set backface-visibility to hidden so the back side of the card is hidden until flipped.

Finally, the parent height of the card collapsed when you took it out of its natural flow by setting position to absolute. This causes the perspective to not function.

The solution is to also add the height to the .card container.

.card6 {
perspective: 150rem;
-moz-perspective: 150rem;
/* height of entire card */
height: 10rem;
}

.card6__side {
background-color: orangered;
/* height of each side of the card. must match height of card */
height: 10rem;
transition: all 2s ease;
position: absolute;
top: 0;
left: 0;
width: 30%;
/* hides back side until flipped */
backface-visibility: hidden;
}

.card6__side--front {
background-color: orangered;
}

.card6__side--back {
background-color: green;
transform: rotateY(180deg);
}

/* hover roates front side 180deg
  when card is hovered, .card7__side--front is transformed
  do not put hover here directly on card7__side since there are 2 sides
  this should only occur on main card component
  this is -180 so it will turn in the same direction the back turns in */
.card6:hover .card6__side--front {
transform: rotateY(-180deg);
}

/* Since this is already rotated 180deg, change to zero */
.card6:hover .card6__side--back {
transform: rotateY(0deg);
}
    
<div class="card6">

<div class="card6__side card6__side--front">  
<h4>FRONT</h4>
</div>

<div class="card6__side card6__side--back">
<h4>BACK</h4>
</div>

</div>

Step 9 - Add heading text over the image

Add heading section with a modifier class of card9__heading-span

Add another class to specify this heading if you have multiple cards.

Position absolute to place on the right.

Adding padding to card9__heading-span and width to card9__heading breaks the heading into 2 lines

CSS normally would still view the broken heading as a single element.

Finally, use box-decoration-break: clone so both lines of the broken header are styled individually.

 *,
*::after,
*::before {
  margin: 0;
  padding: 0;
  box-sizing: inherit;
}

html {
  font-size: 62.5%;
}

body {
  box-sizing: border-box;
  padding: 3rem;
}

/* FRONT SIDE STYLING */
.card9 {
  perspective: 150rem;
  -moz-perspective: 150rem;
  position: relative;
  height: 52rem;
}

.card9__side {
  height: 52rem;
  transition: all .8s ease;
  position: absolute;
  top: 0;
  left: 0;
  width: 33%;
  overflow: hidden;
  -webkit-backface-visibility: hidden;
  backface-visibility: hidden;
  border-radius: 3px;
  box-shadow: 20px 20px 50px 15px rgba(1, 1, 1, .15);
}

.card9__side--front {
  background-color: #fff;
}

.card9__side--back {
  transform: rotateY(180deg);
}

.card9__side--back--1 {
  background-image: linear-gradient(to right bottom, #ffb900, #ff7730);
}

.card9:hover .card9__side--front {
  transform: rotateY(-180deg);
}

.card9:hover .card9__side--back {
  transform: rotateY(0);
}


.card9__picture {
  background-size: cover;
  height: 23rem;
  background-blend-mode: screen;
  -webkit-clip-path: polygon(0 0, 100% 0, 100% 85%, 0 100%);
  clip-path: polygon(0 0, 100% 0, 100% 85%, 0 100%);
}

.card9__picture--1 {
  background-image: linear-gradient(to right bottom, #ffb900, #ff7730), url(nat-5.jpg);
}


.card9__heading {
  font-size: 2.8rem;
  font-weight: 300;
  text-transform: uppercase;
  text-align: right;
  color: #fff;
  /* layer over background image */
  position: absolute;
  top: 12rem;
  right: 2rem;
  /* make more narrow so the text box stays within the card box */
  width: 75%;
}


.card9__heading-span {
  /* this breaks text into two lines */
  padding: 1rem 1.5rem;
  /* this treats broken text as different elements */
  /* applies all declarations to all boxes created by an element both styled the same */
  /* this helps in this example to make the padding the same for both lines of text */
  -webkit-box-decoration-break: clone;
  box-decoration-break: clone;
}

.card9__heading-span--1 {
  background-image: linear-gradient(to right bottom, rgba(255, 185, 0, 0.75), rgba(255, 119, 48, 0.75));
}
