Text gradient

Creates a gradient in the text color.

Step 1 - Add a gradient background color

.heading-secondary {
  font-size: 3.5rem;
  text-transform: uppercase;
  font-weight: 700;

  /* Set gradient on text */
  /* first set to inline so it doesn't take up 100% of width*/
  display: inline;

  /* first set background color of block */
  background-image: linear-gradient(to right, #7ed56f, #162adb);

  /* background clipped to where text is */
  -webkit-background-clip: text;

  /* show the text, not the background gradient 
  You cannot see the gradient yet because the text now sits on top of background
  make text transparent so you can see background color */
  color: transparent;

  letter-spacing: 2px;
  transition: all .2s;
}

/* Add line breaks after each inline header element 
  otherwise, text following the header would appear on the same line*/
h1::after,
h2::after,
h3::after,
h4::after {
  content: "\a";
  white-space: pre;
}

/*____________________________________________
SPACER CLASSES
Add spacer classes since you cannot add margins to the inline header elements
_____________________________________________*/
.h1-spacer-top {
    margin-top: 3rem;
}

.h2-spacer-top {
    margin-top: 2rem;
}

.h3-spacer-top{
    margin-top: 1.5rem;
}


.h1-spacer-bottom {
    margin-bottom: 3rem;
}

.h2-spacer-bottom {
    margin-bottom: 2rem;
}

.h3-spacer-bottom{
    margin-bottom: 1.5rem;
}
