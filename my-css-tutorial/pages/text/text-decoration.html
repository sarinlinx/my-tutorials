Text decoration

text-decoration allows you to add or remove decorations from text.

The value text-decoration: none; is often used to remove underlines from links.

h1 {
      text-decoration: overline;
    }

    h2 {
      text-decoration: line-through;
    }

    h3 {
      text-decoration: underline;
    }

    h3 {
      text-decoration: none;
    }

    Underline

    None

text-decoration

text-decoration specifies the decoration added to text such as underline.

Optional values are:

  • overline
  • underline
  • line-through
  • colors

Examples

overline

div {
          text-decoration-line: overline red;
        }

        This text is overlined in red

underline

div {
          text-decoration-line: underline green;
        }

        This text is underlined in green

line-through

div {
          text-decoration-line: line-through blue;
        }

        This text has a blue line through it
