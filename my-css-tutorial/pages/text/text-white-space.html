white-space

white-space specifies how white-space is handled inside an element. Value are:

  • normal - This is default
  • nowrap - Text will never wrap to the next line. The text continues on the same line until a <br> tag is encountered.
  • pre - Text will never wrap to the next line. Acts like the <pre> tag in HTML. Text will only wrap if the HTML is formatted that way, or if you use a <br> tag. Text will also indent according to how the HTML is formatted.
  • pre-line - Same as <pre> in that text will wrap how HTML is formatted, or if you use a <br> tag. But it's different in that text will still wrap when necessary, for example, if the browser window can't view it all, it will automatically wrap. Also, HTML formatting is ignored for indented text. Only line breaks are recognized in HTML.
  • pre-wrap - This is the same as pre-line, except that indented text is recognized, just like 'pre'. The difference from 'pre' is that the text eventually wraps.

Example: nowrap

Ignore the scrollbar. Text will be cut off and never move down to the next line.

p {
      white-space: nowrap;
    }
    This is a very long line that by default, should break to the next line of text so the reader can view all text.

Example: pre

Text will be cut off and never move down to the next line. Text will indent or wrap if HTML is formatted that way. Text will wrap if a <br> tag is used.

<p>This line breaks here
    since the HTML code is formatted that way.</p>
        This line is indented since HTML is formatted this way. But this will be cut off since pre doesn't wrap text. asdfasdfasdfasdfasdfasdfasdf

    p {
      white-space: pre;
    }

    This line breaks here
    since the HTML code is formatted that way.
        This line is indented since HTML is formatted this way. But this will be cut off since pre doesn't wrap text. asdfasdfasdfasdfasdfasdfasdf

Example: pre-line

Different from 'pre' in that text will wrap and HTML indented text is not recognized.

It's the same as 'pre' in that line breaks in HTML are recognized and <br> tags are recognized.

<p>This line breaks here
    since the HTML code is formatted that way.
    This line is indented in  HTML, but is not recognized.
    If the HTML was on one line, this would eventually wrap to the next line at the end of the browser window.</p>

    p {
      white-space: pre-line;
    }

    This line breaks here
    since the HTML code is formatted that way.
    This line is indented in  HTML, but is not recognized.
    If the HTML was on one line, this would eventually wrap to the next line at the end of the browser window.

Example: pre-wrap

Text will wrap when needed. Otherwise it's the same as 'pre'

<p>This line breaks here
        since the HTML code is formatted that way.
          This line is indented in HTML and IS recognized.
        If the HTML was on one line, this would eventually wrap to the next line at the end of the browser window.</p>

    p {
      white-space: pre-wrap;
    }

    This line breaks here
    since the HTML code is formatted that way.
      This line is indented in HTML and IS recognized.
    If the HTML was on one line, this would eventually wrap to the next line at the end of the browser window.
