Aligning text

Use text-align to set the horizontal alignment of text. Text can be left or right aligned, centered, or justified.

h1 {
      text-align: center;
    }

    h2 {
      text-align: left;
    }

    h3 {
      text-align: right;
    }

justify will stretch so that every line has equal width, and the left and right margins are straight (like in magazines and newspapers).

div {
      text-align: justify;
    }

Transform text to uppercase/lowercase/capitalize

text-transform allows you to turn everything into uppercase or lowercase letters, or capitalize the first letter of each word.

p.uppercase {
  text-transform: uppercase;
}

p.lowercase {
  text-transform: lowercase;
}

p.capitalize {
  text-transform: capitalize;
}

p.capitalize {
  text-transform: none;
}

Indenting text

text-indent is used to specify the indentation of the first line of a block of text.

You can use pixels or %.

<p class="indent-section">Blah blah bl
s dlfjakf
dkjfdkjfa
djfkdf kdjf kdjfkd
asdfdfsadf
asdfasdf
</p>


.indent-section {
  text-indent: 50px;
}

.indent-section {
  text-indent: 10%;
}

Letter spacing

letter-spacing specifies the space between the characters in a text.

.letter-spacing {
  letter-spacing: 3px;
}

.letter-spacing {
  letter-spacing: -3px;
}

.letter-spacing {
  letter-spacing: normal;
}

Line Height

Line Height specifies the space between lines.

.small {
  line-height: 0.8;
}

.big {
  line-height: 1.8;
}

.normal {
  line-height:normal;
}

div {
  line-height: 25px;
}

div {
  line-height:150%;
}

Example. Space between lines goes from small, to normal, to large.

Word Spacing

word-spacing specifies the space between the words in a text.

h1 {
  word-spacing: -5px;
}

h2 {
  word-spacing: 10px;
}

Text direction

direction is used to specify which direction the text grows from.

/* text position to right and moves to left while typing */
div {
  direction: rtl;
}

/* text position to left and moves to right while typing */
div {
  direction: ltr;
}
