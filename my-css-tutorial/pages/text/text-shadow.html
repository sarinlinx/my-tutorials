text-shadow

text-shadow adds a shadow to the text in different directions. The following example specifies the position of the horizontal shadow (3px), the position of the vertical shadow (2px) and the color of the shadow (red):

SYNTAX: text-shadow: h-shadow v-shadow blur-radius color;

h1 {
  text-shadow: 3px 2px red;
}

Example. Horizontal red shadow to the right.

div {
  text-shadow: 3px 0px red;
}

Horizontal red text shadow to the right.

Example. Horizontal red shadow to the left.

div {
  text-shadow: -3px 0px red;
}

Horizontal red text shadow to the left.

Example. Vertical downwards red shadow.

div {
  text-shadow: 0px 3px red;
}

Vertical downwards red text shadow.

Example. Vertical upwards red shadow.

div {
  text-shadow: 0px -3px red;
}

Vertical upwards red text shadow.

Example. Horizontal and Vertical yellow shadow with a blur.

div {
  text-shadow: 3px 3px 3px yellow;
}

Text shadow with a blur.
