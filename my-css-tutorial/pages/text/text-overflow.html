text-overflow

text-overflow specifies how overflowed content (that is not displayed), should be notified to the user.

Both of the following properties are required for text-overflow:

  • white-space: nowrap;
  • overflow: hidden;

Possible values are:

  • clip - Default. The text is clipped and not accessible.
  • ellipsis - Render an ellipsis ("...") to represent the clipped text.
  • string - Render the given string to represent the clipped text.

This does not work on inline elements like a span tag.

clip

clip

<div>Text that won't fit in the box is clipped and not shown.</p>

    div {
      text-overflow: clip;
      white-space: nowrap;
      overflow: hidden;
      width: 335px;
      border: 1px solid #000000;
      color: white;
    }

Example of clip.

Text that won't fit in the box is clipped and not shown.

ellipsis

<div>Text that won't fit in the box is shown as an ellipsis.</p>

    div {
      text-overflow: ellipsis;
      white-space: nowrap;
      overflow: hidden;
      width: 335px;
      border: 1px solid #000000;
      color: white;
    }

Example of ellipsis.

Text that won't fit in the box is clipped and not shown.
