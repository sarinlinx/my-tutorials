Text color

With CSS, a color is most often specified by:

  • A color name like "red"
  • A HEX value like "#ff0000"
  • An rgb value like "rgb(255, 0, 0)"
p {
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

For W3C compliant CSS: If you define the color property, you must also define the background-color.

Change background color when dragging mouse over text

This changes the background color when you drag your mouse over text.

Do not add anything in front of ::selection

::selection {
  background-color: #fff;
  color: #000;
}
