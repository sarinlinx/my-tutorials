CSS Combinators

Combinators explain the relationship between multiple selectors. Between the selectors, you can add a combinator.

Examples of combinators:

  • multiple element selector (comma)
  • grouped class selector (period)
  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Multiple element selector (comma)

You can write a single declaration for multiple selectors at once. Just separate them with a comma.

This is used for styling links. For example: .classname, a {}

Ex 1

This CSS applies to h1, h2, and p tags.

h1, h2, p {
  text-align: center;
  color: red;
}

Grouped class selector (period)

If an element has multiple classes, you can select it by referencing its classes separated by a period.

Ex 1: Targeting an element with multiple classes

This matches the HTML element ONLY IF both classes are set in it.

Even though there is a 3rd class, it will still be selected since the elment is using both the .one and .two classes.

It doesn't matter which classes you reference as long as they are both present on the element.

<p class="one two three">Text here</p>

.one.two {
  /*styles*/
}

or

.one.three {
  /*styles*/
}

or

.two.three {
  /*styles*/
}
Result

Text here

Ex 2: Targeting an element with a specific class

You can use the period to specifically assign a class to only one HTML element.

This CSS applies only to h1 elements with a class of center.

h1.center {
  text-align: center;
}
Result

center class assigned to this element.

Descendant selector (space)

Selects elements that are descendants (inside of) of the parent element.

Ex 1

The paragraph with a class of .two is selected since it's inside of the div with a class of .one 

<div class="one">
  <p class="two">Text here</>
</div>

.one .two {
  /*styles*/
}

Ex 2

Select a link in a paragraph and change the color to red.

<p>Please click <a href="#">HERE!!!</a></p>

p a {
  color: red;
}
Result

Please click HERE!!!

Ex 3: Multiple descendent selectors

Select paragraphs and links inside <section>

<section>
<p>Please click <a href="#">HERE!!!</a></p>
</section>

section p a {
  color: blue;
}
Result

Please click HERE!!!

Ex4: Skipping descendant levels

You can also skip levels. This example looks inside all <ancestor> elements and selects any <descendant> elements inside. 
<div class="ancestor">
  <div class="parent">
   <div class="child">This is not styled</div>
    <div class="descendant">This IS styled</div>
   </div>
  </div>
</div>
CSS to select the descendant. 
.ancestor .descendant {
  color: blue;
}
Result

This is not styled

This IS styled

Child selector (>)

The child selector selects all elements that are the direct children of a specified element.

Ex 1

The following example selects all <p> elements that are children of a <div> element:

Paragraph 3 is not selected since it's not a child of the div. It's a descendant.

<div>
  <p>Paragraph 1 in the div.</p>
  <p>Paragraph 2 in the div.</p>
  <!-- not Child but Descendant -->
  <section><p>Paragraph 3 in the div.</p></section>
  <p>Paragraph 4 in the div.</p>
</div>

div > p {
  color: white;
}
Result

Paragraph 1 in the div.

Paragraph 2 in the div.

Paragraph 3 in the div.

Paragraph 4 in the div.

Adjacent selector (+)

The adjacent sibling selector selects all elements that are the adjacent siblings (immediately follow) of a specified element.

Sibling elements must have the same parent element

Ex 1

This example selects all <p> elements that are placed immediately after <div> elements:

<div>
  <p>Paragraph 1 in the div.</p>
  <p>Paragraph 2 in the div.</p>
</div>

<p>Paragraph 3. Not in a div, but is the sibling the immediately follows.</p>
<p>Paragraph 4. Not in a div.</p>

div + p {
  color: white;
}
Result

Paragraph 1 in the div.

Paragraph 2 in the div.

Paragraph 3. Not in a div, but is the sibling the immediately follows.

Paragraph 4. Not in a div.

Ex 2

This example ONLY selects the paragraph element directly after an <h2> element.

Adjacent Sibling Selectors only select the element immediately following a specified element. They are separated with a + symbol.

<p>Paragraph 1</p>
<h2>Heading</h2>
<p>Paragraph 2</p>
<p>Paragraph 3</p>
CSS to select the paragraph immediately following the h2 tag. The 3rd doesn't immediately follow so it's not selected. 
h2 + p {
  color: blue;
}
Result
Paragraph 1
Heading
Paragraph 2
Paragraph 3

General sibling selector (~)

The adjacent sibling selector selects all elements (with the same parent) that immediately follow that parent.

Ex 1

The following example selects all <p> elements that are placed immediately after <div> elements.

Paragraph 2 is not selected because it's inside the div, not following it.

Paragraphs 3 and 4 follow the parent div.

<p>Paragraph 1.</p>

<div>
  <p>Paragraph 2.</p>
</div>

<p>Paragraph 3.</p>
<p>Paragraph 4.</p>

div ~ p {
  color: white;
}
Result

Paragraph 1.

Paragraph 2.

Paragraph 3.

Paragraph 4.
