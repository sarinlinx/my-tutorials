:first-of-type

This is different from first-child since it doesn't matter what order the HTML is in.

The first type of the specified element will be selected.

Ex 1

Select the first paragraph elements in each parent.

p:first-of-type {
  color: red;
}

<div>
  <p>Paragraph 1</p>
</div>

<p>Paragraph 2</p>
<p>Paragraph 3</p>

<section>
  <p>Paragraph 4</p>
</section>
