Pseudo Class Selectors

pseudo selectors are keywords added after a selector with a double colon symbol.

They are used to define a special state of an element, such as a hover.

Example

Change the color when hovering over a link.

a:hover {
  color: green;
}

<p>Hover over this <a href="#">link</a> to see it change to green.</p>
Result

Hover over this link to see it change to green.
