:first-child

The :first-child pseudo-class matches a specified element that is the first child of another element.

This must be added to the child element.

Select the first and last child elements from the parent.

NOTE how the first-child and last-child selectors are added to the child elements, not the parent.

p:first-child {
  color: red;
}

<section>
<p>Paragraph</p> <!-- this is red -->
<p>Paragraph</p>
</section>
Result

:last-child

The :last-child selector matches every element that is the last child of its parent.

It looks for the last child within a section such as a div.

If there are no sections on the page, it locates the last child within the body tag.

This must be added to the child element.

p:last-child is equal to p:nth-last-child(1).

Ex 1: No sections in the HTML

Select the last paragraph child from the body tag.

Since there are no other sections on this page (such as <div> or <section>, the selector looks for the last paragraph in the body.

p:last-child {
  color: red;
}

<p>Paragraph 1</p>
<p>Paragraph 2</p>
<p>Paragraph 3</p>
Result

Paragraph 1

Paragraph 2

Paragraph 3

Ex 2: Paragraphs in sections in the HTML

Since there are other sections on this page (such as <div> or <section>, the selector does not look at the first paragraphs within the body tag. It only looks for paragraphs in each section type.

p:last-child {
  color: red;
}


<p class="first-last-child-pseudo-elements2">paragraph 1</p>
<p class="first-last-child-pseudo-elements2">paragraph 2</p>

</section>
