nth child selector

The nth child selector allows you to select one or more child elements based on the order within the parent container. You can do this in 3 ways:

  • keyword - odd or even
  • number of element
  • using a formula

Keywords

Use the keyword 'odd' or 'even' to select elements. This is good for tables.

<table class="mytable">
  <tr>
    <td>Row one. Odd row so it's colored.</td>
  </tr>
  <tr>
    <td>Row two. Even row so it's NOT colored.</td>
  </tr>
  <tr>
    <td>Row three. Odd row so it's colored.</td>
  </tr>
  <tr>
    <td>Row four. Even row so it's NOT colored.</td>
  </tr>
</table>

.mytable tr:nth-child(odd) {
  background-color: gray;
}

    

      

        Row one. Odd row so it's colored.
      

      

        Row two. Even row so it's NOT colored.
      

      

        Row three. Odd row so it's colored.
      

      

        Row four. Even row so it's NOT colored.

Using a number

You can also use a number to select a specific element.

<table class="mytable">
  <tr>
    <td>Row one. Odd row so it's colored.</td>
  </tr>
  <tr>
    <td>Row two. Even row so it's NOT colored.</td>
  </tr>
  <tr>
    <td>Row three. Odd row so it's colored.</td>
  </tr>
  <tr>
    <td>Row four. Even row so it's NOT colored.</td>
  </tr>
</table>

.mytable tr:nth-child(3) {
  background-color: gray;
}

    

      

        Row one. Not colored.
      

      

        Row two. Not colored.
      

      

        Row three is colored since it's the value specified.
      

      

        Row four. Not colored.

Using a formula

LOOK THIS UP

You can use a formula to specify the element.

This uses the formula 'an + b'.

<table class="mytable">
  <tr>
    <td>look this up</td>
  </tr>
  <tr>
    <td>look this up</td>
  </tr>
  <tr>
    <td>look this up</td>
  </tr>
  <tr>
    <td>look this up</td>
  </tr>
</table>

.mytable tr:nth-child(3n + 2) {
  background-color: gray;
}
