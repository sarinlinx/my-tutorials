Example using the 'type' attribute

HTML

<button type="button">Click Me!</button>

Add the attribute name in brackets to target it.

Attribute and value selector

This selects the target attribute only if the value is _blank. 
Make sure you <a href="#" target="_blank">Click Here</a>

Add the attribute and value names in brackets to target them.

[target=_blank] {
  color: green;
}

This would make only the link with a target of _blank green.
