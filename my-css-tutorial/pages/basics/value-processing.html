How CSS Values are Processed

CSS follows these stpes to determine which values are processed.

  • #1: Declared value (author declaration)
  • Cascaded value
  • Specified value: The default value in CSS if no cascaded value exists
  • Computed value: Values with relative units are converted
  • Used value: final calculations based on layout
  • Actual value: browser and device restrictions

The image below breaks down some code and how the values are calculated.

Unit Conversion

Percentages

% for font-size

Percentages are relative to the Parent's font-size.

% for length

Percentages are relative to the Parent's width if used to specify lengths.

Ex 1:

The header font size is 24px since it's 150% the size of its parent element.

html, body {
  font-size: 16px;
}

header {
  font-size: 150%;
}

&body>
  <header>
    <p>Paragraph section</p>
  </header>
</body>/

em and rem

These values are based off of font-size, even if they are in reference to something else.

This makes it eaiser to adjust responsive layouts.

em

em with fonts

When em is used with a font, the value is in reference to the parent element.

html, body {
  font-size: 16px;
}

/* 24px (16px * 2) */
header {
  font-size: 150%;
}

/* 72px (24px * 3) */
.header-child {
  font-size: 3em;
}

<p>Normal text 16px</p>

<header>
  <p>Header section 24px</p>
  <p class="header-child">Header Child 72px</p>
</header>
Result

Normal text 16px

Header section 24px

Header Child 72px

em with lengths

When em is used with a length, it's in reference to the current element's font size. 

html, body {
font-size: 16px;
}

header {
  /* 24px (16px * 2) */
  font-size: 150%;
  /* uses the font size as a reference (48px = 24px * 2) */
  padding: 2em;
}

/* 72px (24px * 3) */
.header-child {
  font-size: 3em;
}

<p>Normal text 16px</p>

<header>
  <p>Header section 24px</p>
    <p class="header-child">Header Child 72px</p>
</header>
Result

Normal text 16px

Header section 24px

Header Child 72px

rem

Always used root font size

Ex1

The margin-bottom would be 160px since this is in reference to the font-size of the root (which is 16px).

html, body {
  font-size: 16px;
}

header {
  font-size: 150%;
  /* 160px since root font-size is 16px */
  margin-bottom: 10rem;
}

<p>Normal text 16px</p>

<header>
  <p>Header section 24px</p>
    <p class="header-child">Header Child 72px</p>
</header>
Result

Normal text 16px

Header section 24px

Header Child 72px

vh (viewport height) and vw (viewport width)

1vh is 1% of the viewport height. So 90vh would be 90% of the current viewport height.

1vw is 1% of the viewport width
