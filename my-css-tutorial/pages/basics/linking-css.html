3 ways to link CSS to you page

There are 3 ways to link CSS to your page.

  • External style sheet (Recommended)
  • Internal style sheet
  • Inline

External style sheet (Recommended)

Linking from an external style sheet is the recommended way to load CSS into your website.

Link the external sheet within the <head> section of the HTML page.

type="text/css" isn't needed in HTML5

<head>
    <link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

Internal style sheet

You can add code to the <head> section within <style> tags.

However this can get difficult to read very quickly as the CSS grows.

<head>
<style>
body {
    background-color: linen;
}
</style>
</head>

Inline style

Inline styles are written directly into the body of the HTML page. Since there is no separation of content and style, this is an outdated method to add CSS and should never be used.

<h2 style="color:blue;margin-left:30px;">This is a heading.</h2>

Multiple Style Sheets

If some properties have been defined for the same selector (element) in different style sheets, the value from the last read style sheet will be used.

The <h2> elements will be "orange" since it was called last

<head>
<link rel="stylesheet" type="text/css" href="styles.css">

<style>
h2 {
  color: orange;
}
</style>
</head>

Cascading order

CSS will cascade in this order:

  • Inline style (inside an HTML element)
  • External and internal style sheets (in the head section)
  • Browser default
