The 4 link states

Hyperlinks can be styled differently depending on what state they are in.

a:link — a normal, unvisited link

/* unvisited link */
a:link {
  color: red;
}

a:visited — a link the user has visited

/* visited link */
a:visited {
  color: green;
}

a:hover — a link when the user mouses over it

/* mouse over link */
a:hover {
  color: hotpink;
}

a:active — a link the moment it is clicked

/* selected link */
a:active {
  color: blue;
}

Order or link pseudo classes

Link classes must be in this order in CSS in order to function.

  1. a:link must come first
  2. a:visited
  3. a:hover MUST come after a:link and a:visited
  4. a:active MUST come after a:hover

Make the link look like button

If you're using a class or ID, it must come after the a character.

<a class="link-btn" href="#">Click Here</a>

a.link-btn:link, a.link-btn:visited {
  background-color: #f44336;
  color: white;
  padding: 14px 25px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
}

a.link-btn:hover, a.link-btn:active {
  background-color: red;
}
