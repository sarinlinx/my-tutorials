Block Element Modifier (BEM)

A way to name classes in a consistent way.

Block

A standalone component that is meaningful on its own and can be reused anywhere in the project.

Element

Part of a block that has no meaning on its own. You cannot reuse these as they wouldn't make sense outside of the block.

Modifier

Something that modifies a block. Like a different look to a button.

Ex

BEM examples

.block {}
.block__element {}
.block__element--modifier {}

Ex

BEM examples

/* block__component */
header__logo-box
/* block__component--modifier */
header__logo-box--white
/* block--modifier */
heading-primary--main
heading-primary--sub
btn--white
btn--animated
