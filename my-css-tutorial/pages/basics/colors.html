Color codes

You can type in the name of the color, or use a HEX code. All major browsers support 140 colors.

  • 140 colors supported in all browsers 

p {
  background-color: color: white;
  background-color: color: #ffffff;
}

rgb colors

rgb color values specify Red, Green, and Blue.

{
  background-color: rgb(255, 0, 0); /*red*/
}

rgba colors

rgba color values specify an opacity value. The 'a' stands of 'Alpha Channel'.

The opacity parameter is specified as a number between 0 (transparent) and 1 (opaque).

{
  background-color: rgba(255, 0, 0, 0.3); /*red with opacity */
}

HSL colors

HSL stands for hue, saturation, and lightness. These values represent a cylindrical-coordinate representation of colors.

Hue

On a circular color wheel, 0 and 360 would represent red.

120 is green

240 is blue

Saturation

Saturation is a % value. 0% is a shade of gray. 100% is full color.

Lightness

Lightness is a % value. 0% is black. 100% is white. 50% is the normal color.

/* green: 50% is normal color */
.p1 {background-color: hsl(120, 100%, 50%);}

Green


/* light green: 75% makes it lighter */
.p2 {background-color: hsl(120, 100%, 75%);}

Light Green


/* dark green: 25% makes it darker */
.p3 {background-color: hsl(120, 100%, 25%);}

Dark Green


/* pastel green: 60% gives it a gray shade, 70% lightens it */
.p4 {background-color: hsl(120, 60%, 70%);}

Pastel Green

HSLA colors

HSLA colors add an opacity value (Alpha Channel), just like rgba colors.

Green

/* green with .3 opacity */
.p1 {background-color: hsla(120, 100%, 50%, 0.3);}

Light Green

/* light green: 75% lightens with .3 opacity */
.p2 {background-color: hsla(120, 100%, 75%, 0.3);}

Dark Green

/* dark green: 25% darkens with .3 opacity */
.p3 {background-color: hsla(120, 100%, 25%, 0.3);}

Pastel Green

/* pastel green: 60% gives it gray shade, 70% of lightens it, with .3 opacity */
.p4 {background-color: hsla(120, 60%, 70%, 0.3);}
