Recommended comment sections

There are a few standards to use when adding comment sections to your CSS code.

For example, if you have an entire section to style an unordered list, you should write comment block above so it's easy to view what that block is about:

/*____________________
  Lists
  ____________________*/
      
ul {
  /* styles */
}

Keyboard shortcuts to add comments

There is a shortcut to adding comments in HTML and CSS.

Windows
Ctrl + /
Mac
Cmd + /
