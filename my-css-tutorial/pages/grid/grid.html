The grid container

All CSS grid elements go inside of a container.

Assign display: grid; to it.

All direct children of the grid container automatically become grid items.

.grid-container {
  display: grid;
}
          
<div class="grid-container">
.....
.....
.....
.....
</div>

Creating columns

Create a specific amount of columns using grid-template-columns.

It defines the number of columns their width.

Use "auto" if all columns should have the same width.

Example

This creates 4 columns of different widths.

.grid-container {
  display: grid;
  grid-template-columns: 80px 200px 100px 40px;
}
          
<div class="grid-container">
.....
.....
.....
.....
</div>

Example

This creates 4 columns of equal widths.

.grid-container {
  display: grid;
  grid-template-columns: auto auto auto auto;
}
          
<div class="grid-container">
.....
.....
.....
.....
</div>

Example

This combines specific widths with auto widths.

.grid-container {
  display: grid;
  grid-template-columns: 80px 200px auto 100px;
}
          
<div class="grid-container">
.....
.....
.....
.....
</div>

Set row height

grid-template-row sets the height of each row.

You must specify a vaule for each row.

.grid-container {
  display: grid;
  grid-template-columns: auto auto auto auto;
  grid-template-rows: 80px 200px;
}
          
<div class="grid-container">
.....
.....
.....
.....
</div>

Horizontally align content

justify-content is used to align the whole grid inside the container.

The grid's total width has to be less than the container's width for the justify-content property to have any effect.

Example: center

This centers the grid inside of its container.

.grid-container {
    display: grid;
    grid-template-columns: auto auto auto auto;
    justify-content: center;
  }
            
  <div class="grid-container">
  .....
  .....
  .....
  .....
  </div>

Example: space-evenly

This gives the columns equal amount of space between, and around them.

.grid-container {
    display: grid;
    grid-template-columns: auto auto auto auto;

  }
            
  <div class="grid-container">

Gaps

The spaces between each column/row are called gaps.

  • column-gap: sets the gap between columns
  • row-gap: sets the gap between rows
  • gap: shorthand for both row-gap and column-gap

Example: column-gap

This sets a column-gap of 100px between each column.

.grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    column-gap: 100px;
  }
