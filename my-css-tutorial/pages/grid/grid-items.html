Grid items

A grid container contains grid items.

By default, a container has one grid item for each column, in each row.

You can style the grid items so they span multiple columns and/or rows.

Horizontally span items

grid-column defines on which column(s) to place an item.

  • Target an item by its id= or class= attribute.
  • Define where the item will start
  • Define where the item will end.

The grid-column property is a shorthand property for:

  • grid-column-start
  • grid-column-end

Example: start and end before

tem1 will start on column 1 and end before column 5:......

    display: grid;
    grid-template-columns: auto auto auto auto auto auto;
}
    
.item1 {
    grid-column: 1 / 5;
}

Specify a horizontal and vertical area

grid-area property can be used as a shorthand property for:

  • grid-row-start
  • grid-column-start
  • grid-row-end
  • grid-column-end

Example:

Item8 will start on row-line 1 and column-line 2,
and end on row-line 5 column-line 6

.grid-container {
  display: grid;

Name grid items

grid-area property can also be used to assign names to grid items.

Named grid items can be referred to by the grid-template-areas property of the grid container.

Each row is defined by apostrophes (' ')

The columns in each row is defined inside the apostrophes, separated by a space.

Example:

Item1, is called "myArea" and will take up the place of all five columns:

.grid-container {
  display: grid;
  grid-template-areas: 'myArea myArea myArea myArea myArea';
}
    
.item1 {
  grid-area: myArea;
}          


<div class="grid-container">
<div class="item1">1</div>
<div class="item2">2</div>
<div class="item3">3</div>  
<div class="item4">4</div>
<div class="item5">5</div>
<div class="item6">6</div>
</div>

Example: using periods to span columns

The div with a class of item1 is named "myArea" and spans 2 columns out of 5.

.grid-container {
  display: grid;
  grid-template-areas: 'myArea myArea . . .';
}   
    
.item1 {
  grid-area: myArea;
}          


<div class="grid-container">
<div class="item1">1</div>
<div class="item2">2</div>
<div class="item3">3</div>  
<div class="item4">4</div>
<div class="item5">5</div>
<div class="item6">6</div>
</div>

Example: arrange multiple sections

To create a grid for multiple sections of a page:

  1. Give each section an ID or Class name.
  2. For grid item, give it a property of grid-area with a specific area name.
  3. In the grid-container, the property grid-template-areas must list each area name inside single quotes.

The visual layout of the grid-template-areas property is how the webpage will look.

.grid-container {
  display: grid;
  grid-template-areas:
  'header header header header header header'
  'menu   main   main   main   right  right'
  'menu   footer footer footer footer footer';
}   

.item1 { 
    grid-area: header; 
}
.item2 { 
    grid-area: menu; 
}
.item3 { 
    grid-area: main; 
}
.item4 { 
    grid-area: right; 
}
.item5 { 
    grid-area: footer; 
}          


<div class="grid-container">
  <div class="item1">Header</div>
  <div class="item2">Menu</div>
  <div class="item3">Main</div>  
  <div class="item4">Right</div>
  <div class="item5">Footer</div>
</div>
