Overview

Web fonts are a CSS feature that allows you to specify font files to be downloaded along with your website as it is accessed, meaning that any browser that supports web fonts can have exactly the fonts you specify available to it.

There are two types of Web Fonts:

  • Internal
  • External

Internal web fonts

Internal web fonts are files you download.

You can then include them in your site just like an image or CSS file.

Declare and link to it using @font-face

@font-face {
  font-family: 'Museo Sans';
  src: url(museo-sans.tff);//font file name
}

You must then add it to the font stack in CSS like you would add any other.

body {
  font-family: 'Museo Sans', Arial, sans-serif;
}

File formats

Most browsers support WOFF and WOFF2 fonts

@font-face {
  font-family: 'Museo Sans';
  src: url(museo-sans.woff2) format('woff');
}

Using Google fonts

You can add a Google font by adding a special style sheet link in the <head> section and then refer to the font in the CSS.

  1. Visit fonts.google.com
  2. On the font page, click the + Select this style link. This adds the font to your drawer on the top right.
  3. Click the drawer on the top right to view your fonts and their code.
  4. Copy the link code into the <head> of your HTML page BEFORE your custom css file.
  5. Add the CSS declaration to your CSS file. Make sure to always add a fallback font.

Code added to the HEAD of the HTML file.

<link rel="preconnect" href="https://fonts.gstatic.com">
<link href="https://fonts.googleapis.com/css2?family=Krona+One&display=swap" rel="stylesheet">
  
<style>
  #google-font {
    font-family: 'Krona One', sans-serif;
  }
</style>

Multiple Google Fonts

To use multiple Google fonts, just separate the font names with a pipe character (|).

NOTE: This only works if the tag in the head is css (APIv1), not css2(APIv2). For example:

<link rel="preconnect" href="https://fonts.gstatic.com">
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Audiowide|Krona+One">

<style>
#google-font {
  font-family: 'Krona One', sans-serif;
}
#google-font2 {
  font-family: Audiowide, sans-serif;
}
</style>

Difference between Google Font tags

Google Fonts use either the API or APIv2 version

  • APIv1 (css)
  • APIv2 (css2)

Style Google Fonts

Use CSS to style your fonts.

<style>
#google-font {
font-family: 'Krona One', sans-serif;
font-size: 30px;
text-shadow: 3px 3px 3px #ababab;
}

Font Effects using the Google Font API

Google have also enabled different font effects that you can use. First add effect=effectname to the Google API, then add a special class name to the element that is going to use the special effect. The class name always starts with font-effect- and ends with the effectname.

View the API pages for correct syntax depending on which API version you're using.

  • APIv1 (css)
  • APIv2 (css2)

APIv1 example

<style>
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Sofia&effect=fire">

#google-font-effect {
  font-family: "Sofia", sans-serif;
  font-size: 30px;
}
</style>

<p id="google-font-effect" class="font-effect-fire">This uses a Google Font Effect APIv1</p>
