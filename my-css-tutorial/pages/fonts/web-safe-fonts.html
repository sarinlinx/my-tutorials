Web Safe Fonts

Web Safe fonts are supported on different operating systems and devices.

Because they are fully supported, it's a good idea to use these as fallback fonts, just in case the first font you tell teh browser to use is NOT supported.

View the following site for a list of Web Safe Fonts:

  • Web Safe Fonts

This means that you should add a list of similar "backup fonts" in the font-family property. If the first font does not work, the browser will try the next one, and the next one, and so on. Always end the list with a generic font family name.

Best Web Safe Fonts for HTML and CSS

The following list are the best web safe fonts for HTML and CSS:

  • Verdana (sans-serif)
  • Arial (sans-serif)
  • Helvetica (sans-serif)
  • Tahoma (sans-serif)
  • Trebuchet MS (sans-serif)
  • Times New Roman (serif)
  • Georgia (serif)
  • Garamond (serif)
  • Courier New (monospace)
  • Brush Script MT (cursive)

Here, there are three font types: Tahoma, Verdana, and sans-serif. The second and third fonts are backups, in case the first one is not found.

p {
  font-family: Tahoma, Verdana, sans-serif;
}

Commonly Used Font Fallbacks

Below are some commonly used font fallbacks, organized by the 5 generic font families:

Serif Fonts

Examples of Serif fallback fonts

body {
  font-family: "Times New Roman", Times, serif;
}

body {
  font-family: Georgia, serif;
}

body {
  font-family: Garamond, serif;
}

Serif Fonts

Examples of Sans-Serif fallback fonts

body {
  font-family: Arial, Helvetica, sans-serif;
}

body {
  font-family: Tahoma, Verdana, sans-serif;
}

body {
  font-family: "Trebuchet MS", Helvetica, sans-serif;
}

body {
  font-family: Georgia, Verdana, sans-serif;
}

Monospace

Examples of Monospace fallback fonts

body {
  font-family: "Courier New", Courier, monospace;
}

Cursive

Examples of Cursive fallback fonts

body {
  font-family: "Brush Script MT", cursive;
}

Fantasy

Examples of Fantasy fallback fonts

body {
  font-family: Copperplate, Papyrus, fantasy;
}
