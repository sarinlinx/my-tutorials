Unordered Lists

You can style how lists look depending on if they are unordered or ordered.

unordered lists
<ul class="a">
  <li>Coffee</li>
</ul>
<ul class="b">
  <li>Coffee</li>
</ul>

ul.a {
  list-style-type: circle;
}

ul.b {
  list-style-type: square;
}

Add a custom image for the list items

Example of ordered lists.

<ol>
 <li>Coffee</li>
</ol>

ul {
  list-style-image: url('sqpurple.gif');
}

Shorthand for all properties on a custom image. The following can be used:

ul {
  list-style: square inside url("sqpurple.gif");
}

list-style-type (if a list-style-image is specified, the value of this property will be displayed if the image for some reason cannot be displayed). View the following for a full list of types:

  • list-style-type options

list-style-position (specifies whether the list-item markers should appear inside or outside the content flow)

list-style-image (specifies an image as the list item marker)

If one of the property values above are missing, the default value for the missing property will be inserted, if any.
