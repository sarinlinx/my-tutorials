Make a menu horizontal

There are a few ways to do this.

  • Creating an html section with anchor tags, then using inline-block to force them horizontal
  • Creating an html unordered list, then using inline-block to force them horizontal
  • Using float

Nav section with anchors using display: inline-block

This first example does not use a display property

Wrap the menu in a <nav> tag with a class name.

You can then pinpoint the anchor tags within it.

This example does not use a display property, so it defaults to display: inline.

<nav class="display-ex">
  <a href="#">About</a>
  <a href="#">Contact</a>
</nav>

.display-ex {
  background-color: green;
  text-align: center;
}

.display-ex a {
  background-color: yellow;
  border: 1px solid red;
  padding: 10px 0px; /*add size to links*/
}

Example:

AboutContact

Notice how the green background does not expand to fit the padding of the links. This is default behavior for inline elements. To fix, just make the anchor links inline-block.

This example uses a display:inline-block

Add display: inline-block;

This forces the background (parent element) to expand to the child's size.

.display-ex {
  background-color: green;
  text-align: center;
}

.display-ex a {
  background-color: yellow;
  border: 1px solid red;
  padding: 10px 0px; /*add size to links*/
  display: inline-block; /*to expand parent element with child*/
}

Notice how there is a space between the anchor tags? That is because the HTML for each link is on a different line.

Instead of changing your HTML, just change the font to zero for the parent element, then add it for the child element.

Remove space between anchors by setting font-size: 0;

Set parent font to zero.

Set child font (otherwise it would also be zero since it would inherit the parent font).

<nav class="display-ex">
  <a href="#">About</a>
  <a href="#">Contact</a>
</nav>

.display-ex {
  background-color: green;
  text-align: center;
  font-size: 0;
}

.display-ex a {
  background-color: yellow;
  border: 1px solid red;
  padding: 10px 0px; /*add size to links*/
  display: inline-block;
  font-size: 16px;
}

Create horizontal nav using an unordered list

You can do the same as shown above using an unordered list.

<ul class="display-ex">
  <li><a href="#">About</a></li>
  <li><a href="#">Contact</a></li>
</ul>

.display-ex {
  background-color: yellow;
  list-style-type: none;
  text-align: center;
  margin: 0; /*necessary to keep links from overlapping*/
  padding: 0; /*necessary to keep links from overlapping*/
}

.display-ex li {
  display: inline-block;
  background-color: green;
  font-size: 20px;
  padding: 20px;
}

Make a menu horizontal using the 'float' property

Using display:inline-block can be achieved with float, but floating requires you to clear the element after the floating boxes. So inline-block is easier.

Also, when using float, the display property has no affect.

Background colors only

<nav class="float">
  <a href="#">About</a>
  <a href="#">Contact</a>
</nav>

.float {
  background-color: yellow;
}

.float a {
  background-color: red;
}

Original appearance BEFORE adding float:left; to anchor tag:


  About Contact

Addding float: left; causes the spacing to dissapear, but also causes the parent element (in yellow) to dissapear.

<nav class="float">
  <a href="#">About</a>
  <a href="#">Contact</a>
</nav>

.float {
  background-color: yellow;
}

.float a {
  background-color: red;
  float: left;
}

Example: The parent container does not recognize the height of floated child elements and collapses to a height of zero.


  About Contact

To restore parent element size, add overflow:hidden;

<nav class="float">
  <a href="#">About</a>
  <a href="#">Contact</a>
  </nav>

  .horz-nav-float-overflow {
  background-color: yellow;
  overflow: hidden;
  }
  .horz-nav-float-overflow a {
  background-color: red;
  float: left;
}

If you apply float to an inline element, the browser automatically switches display of element to block. This means you can add padding without adjusting the display value.

Center a horizontal menu using the 'float' property

text-align: center does NOT work with float you must set width. Divide number of links by a percent.

For example, for two links use 50%, for 4 links use 25%.

<nav class="float">
  <a href="#">About</a>
  <a href="#">Contact</a>
  <a href="#">Email</a>
  <a href="#">Links</a>
</nav>

.float a {
  background-color: red;
  float: left;
  width: 25%;
}

This works best with 4 or more links.

Create a horizontal list from an unordered list

Everything above applies the same to unordered lists that you're trying to make a navigation menu from. Just make sure to remove the style from the list.

.ul-class {
  list-style-type: none;
