Icon with gradient

Add the icon library in a link in the <head> section of the HTML page.

You can then add the icons and CSS as shown below.

Use a linear-gradient with the -webkit-background-clip and color: transparent properties to give the icon a gradient.

.feature-box__icon {
  font-size: 6rem;
  background-image: linear-gradient(to right, #7ed56f, #03aef1);
  -webkit-background-clip: text;
  color: transparent;
}
