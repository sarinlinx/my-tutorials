There is a FREE version with 1588 icons, or a Pro version with 7842 icons.

2 ways to add Font Awesome

Option 1 - Download the library

Download the library

When you download the library, several folders and files are downloaded.

  • /css - Stylesheets for Web Fonts (main file is all.css) - this file contains the core styling ( and additional files for all of Font Awesome’s style options - solid, regular, light, and brands.), plus all of the icon styles that you’ll need when using Font Awesome.
  • /js - SVG with JavaScript (main file is all.js) - loads all of the base features, plus all of the icon styles that you’ll need when using Font Awesome.
  • /less - Less pre-processor (main file is fontawesome.less)
  • /scss - Sass pre-processor (main file is fontawesome.scss)
  • /sprites - SVG sprites (main file is solid.svg)
  • /svgs - Individual SVG for each icon (main file are individual *.svg icons)
  • /webfonts - Web Font files used with CSS (See /css) - contains all of the typeface files that the above CSS references and depends on.

Load all styles and JS

all.css

Copy both the /webfonts and the /css folders into your project’s static assets directory

Add a reference to the /css/all.css file into the <head> of each template or page that you want to use Font Awesome on.

<link href="/your-path-to-fontawesome/css/all.css" rel="stylesheet">

all.js

Add a reference to the /js/all.js file into the <head> of each template or page that you want to use Font Awesome on.

<script defer src="/your-path-to-fontawesome/js/all.js"></script>

Using Just Certain Styles

The core styling files is /css/fontawesome.css

Copy both the /webfonts and the /css folders into your project’s static assets directory

Add a reference to the /css/fontawesome.css file into the <head> of each template or page that you want to use Font Awesome on.

<link href="/your-path-to-fontawesome/css/fontawesome.css" rel="stylesheet">

Add a reference to individual style files such as brands.css and solid.css.

<link href="/your-path-to-fontawesome/css/brands.css" rel="stylesheet">
<link href="/your-path-to-fontawesome/css/solid.css" rel="stylesheet">

fontawesome.js

Repeat the steps above for specific files using /js/fontawesome.js

<script defer src="/your-path-to-fontawesome/js/fontawesome.js"></script>
<script defer src="/your-path-to-fontawesome/css/brands.js"></script>
<script defer src="/your-path-to-fontawesome/css/solid.css"></script>

Option 2 - Sign up for a KIT code

Sign up.

Get your Kit Code number.

You then insert a <script> tag at the top of your HTML page.

The KIT code is placed in the head of the HTML page and looks like this:

<script src="https://kit.fontawesome.com/yourcode.js" crossorigin="anonymous"></script>

Using Font Awesome

Prefixes

Font Awesome 5 uses fas or far.

  • fas stands for solid
  • far stands for regular
  • fab stands for brands

Font Awesome 4 uses fa.

HTML

  • Basic use

Add the code to an inline element.

<i class="fas fa-camera"></i>
<span class="fas fa-camera"></span>
<i class="fab fa-font-awesome"></i>


CSS

font-weight

Only solid (font-weight: 900;) and brands (font-weight: 400;) are supported on the free plan.

But you can use pseudo styling to customize further.

  • CSS Pseudo-elements

inherited styles

If you change the font-size or color of the icon's container, the icon changes. Same things goes for shadow, and anything else that gets inherited using CSS.

Font Awesome 4

You can use Font Awesome 4 by placing the following code in the head of your HTML file.

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

Then insert the Font Awesome code into an inline element.

<i class="fa fa-car"></i></span>
