Setting a background image

Add a background image.

NOTE: Double quotes are needed. Single quotes do not work.

body {
  background-image: url("paper.gif");
}

To specify multiple images, separate the URLs with a comma.

This could be used to layer a small image over a wallpaper background image

body {
  background-image: url("image1.png"),
  url("image2.png");
  background-repeat: no-repeat, repeat;
}

Example of 2 images

The first image only displays once.

 

 

 

 

Different background images for different displays

If you have a high resolution display (retina) you may want to display a different image than a smaller screen with less resolution.

Use a media query.

header {
  background: url(image.jpg);
}

@media (min-resolution: 192dpi) {
  header {
  /* Using a larger image for retina screens */
  background: url(image@2x.jpg);
  }
}

                

            












    

        
Shorthand

        

            

                
You can use shorthand to list all background CSS properites in a single declaration.

            

        


        

            

                
When using the shorthand property, the recommended order of the property values is:

                
    
                    
  • background-color
    • 
                    
  • background-image
    • 
                    
  • background-repeat
    • 
                    
  • background-attachment
    • 
                    
  • background-position
    • 
                

                
But the order doesn't matter.

                
If you mix shortland with full declarations, shorthand will override.

            



            

                
body {
  background: #ffffff url("img_tree.png") no-repeat scroll right top;
}

        


        

            

                
If you include background-size, it MUST be after background-position and separated with /.

            


            

                
body {
  background: url(image.png) no-repeat fixed 0px 0px / cover;
}

        

    








    

        
Shorthand for multiple images

        

            

                
Longhand for multiple image styles.

            



            

                
body {
  background-image: url(image1.png), url(image2.png);
  background-repeat: no-repeat, no-repeat;
  background-position: top right, bottom left;
}

        


        

            

                
Shorthand for multiple image styles.

            


            

                
body {
  background: url(image1.png) no-repeat top right,
              url(image2.png) no-repeat bottom left;
}
