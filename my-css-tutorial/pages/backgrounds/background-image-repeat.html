Background repeat

By default, the background-image property repeats an image both horizontally and vertically.

Repeat the image on the x axis

This repeats the image horizontally.

body {
  background: url("../img/purple-dot.jpg");
  background-repeat: repeat-x;
}

The image is repeated horizontally only. If the image has a short height, anything below it inherits its background color.

Repeat the image on the y axis

This repeats the image vertically.

body {
  background: url("../img/purple-dot.jpg");
  background-repeat: repeat-y;
}

The image is repeated horizontally only. If the image has a short width, anything to the right inherits its background color.

Do not repeat the image

This forces the image to not repeat at all.

body {
  background: url("../img/purple-dot.jpg");
  background-repeat: no-repeat;
}

The result is that the image shows once in the top left corner.

Add space between the repeated images

The background-repeat: space; property repeats as much as possible without clipping. The first and last images are pinned to either side of the element, and whitespace is distributed evenly between the images

body {
  background: url("../img/purple-dot.jpg");
  background-size: 50px;
  background-repeat: space;
}

Evenly distributed space is added between the images.

squish or stretch (no gaps)

The background-image is repeated and squished or stretched to fill the space (no gaps)

body {
  background: url("../img/purple-dot.jpg");
  background-size: 50px;
  background-repeat: round;
}

No gaps.
