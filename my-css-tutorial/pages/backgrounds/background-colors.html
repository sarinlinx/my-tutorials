Opacity

The opacity property specifies the opacity/transparency of an element. It can take a value from 0.0 - 1.0. The lower value, the more transparent:

When using the opacity property to add transparency to the background of an element, all of its child elements inherit the same transparency. This can make the text inside a fully transparent element hard to read.

.bg-opacity-1 {
  background-color: green;
  opacity: 0.1;
}

.bg-opacity-3 {
  background-color: green;
  opacity: 0.3;
}

.bg-opacity-6 {
  background-color: green;
  opacity: 0.6;
}

.bg-opacity-0 {
  background-color: green;
}

Opacity RGBA

If you do not want to apply opacity to child elements, like in our example above, use RGBA color values. The following example sets the opacity for the background color and not the text:

.bg-opacity-rgba-1 {
    background: rgba(0, 128, 0, 0.1)
}

.bg-opacity-rgba-3 {
    background: rgba(0, 128, 0, 0.3)
}

.bg-opacity-rgba-6 {
    background: rgba(0, 128, 0, 0.6)
}

.bg-opacity-rgba-0 {
    background: rgba(0, 128, 0, 1)
}

Background transparency

If there is already a background added, you can add a second layer over it. You can then adjust this second layer's transparency value.

Make sure this is NO SPACE between rgba and the parenthese.

<div class="image-bg">
  <p class="rgba-transparency">Testing to see how the transparency works.</p>
</div>

.image-bg {
  background: url("images/note.jpg")
}

.rgba-transparency {
  background: rgba(199, 21, 133, .25);
}

The result will be a transparent layer over top the current background.
