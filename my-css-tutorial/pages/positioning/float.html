Float

The float property positions content in a container.

It can be used to wrap text around images.

You can float to the left or right.

Float right

The image will float to the right in the paragraph, and the text in the paragraph will wrap around the image.

#float-right {
  float: right;
}

Float divs

Normally div elements will be displayed on top of each other. However, if we use float: left we can let elements float next to each other:

.div-class {
  float: left;
  padding: 15px;
}

#div1 {
  background: red;
}

#div2 {
  background: yellow;
}

#div3 {
  background: green;
}

<div class="div-class" id="div1">Div 1</div>
<div class="div-class" id="div2">Div 2</div>
<div class="div-class" id="div3">Div 3</div>

clear

The clear property specifies what elements can float beside the cleared element and on which side.

The clear property can have one of the following values:

  • none - Allows floating elements on both sides. This is default
  • left - No floating elements allowed on the left side
  • right- No floating elements allowed on the right side
  • both - No floating elements allowed on either the left or the right side
  • inherit - The element inherits the clear value of its parent

When clearing floats, you should match the clear to the float

If an element is floated to the left, you should clear to the left. Your floated element will continue to float, but the cleared element will appear below it on the web page.

Example

Without clear

The following example clears the float to the left. Means that no floating elements are allowed on the left side (of the div)

#div1-clear {
  float: left;
  width: 100px;
  height: 50px;
  margin: 10px;
  border: 3px solid #73AD21;
}

#div2-clear {
  border: 1px solid red;
}

<div id="div1-clear">div1</div>
<div id="div2-clear">div2 - div2 is after div1 in the HTML code. However, since div1 floats to the left, the text in div2 flows around div1.</div>

With clear

The following example clears the float to the left. Means that no floating elements are allowed on the left side (of the div)

.div3 {
  float: left;
  width: 100px;
  height: 50px;
  margin: 10px;
  border: 3px solid #73AD21;
}

.div4 {
  border: 1px solid red;
  clear: left;
}

<div class="div3">div3</div>
<div class="div4">div4 - Here, clear: left; moves div4 down below the floating div3. The value "left" clears elements floated to the left. You can also clear "right" and "both".</div>

The clearfix hack

If an element is taller than the element containing it, and it is floated, it will "overflow" outside of its container:

Old clearfix hack

add overflow: auto; to the containing element to fix this problem:

Without clearfix

The image is taller than the element containing it, and it is floated, so it overflows outside of its container:

  border: 3px solid #4CAF50;
  padding: 5px;
}

#img1-clearfix {
  float: right;
}

#img2-clearfix {
  float: right;
