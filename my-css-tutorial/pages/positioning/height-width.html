Overview

The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element.

Height and Width

Single container

Here's a basic container with set height and width.

#div1 {
  height: 200px;
  width: 50%;
}

Example

Height is 200px. Width is 50% of the container.

Container inside a container

This shows a container in a container. The 50% is now 1/2 of the container it's in.

#div1 {
  height: 200px;
  width: 50%;
  color: #000;
}

#div2 {
  height: 200px;
  width: 50%;
}

Example

Height is 200px. Width is 50% of the container.
Height is 200px. Width is 50% of the container.

Container inside a container #2

This shows a container in a container. 50% height and width of parent container.

#div1 {
  height: 200px;
  width: 50%;
  color: #000;
}

#div3 {
  height: 50%;
  width: 50%;
}

Example with text in parent
