Equal width grid boxes

Without padding or borders

This creates 3 equal sized boxes side by side.

.box1 {
  float: left;
  width: 25%;
}
 
<div class="box1" style="background-color:#bbb">
  <p>Some text inside the box.</p>
</div>

<div class="box1" style="background-color:#ccc">
  <p>Some text inside the box.</p>
</div>

<div class="box1" style="background-color:#ddd">
  <p>Some text inside the box.</p>
</div>

Adding border-box property to fix the broken float caused by adding padding.

This creates 3 equal sized boxes side by side.

But the padding breaks the float.

Adding border-box fixes it.

Add the clearfix hack to take care of the page flow.


* {
  box-sizing: border-box;
}

.box1 {
float: left;
width: 25%;
padding: 50px;
}

.clearfix::after {
  content: "";
  clear: both;
  display: table;
}

<div class="clearfix">
  <div class="box1" style="background-color:#bbb">
  <p>Some text inside the box.</p>
  </div>

  <div class="box1" style="background-color:#ccc">
  <p>Some text inside the box.</p>
  </div>

  <div class="box1" style="background-color:#ddd">
  <p>Some text inside the box.</p>
  </div>
</div>

Equal height grid boxes

If you want to used float with equal height boxes, you must set a fixed height for them.

This is not flexible as the text will overflow the container when the page size is adjusted.

The better option is to use flex-wrap: nowrap;

.box {
  float: left;
  width: 50%;
  padding: 50px;
  height: 300px;
}

Float first letter of paragraph

Add a CSS span tag to the first letter. Then use float: left; to make it stand out.

#letter {
  float: left;
  width: 0.7em;
  font-size: 400%;
  font-family: algerian, courier;
  line-height: 80%;
}
