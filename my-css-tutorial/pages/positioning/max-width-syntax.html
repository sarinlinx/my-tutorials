max-width

Sets the maximum width of an element.

It can be set using values like px or %.

When should it be used?

max-width overrides width.

It's only useful if you combine a set width with max-width. That way, the section will have a specific width at first. As you shrink, max-width allows the container to also shrink while never becoming smaller than the % you set.
