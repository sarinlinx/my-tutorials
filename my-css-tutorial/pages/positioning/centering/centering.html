Center a section

To horizontally center a block element (like <div>), use margin: auto;

Make sure to also add a width. Center aligning has no effect if the width property is not set (or set to 100%).

Adding a width will cause the remaining space to be split equally in both margins.

.center {
  margin: auto;
  width: 60%;
}

Center horizontally from the parent element

To just center the text inside an element, use text-align: center;

If that parent element is set to text-align: center, the text in the block element will be centered.

This works for block or inline text element inside the parent.

.ex-02 {
  width: 60%;
  border: 1px solid red;
  text-align: center;
}

<div class="ex-02">
  <p>Normal paragraph with no styles</p>
</div>

<div class="ex-02">
  <span>Normal paragraph with no styles</span>
</div>

Text without a parent

text-align: cetner works with a block element if it does not have a parent.

This will NOT work with inline elements. inline elements require a parent in order to cetner the text.

#blocktext {
  border: 1px solid red;
  text-align: center;
}

#inlinetext {
  border: 1px solid red;
  text-align: center;
}

<p id="blocktext">Normal paragraph with no styles</p>

<span id="inlinetext">Span text with no styles</span>

Center with a utility class

Create a utility class that can be reused.

.u-center-text {
  text-align: center;
  width: 60%;
  border: 1px solid red;
}

<div class="u-center-text">
  <p>Normal paragraph</p>
</div>

Center an image

set left and right margin to auto and make it into a block element:

.center-img {
  display: block;
  margin-left: auto;
  margin-right: auto;
  width: 40%;
}
