SVGs

Best practices

SVGs are preferred over icon fonts as they are more reliable and work with screen-readers.

You can convert an icon font to an SVG at this site.

  • IcoMoon.com
  • Free libraries

Download SVGs

  1. Visit the libraries page.
    • Free libraries
  2. Click the 'ADD' button under a library.
  3. Select the icons you want.
  4. Click 'Generate SVG and more' button.
  5. Click the gear icon at the bottom to view settings.
  6. Un-check png
  7. Click download
  8. Create a new directory in your project where you'll unzip this file.
  9. Unzip. Several files and folders are unzipped.
  10. You only need the SVG files in the /SVG directory and a file named symbol.defs.svg. This is a sprite file which contains all the SVGs in it. This allows you to include 1 HTTP request instead of including 10 individual files. There's also a demo file you can use to view the SVGs and their names.
  11. Rename the symbol.defs.svg file sprites.svg
  12. Copy these files into your main project folder /img directory.

Include the SVG in your HTML

Add an svg tag

Inside, add <use xlink:href= to include the SVG file.

Link to the sprite.svg file, then a #, and finally the specific SVG file

NOTE: You'll only be able to view the xlink file on a web server

<form action="#" class="search">
  <input type="text" class="search__input" placeholder="Search hotels">
  <button class="search__button">
    <svg class="search__icon">
      <use xlink:href="img/sprite.svg#icon-magnifying-glass"></use>
    </svg>
  </button>
</form>

Style the SVG in CSS

Resize the SVG.

.search__icon {
  height: 2rem;
  width: 2rem;
  fill: #999;
}
