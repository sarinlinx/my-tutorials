Image size

Images should be twice the size as the "correct" size on the page. This is for high-resolution displays (such as retina)

Reduce image size

To adjust the size of an image, add a class. Then, in CSS, add only the width value. The height automatically adjusts to aspect-ratio.

<img class="blah" asdf>

.blah {
  width: 300px;
}

Reshape image

Add border radius. Make sure you start with a square image or it will turn it into an oval.

img {
  border-radius: 50%;
}

Here's an example using pixels.

img {
  border-radius: 22px;
}

Center an image

Use margin:auto to center and image.

img {
  display: block;
  margin: 0 auto;
}

Here's an example using pixels.

img {
  display: block;
  margin: 0 50%;
}

EX3

img {
  max-width: 100%;
  height: auto;
}

Response images

If you want an image to scale down if it has to, but never scale up to be larger than its original size.

img {
  max-width: 100%;
  height: auto;
}

Card effect

This adds a container around and below the image to look like a Polaroid (card) image.

#polaroid {
  width: 50%;
  background-color: white;
  box-shadow: 0 0 10px 10px rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
  margin-bottom: 25px;
  margin: 0 auto;
}

#container {
  text-align: center;
  padding: 10px 20px;
}

<div class="polaroid">

  <img src="lights600x400.jpg" alt="Norther Lights" style="width:100%">
  
  <div class="container">
  <p>Northern Lights</p>
  </div>
  
</div>

Image Opacity

The opacity property can take a value from 0.0 - 1.0. The lower value, the more transparent:

#opacity1 {
  opacity: 0.1;
}

#opacity2 {
  opacity: 0.5;
}

#opacity31 {
  opacity: 1;
}
