clip-path

lets you clip an element to a basic shape or to an SVG source.

You use coordinates as percentages to draw the shape.

Online tool

This tool lets you create a shape and gives you the code.

  • Clippy

Circle

This cliips a square image into a circle.

Clipped at 50%

#clip-circle-50 {
  clip-path: circle(50%);
}

Clipped at 20%

#clip-circle-20 {
  clip-path: circle(20%);
}

Ellipse

This clips a square image into an ellipse.

vertical ellipse

The first value is horizontal

The 2nd value is vertical

The last two center it.

#ellipse {
  clip-path:ellipse(25% 40% at 50% 50%);
}

Polygon

The polygon can be any shape.

Use percentages to adjust the shape.

Polygon shape

Syntax: clip-path: polygon( x y, x y, x y)

The first is left top

The 2nd is right top.

3rd is right bottom.

4th is left bottom.

#polygon {
  clip-path: polygon(0 0, 100% 0, 100% 75%, 0 100%);
}

Diamond

The first value is left 50% at the top

The 2nd value is right and bottom 50%.

The 3rd value right 50% at the bottom

The 4th value is left at bottom 50%.

#diamond {               
  clip-path:polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
}

Triangle

Only 3 coordinates.

The first value it top center.

The 2nd value is left bottom.

The 3rd value is right bottom.

#triangle {               
  clip-path:polygon(50% 0%, 0% 100%, 100% 100%);
}
