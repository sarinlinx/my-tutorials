Graceful Degradation

It's important to check if the newer CSS you're using will work on older browsers.

Start by checking caniuse.com

If a browser doesn't support the newer CSS, you can apply older CSS for older browsers. This is called 'Graceful Degradation'.

The @supports rule

This example uses @supports to check if the browser supports the backdrop-filter property.

.popup {
  height: 100vh;
  width: 100%;
  position: fixed;
  top: 0;
  left: 0;
  background-color: rgba(0, 0, 0, 0.8);
  z-index: 9999;
  opacity: 0;
  visibility: hidden;
  transition: all .3s
}
              
/* feature query. Exclude from a different browser that doesn't support it
This says what to do IF the propert is supported */
@supports (-webkit-backdrop-filter: blur(10px)) or (backdrop-filter: blur(10px)) {
  .popup {
      -webkit-backdrop-filter: blur(10px);
      backdrop-filter: blur(10px);
      background-color: rgba(255, 255, 255, 0.3)
  }
}
