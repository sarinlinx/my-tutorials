Flexbox mask example

On the list, use flex-wrap: wrap; to break items onto next line.

On the list items, use flex: 0 0 50%; so the line splits into two columns both 50% wide.

Use the ::before pseudo element to add the SVG

 body {
  background-color: rgb(126, 126, 126);
}

.container {
  color: rgb(53, 53, 53);
  background-color: rgb(233, 233, 233);
  padding: 3rem;
  -webkit-box-flex: 0;
  -ms-flex: 0 0 60%;
  flex: 0 0 60%;
  margin-right: 4.5rem;
}

.list {
  list-style: none;
  margin: 3rem 0;
  padding: 3rem 0;
  display: -webkit-box;
  display: -ms-flexbox;
  display: flex;
  -ms-flex-wrap: wrap;
  flex-wrap: wrap;
}

.list__item {
  -webkit-box-flex: 0;
  -ms-flex: 0 0 50%;
  /* 50% so elements are side by side */
  flex: 0 0 50%;
  margin-bottom: .7rem;
}

.list__item::before {
  content: "";
  display: inline-block;
  height: 1rem;
  width: 1rem;
  margin-right: .7rem;
  /* older browsers  */
  background-image: url(img/chevron-thin-right.svg);
  background-size: cover;
}

Flexbox mask example

This uses mask-image and mask-size to see through the SVG to the background color.

.list__item2::before {
  content: "";
  display: inline-block;
  height: 1rem;
  width: 1rem;
  margin-right: .7rem;
  background-color: rgb(255, 76, 22);
  -webkit-mask-image: url(img/chevron-thin-right.svg);
  /* so icon covers entire element */
  -webkit-mask-size: cover;
  mask-image: url(img/chevron-thin-right.svg);
  mask-size: cover;
  background-image: none;
}

You can also add both and use @supports to check if the browser supports the newer code.

Must set background-image to none for the @supports to work.


.list__item {
  -webkit-box-flex: 0;
  -ms-flex: 0 0 50%;
  flex: 0 0 50%;
  margin-bottom: .7rem;
}

.list__item::before {
  content: "";
  display: inline-block;
  height: 1rem;
  width: 1rem;
  margin-right: .7rem;
  background-image: url(../img/chevron-thin-right.svg);
  background-size: cover;
}

@supports ((-webkit-mask-image: url()) or (mask-image: url())) {
  .list__item::before {
    background-color: var(--color-primary);
    -webkit-mask-image: url(../img/chevron-thin-right.svg);
    -webkit-mask-size: cover;
    mask-image: url(../img/chevron-thin-right.svg);
    mask-size: cover;
    background-image: none;
  }
}

This example creates a mask over a background color.

You can then look through the mask to see the background color.
