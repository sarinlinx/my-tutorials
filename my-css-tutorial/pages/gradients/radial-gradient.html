Radial gradients

radial-gradient move from the center out in a circle.

Syntax

background-image: radial-gradient(shape size at position, start-color, ..., last-color);
  • shape: Defines the shape. Either ellipse (default) or circle
  • size: Defines the size. Values are: farthest-corner (default), closest-side, closest-corner, farthest-side
  • postition: default is center

Examples

Ex

This shows different colors and the % where they change.

<p class="myClass">Testing to see how the radial gradient works.</p>

.mydiv {
  background-image: radial-gradient(red 5%, green 15%, blue 60%);
}

The Ellipse shape is default if not specified.

Different keywords

The size parameter defines the size of the gradient. It can take four values:

  • closest-side
  • farthest-side
  • closest-corner
  • farthest-corner
