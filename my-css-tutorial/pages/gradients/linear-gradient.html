Linear gradients

Syntax

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

direction: Default is top down. You can also specify different directions it should move in. For example:

  • to right
  • to bottom
  • to bottom right
  • 180deg

Stops: To create a linear gradient you must define at least two color stops. Color stops are the colors you want to render smooth transitions among. This value consists of a color value, followed by an optional stop position (a percentage between 0% and 100% or a length along the gradient axis)

Word direction

Top down

This shows how to add a background of 2 colors that blend into each other.

This linear gradient starts at the top and move down.

<p class="linear-gradient">Testing to see how the linear gradient works.</p>

.linear-gradient {
  background: linear-gradient(red, green)
}

The result will be a transparent layer over top the current background.

Transparent linear gradient over a background image

This shows how to add a transparent linear gradient of 3 colors over a background image.

The linear-gradient values must come first.

<div class="image-bg">
  <p class="transparent-linear-gradient">Transparent linear gradient over an background image.</p>
<div>


.transparent-linear-gradient {
  height: 40px;
  background: linear-gradient(rgba(255, 0, 0, 0.35),  /*red*/
  rgba(255, 255, 0, 0.35), /*yellow*/
  rgba(0, 0, 255, 0.35)),  /*blue*/
  url("images/backgrounds/examples/white-bg.jpg");
}

The result will be a transparent gradient over a background image.
