Linear gradients

Syntax

background-image: repeating-linear-gradient(angle | to side-or-corner, color-stop1, color-stop2, ...);

Same as linear-gradient, but repeats a gradient within a section

Examples

45 deg

This repeats the gradient on a 45deg angle.

<p class="myClass">Testing to see how the linear gradient works.</p>

.mydiv {
  background-image: repeating-linear-gradient(45deg,red,blue 7%,green 10%);
}
