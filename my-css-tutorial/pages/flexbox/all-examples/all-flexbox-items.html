align-self

Similar to align-items for the container, but only for a single item

This overrides the flexible container's align-items property.

align-self: auto

Default. The element inherits its parent container's align-items property, or "stretch" if it has no parent container

The container sets: align-items: center, so align-self inherits this and becomes align-self: center.

3 Width Properties

flex-grow

defines how much an item can grow

Specify # to grow item.

Add flex-grow: 1; to all items so items grow to as much as they can.

The # is in relation to other elements.

So, if you set flex-grow: 1; on all items, and then add flex-grow: 2; to a specific item, that item will grows 2x the size of the other items.

NOTE: The items CSS section must come before the specific item flex-grow property. If not, it will not be read.

flex-shrink

defines how much an item can shrink

Initial value is 1 which means it's allowed to shrink.

Set it to zero so it's NOT allowed to shrink.

flex-shrink: 0

Each item has a 100px width.

The container is 150px.

Since item 2 is set to flex-shrink: 0, it's size does NOT shrink.

1
2

flex-shrink: 1

Each item has a 100px width.

The container is 150px.

Since item 2 is set to flex-shrink: 1, it's size shrinks.
