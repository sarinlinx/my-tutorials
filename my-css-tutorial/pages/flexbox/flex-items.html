Flex items

align-self

Overrides the align-contents property for a single item.

This example aligns all items across the CROSS access first in the container using align-items: center;

Item with class i4 then overrides this with align-self: end;

Other options are:

  • flex-start
  • stretch

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

.i2 {
  height: 200px;
}

.i4 {
  font-size: 90px;
  align-self: flex-end;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

<div class="container">
  <div class="item">1</div>
  <div class="item i2">2</div>
  <div class="item">3</div>
  <div class="item i4">4</div>
  <div class="item">5</div>
</div>

Order

The order # changes the items position.

All items have order value of zero.

This moves '4' to the first position.

flex-grow

Specify # to grow item.

Add flex-grow: 1; to all items so items grow to as much as they can.

The margin is removed so you can see how they grow all the way to each other.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

.i2 {
  height: 200px;
}

.i4 {
  font-size: 90px;
  order: -1;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

<div class="container">
  <div class="item">1</div>
  <div class="item i2">2</div>
  <div class="item">3</div>
  <div class="item i4">4</div>
  <div class="item">5</div>
</div>

The # is in relation to other elements.

So, if you set flex-grow: 1; on all items, and then add flex-grow: 2; to a specific item, that item will grows 2x the size of the other items.

NOTE: The items CSS section must come before the specific item flex-grow property. If not, it will not be read.

NOTE: Setting an item to 1 will always allow it to occupy as much space that is allowed.

Example

  • Example
.container {
background-color: #ccc;
padding: 10px;
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  /* margin: 30px; */
  color: #fff;
  font-size: 40px;
  flex-grow: 1;
}

.i2 {
  height: 200px;
  flex-grow: 2;
}

.i4 {
  font-size: 90px;
  order: -1;
}

<div class="container">
<div class="item">1</div>
<div class="item i2">2</div>
<div class="item">3</div>
<div class="item i4">4</div>
<div class="item">5</div>
</div>

flex-basis

flex-basis allows you to set the width of a flex item.

Initial value is 'auto'.

Set a % value.

Example

  • Example
.container {
background-color: #ccc;
padding: 10px;
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
}

.item {
background-color: #f1425d;
padding: 40px;
margin: 30px;
color: #fff;
font-size: 40px;
}

.i2 {
height: 200px;
flex-basis: 20%;
}

.i4 {
order: -1;
}

<div class="container">
<div class="item">1</div>
<div class="item i2">2</div>
<div class="item">3</div>
<div class="item i4">4</div>
<div class="item">5</div>
</div>

flex-shrink

Even if you set flex-basis to a pixel value, it will still shrink when the size of the windows shrinks.

flex-shrink controls how an element can shrink.

Initial value is 1 which means it's allowed to shrink.

Set it to zero so it's NOT allowed to shrink.

Example

  • Example
.container {
background-color: #ccc;
padding: 10px;
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
}

.item {
background-color: #f1425d;
padding: 40px;
margin: 30px;
color: #fff;
font-size: 40px;
}

.i2 {
  height: 200px;
  flex-grow: 0;
  flex-basis: 300px;
  flex-shrink: 0;
}

.i4 {
order: -1;
}

<div class="container">
<div class="item">1</div>
<div class="item i2">2</div>
<div class="item">3</div>
<div class="item i4">4</div>
<div class="item">5</div>
</div>

Shorthand

Shorthand properties go in this order:

  • flex-grow
  • flex-shrink
  • flex-basis
flex: 0 0 300px;

Margin auto trick

Use margin-right: auto; to expand section to occupy all the space available, while at the same time without stretching the entire element.

This allows the space to be taken up, but only the content of the element is selected.

Margin Auto

margin-right: auto;
