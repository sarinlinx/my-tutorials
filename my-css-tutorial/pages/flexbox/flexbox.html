Overview

The flex container

The element you use flexbox on is called the flex container.

You add this with a display property of flex

.class {
  display: flex;
}

or for an inline element
.class {
  display: flex-inline;
}

Flex container properties

flex-direction

The direction of the MAIN AXIS

row is the default

flex wrap

If items should wrap into next line if there isn't enough space.

justify-content

How items are aligned on the MAIN AXIS

align-items

How items are aligned on the CROSS AXIS

align-content

Only applies when there is more than 1 row of flex items

Controls how lines are aligned along the CROSS AXIS if there's some empty space.

Flex items

Flex items are the direct children of the flex container.

They are laid out on the MAIN AXIS

The vertical axis is called the CROSS AXIS

You can also change the direction of the MAIN AXIS

Flex item properties

align-self

Similar to align-items for the container, but only for a single item

orde

Order a single item should appear within the container

3 properties that help flexbox decide on the width of an item

flex-grow

defines how much an item can grow

flex-shrink

defines how much an item can shrink

flex-basis

define its base width

A shortend property called flex should be used instead of writing them individually
