Step 10 - Add hover effect

Add ::before class to add vertical bar to each menu item.

side-nav__item to relative so you can position the vertical bar.

Add side-nav__item::before to position and style the vertical bar.

Finally, use the :not pseudo class to add space between the items.

  • Example
.side-nav__item {
    position: relative;
}

.side-nav__item::before {
  /* Set content so it can display */
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  /* same height as list item */
  height: 100%;
  width: 3px;
  background-color: var(--color-primary);
}

.side-nav__item:not(:last-child) {
  margin-bottom: .5rem;
}

Step 11 - Create hover effect

The left vertical line is hidden because transform: scaleY(0); is set to zero.

When hovered, it changes to 1.

It starts scaling at the center because that is the default for transform-origin.

When hovered, the vertical line appears immediately.

  • Example
 .side-nav__item::before {
  /* Set content so it can display */
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  /* same height as list item */
  height: 100%;
  width: 3px;
  background-color: var(--color-primary);
  /* Set initial state to zero so when hovered it expands to 1 */
  -webkit-transform: scaleY(0);
  transform: scaleY(0);
}

.side-nav__item:hover::before,
.side-nav__item--active::before {
  -webkit-transform: scaleY(1);
  transform: scaleY(1);                 
}

Step 12 - Allow vertical line to grow

Add a transition so the vertical lines grows when hovered.

transform starts after .2s

width begins after a .2s delay and takes .4s to complete using a cubic-bezier function

You could also add a background-color transform of .1s so the transition occurs in step #15 below.

Set width to 100% on hover

  • Example
.side-nav__item::before {
  /* Set content so it can display */
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  /* same height as list item */
  height: 100%;
  width: 3px;
  background-color: var(--color-primary);  
  -webkit-transform: scaleY(0);
  transform: scaleY(0);
  transition: transform 0.2s, width 0.4s cubic-bezier(1, 0, 0, 1) 0.2s, background-color 0.1s;
}

.side-nav__item:hover::before,
.side-nav__item--active::before {
  -webkit-transform: scaleY(1);
  transform: scaleY(1);
  width: 100%;
}
