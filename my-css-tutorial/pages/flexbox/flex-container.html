The flex container

display: flex;

To use flexbox, you must set the container's display property to flex.

.container {
  display: flex;
}

<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
</div>

Example

  • Example

flex-direction

flex-direction changes which way the axis points.

The default is row which is left to right.

row-reverse

To reverse the direction, use row-reverse

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row-reverse;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

column

Use 'column' to stack vertically

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: column;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

column-reverse

Use 'column-reverse' to stack vertically and change the direction.

Example

  • Example

.item {
  background-color: #f1425d;

justify-content

Position the items within the container.

center

This center the items within the container.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: center;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

space-between

evenly distributes space between the flex items.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: space-between;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

space-around

Puts the same amount of space on both the left and right side of the items.

Note that the left and right items will only have one amount of the space, but the space between items has 2 amounts.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: space-around;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

space-evenly

Similar to space-around, but the items all have the same amount of space on all sides.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: space-evenly;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

flex-start and flex-end

flex-start positions the elements to the starting position of the container (default is left).

flex-end positions the elements to the end position of the container (default is right).

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: flex-start;
  justify-content: flex-end;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

align-items

align-items positions the items along the CROSS AXIS.

stretch

The initial value is stretch.

If a single item has a larger height than others, the rest will STRETCH to the same height.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: center;
  justify-content: center;
  align-items: stretch;
}

.i2 {
  height: 200px;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

<div class="container">
  <div class="item">1</div>
  <div class="item i2">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
</div>

Centers all items vertically.

If a single item has a larger height than others, the vertically center position is used for all.

flex-start and flex-end

Positions vertically starting at the top or bottom.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: center;
  justify-content: center;

.i2 {

  background-color: #f1425d;
  padding: 40px;

  color: #fff;
  font-size: 40px;

baseline

baseline positions based on the TEXT size.

So if one item has a larger text size, the horizontal baseline of all text items is what is used.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: center;
  justify-content: center;
  align-items: baseline;
}

.i2 {
    height: 200px;
}

.i4 {
    font-size: 90px;
}

.item {
    background-color: #f1425d;
    padding: 40px;
    margin: 30px;
    color: #fff;
    font-size: 40px;
}

<div class="container">
  <div class="item">1</div>
  <div class="item i2">2</div>
  <div class="item">3</div>
  <div class="item i4">4</div>
  <div class="item">5</div>
</div>

flex-wrap

If there are too many items in a container, they will start to overflow outside of it.

flex-wrap takes care of this:

flex-wrap: wrap; creates a new line for the items that can't fit the container.

Example

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

.item {
  background-color: #f1425d;
  padding: 40px;
  margin: 30px;
  color: #fff;
  font-size: 40px;
}

.i2 {
  height: 200px;
  flex-grow: 0;
  flex-basis: 300px;
  flex-shrink: 0;
}

.i4 {
  order: -1;
}



<div class="container">
  <div class="item">1</div>
  <div class="item i2">2</div>
  <div class="item">3</div>
  <div class="item i4">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
  <div class="item">7</div>
  <div class="item">8</div>
  <div class="item">9</div>
  <div class="item">10</div>
</div>

align-content

Aligns rows along the CROSS AXIS..

Since the height of the container is 1000px, all items are in the center.

align-content: flex-start; pushes them to the top.

The stretch option just stretches the rows which aren't visible, so it's more similar to space-between.

  • Example
.container {
  background-color: #ccc;
  padding: 10px;
  height: 1000px;

  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
