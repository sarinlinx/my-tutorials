Overview

The CSS box-sizing property allows us to include the padding and border in an element's total width and height.

This prevents us from having to do math to calculate the actual size of a box (as done with the old box model).

border-box

Setting box-sizing: border-box; on an element sets the padding and border in the width and height.

.div1 {
    width: 300px;
    height: 100px;
    border: 1px solid blue;
    box-sizing: border-box;
}

Global setting

Most developers set the entire document to use box-sizing: border-box;.

* {
  box-sizing: border-box;
}
