JSX

JSX is a set of instructions that tells React exactly what content to display on the screen at any given time.

It looks very similar to HTML.

React iterates over every JSX element. If it's a DOM element, it displays to the page.

If it's a React Component, the Component function is called and inspects the JSX it gets back.

Your browser doesn't understand JSX. Bable converts it into JavaScript code.

You could write out the JavaScript functions manually, but JSX makes it much easier. For example:

One Top Level Element

All HTML code must be wrapped in ONE top level element.

This is often a div tag.

Elements Must be Closed

JSX follows XML rules, and therefore HTML elements must be properly closed.

Example Close empty elements with />

const myelement = <input type="text" />

The return statement line with multi-line JSX

There must be something on the same line after return, otherwise it will not work.

The convention is to put an opening parenthesis after the 'return' keyword, then JSX on the following lines. 

const App = () => {
  return (
    <div>
      <label className="label" for="name">
        Enter name:
      </label>
      <input id="name" type="text" />
      <button style={{ backgroundColor: "blue", color: "white" }}>
        Submit
      </button>
    </div>
  ); 
};

Converting Inline CSS to JSX

The JSX must be formatted differently than the HTML when adding CSS styles.

The first curly brace indicates a JavaScript variable inside JSX.

The 2nd references a JavaScript Object.

If the CSS property has a hyphen, remove it and capitalize the next letter.

Add single quotes around value to create a string.

Remove semicolon.

<div style="background-color: red;"></div> 
// change to 
<div style={{backgroundColor: 'red'}}></div>

Quotes in JSX

Use double quotes for JSX properties.

For any NON-JSX property, use single quotes instead. For example, when adding CSS properties.

HTML classes in JSX

Do not use the word class in JSX. Use className.

This is to avoid confusion with JavaScript thinking the word class should be a JavaScript class, not HTML.

NOTE: This may not be required in future versions of React.

const App = () => {
    return (
      <div>
        <label className="label" for="name">
          Enter name:
        </label>
        <input id="name" type="text" />
        <button style={{ backgroundColor: "blue", color: "white" }}>
          Submit
        </button>
      </div>
    );
  };

Expressions in JSX

With JSX you can write expressions inside curly braces { }.

The expression can be a React variable, or property, or any other valid JavaScript expression. JSX will execute the expression and return the result.

JSX can reference JavaScript variables

This means you can add a JavaScript variable inside JSX.

This creates a variable named buttonText. It's then added to the JSX inside of curly braces.

const App = () => {
  const buttonText = 'Click Me!'; 

  return (
    <div>
      <label className="label" for="name">
        Enter name:
      </label>
      <input id="name" type="text" />
      <button style={{ backgroundColor: "blue", color: "white" }}>
      {buttonText}
      </button> 
    </div>
  );
};

JSX can reference JavaScript functions outside the App()

This creates a function outside App().

You can then call it in JSX by placing it inside curly braces.

Make sure to add () to invoke then function.

function getButtonText() {
  return 'Click on me!';    
} 

const App = () => {

  return (
    <div>
      <label className="label" for="name">
        Enter name:
      </label>
      <input id="name" type="text" />
      <button style={{ backgroundColor: "blue", color: "white" }}>
       {getButtonText()} 
      </button>
    </div>
  );
};

JSX and Objects

You cannot reference a JavaScript Object to print text. This will throw an error.

Error: Objects are not valid as a React child (found: object with keys {text}). If you meant to render a collection of children, use an array instead.

const App = () => {
  const buttonText = { text: 'Cick Me!' }

  return (
    <div>
      <label className="label" for="name">
        Enter name:
      </label>
      <input id="name" type="text" />
      <button style={{ backgroundColor: "blue", color: "white" }}>
      {buttonText} 
      </button>
    </div>
  );
};

Solution. Use the key name to display text.

You must reference the property.

For example: {buttonText.text}

const App = () => {
  const buttonText = { text: 'Cick Me!' }

  return (
    <div>
      <label className="label" for="name">
        Enter name:
      </label>
      <input id="name" type="text" />
      <button style={{ backgroundColor: "blue", color: "white" }}>
      {buttonText.text}
      </button>
    </div>
  );
};

The 'for' keyword

React will throw a console error if the HTML word 'for' is used.

Instead, change this to htmlFor.

const App = () => {
  const buttonText = { text: 'Cick Me!' }

  return (
    <div>
      <label className="label" htmlFor="name">
        Enter name:
      </label>
      <input id="name" type="text" />
      <button style={{ backgroundColor: "blue", color: "white" }}>
     {buttonText} 
      </button>
    </div>
  );
};

if statements

React supports if statements, but not inside JSX.

if statements must be placed outside of the JSX.

But, you can use a ternary expression inside JSX.

const myelement = <h1>{(x) < 10 ? "Hello" : "Goodbye"}</h1>;

Adding images using faker.js

This uses faker.js to generate a fake image.

Stop your app.

Install the faker library into your local project.

npm install --save faker

Import faker.js

Add an import statement at the top.

import React from "react";
import ReactDOM from "react-dom";
import faker from "faker"; 

if (module.hot) {
  module.hot.accept();
}

const App = () => {
  return (
    <div className="ui container comments">
      <div className="comment">
        <a href="/" className="avatar">
          // call the faker API 
          <img alt="avatar" src={faker.image.image()} /> 
        </a>
        <div className="content">
          <a href="/" className="author">
            Sam
          </a>
          <div className="metadata">
            <span className="date">Today at 6:00PM</span>
          </div>
          <div className="text">Nice blog post!</div>
        </div>
      </div>
    </div>
  );
};

ReactDOM.render(<App />, document.querySelector("#root"));
