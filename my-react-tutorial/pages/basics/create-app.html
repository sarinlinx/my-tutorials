Creating an app

Create the app

Run this command to create a new React app.

This creates a new directory named after the app.

npx create-react-app my-react-app

Files in the new app directory

Run the app

Change into the app directory.

Then, run npm start

This opens a browser with the default Reacct app.

cd my-react-app
npm start

Stop the app

Click CTRL + C on your keyboard to stop the app.

CTRL + C

Building an app from the default code

Navigate into the new app directory, then the /src directory.

Delete everything.

Create a new file inside called index.js.
