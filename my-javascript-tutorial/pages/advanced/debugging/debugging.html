Debugger statement

Put the debugger statement anywhere you want to pause the code and check what variables equal.

In your code, simpley type debugger. This triggers debugging tools in your browser.

The browser will stop JavaScript execution at the point where you add this statement.

debugger

Working with Debugger (Firefox)

In Firefox, open dev tools and click the 'Debugger' tab to view your JS files and content.

Click ESC so you can enter console code.

You can then print out information. For example, type the name of a variable to view its content.

This allows you to quickly search through your variables.

If everything looks good, click the PLAY button. This continues execution through the file. This is helpful if you add multiple 'debugger' statements. It will then go to the next one and stop execution there.

Working with Debugger (Chrome)

Open dev tool. Sources tab. Click the script.js file.

Create a breakpoint by clicking to the left of a line. This creates a red dot notifying you a breakpoint has been set.

Reload page and execution will stop at that place.

Right pane, click PLAY button (or step button) to continue execution.

You can view info in right pane as the script continues each time. This helps you to figure out where the issue occurs.
