Downwards

Create a variable

Create a variable for an element to use with all functions.

const h1 = document.querySelector('h1');

querySelectorAll()

This returns a Node list of all elements with a class of .highlight that are direct children of h1.

In this case, a Node list of 2 elements with the class of .highlight inside the h1

console.log(h1.querySelectorAll('.highlight'));

childNodes()

direct children

This returns all children elements inside the h1 tags

This includes text, comments, classes

console.log(h1.childNodes);

children()

direct children

Returns an HTMLCollection of all child tags (not text) within the h1 element

This retunrs the first span, br tag, and final span tag

console.log(h1.children);

firstElementChild

direct children

Selects the first child after the opening h1

This selects the first span tag

h1.firstElementChild.style.color = 'white';

lastElementChild

Selects the last child within the h1 tags

This selects the last span tag

h1.lastElementChild.style.color = 'orangered';

Upwards

parentNode()

direct parents

This returns the parent element of the h1 tag

In this example, <div class="header__title">

console.log(h1.parentNode);

parentElement()

returns the same as parentNode, need a better example 

console.log(h1.parentElement);

closest() ex 1

closest() can be thought of as the opposite of querySelector since querySelector searches for children

closest() searches for parents

This changes the closest parent with a class of header

h1.closest('.header').style.background = 'var(--gradient-secondary)';

closest() ex 2

need a better example. Since there's only 1 h1 tag, it gets selected

h1.closest('h1').style.background = 'var(--gradient-primary)';

Siblings

previousElementSibling

Can only access direct siblings in JS

Select previous sibling

This returns null since there is no sibling h1 tag

console.log(h1.previousElementSibling);

nextElementSibling

This returns the h4 tag since it's the next sibling

console.log(h1.nextElementSibling);

previousSibling

Sibling Nodes

This returns the Node before the h1

This returns #test

console.log(h1.previousSibling);

nextSibling

Sibling Nodes

This returns the Node after the h1

console.log(h1.nextSibling);

Return all siblings

There's not a direct way to do this in JavaScript.

You must first go to the parent, then check for children

Frst list all children of the parent of the h1, which includes the h1

This returns an HTMLCollection

Then spread the HTMLCollection into an Array, then loop

console.log(h1.parentElement.children);
  
[...h1.parentElement.children].forEach(function (el) {
  if (el !== h1) el.style.transform = 'scale(0.5)';
