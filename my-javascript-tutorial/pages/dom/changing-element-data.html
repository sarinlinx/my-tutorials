Methods to add data to elements in the DOM

This section lists different ways to add data to elements in the DOM.

createElement()

.createElement() creates a new element but does not add it to the DOM.

const message = document.createElement('div');

Add class to the element

Use className to add a class to the newly created element.

let cssSpanTag = document.createElement('span');
cssSpanTag.className = x;

createTextNode()

.createTextNode() creates text that goes inside the elmenet.

This prints: Testing new text to add

<p id="p1">Testing </p>
              
let newText = document.createTextNode('new text to add');
let paragraph1 = document.getElementById("p1");
paragraph1.appendChild(newText);

textContent()

textContent() changes the text within an element.

It cannot adjust HTML.

This prints: We use cookies for analytics

<p id="p1">Testing </p>
              
const message = document.querySelector('#test1');
message.textContent = 'We use cookies for analytics.';

innerHTML()

innerHTML() can either:

  • change the text within an element (if only text is entered)
  • Add HTML to an element

This example surrounds the text in strong tags.

<p id="p1">Testing </p>
              
const message = document.querySelector('#test1');
message.innerHTML = '<strong>We use cookies for analytics.</strong>';
