Methods to retrieve elements

Older methods

These are old methods which are good for forms or if you want your code to be unambiguous.

Get elements with specific ID.

document.getElementById("some-ID");

NOTE: The following two return an HTMLCollection, not a node list.

An HTMLCollection is a live collection. If the DOM changes, this collection is also updated.

Get all element with a specified class name as array.

document.getElementsByClassName("classname");

Get all elements with specified HTML tag as array.

document.getElementsByTagName("HTML tag");

Newer methods

These are easier to use.

querySelector()

These must be triggered. A button click is used below.

querySelector. Get the first element matching specified selectors.

  • querySelector
document.querySelector(".main-nav a");

Retrieve the text inside the element (by class)

This example retrieves the HTML inside the element.

This returns: <p>Main Title</p>

Note that everything inside the .main-title class is returned, even the paragraph tags.

<div class="main-title"><p>Main Title</p></div>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var innerhtml = document.querySelector(".main-title").innerHTML;
  alert(innerhtml);
}

Retrieve the HTML and text of an element (by class)

This example retrieves the HTML around the element.

This returns: <div class="main-title"><p>Main Title</p></div>

<div class="main-title"><p>Main Title</p></div>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var outerhtml = document.querySelector(".main-title").outerHTML;
  alert(outerhtml);
}

Change the text inside an element (by class)

This example retrieves the HTML inside the element, then uses innerHTML = to change it to something else.

This returns: <p>New Title</p>

<div class="main-title">Main Title</div>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var innerhtml = document.querySelector(".main-title").innerHTML = "New Title";
}

Change the ID of an element

This changes the 'id' attribute of an element.

This changes the id of 'firstID' to 'secondID'.

You can see this since the second alert box is different.

<p id="firstID">First paragraph</p>
<button onclick="myFunction()">Click to change</button>

function myFunction() {
  var firstID = document.querySelector("#firstID").id;
  alert(firstID);
  var firstID = document.querySelector("#firstID").id = "secondID";
  alert(firstID);
}

Remove elements when button is clicked

This removes all elements with a class of .note when the button is clicked.

<p class="note">This application was created by Andrew Mead</p>
<p class="note">Some note text</p>
<button id="remove-all">Remove All Notes</button>

document.querySelector('#remove-all').addEventListener('click', function () {
    document.querySelectorAll('.note').forEach(function( note) {
        note.remove()
    })
})

querySelectorAll

querySelectorAll. This returns a node list of all elements that match the specified selectors.

This example retrieves all paragraph tags.

You can then alter them.

  • querySelectorAll
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<p>Paragraph 3</p>

document.querySelectorAll("a");

Change all paragraph content to *

This queries all paragraph tags.

They are then looped through and their content changed.

textContent is a property

const ps = document.querySelectorAll('p')

ps.forEach(function (p) {
    p.textContent = '******'
})

Remove an element

Use the remove() method to remove an element.

const ps = document.querySelectorAll('p')

ps.forEach(function (p) {
    p.remove()
})

Remove paragraphs that contain the word 'the'

This removes all paragraphs that contain the word 'the'.

const paragraphs = document.querySelectorAll('p')

paragraphs.forEach(function (paragraph) {
  if (paragraph.textContent.includes('the')) {
    paragraph.remove()
  }
})

Capturing text input

This prints to the console what you're typing.

<input id="search-text" type="text" placeholder="Filter todos">

//the 'input' event fires on every character change
document.querySelector('#search-text').addEventListener('input', function (e) {
    console.log(e.target.value)
})
