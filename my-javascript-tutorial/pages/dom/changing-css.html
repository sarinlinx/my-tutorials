View INLINE styles of an element

This shows inline styles ONLY.

The styles that are being applied from an external style sheet, or even internal styles in the head, do not show up here because JavaScript is just requesting the element and looking up what's inside the element.

When the command is ran in a console, it prints a CSS style declaration list of every possible property that could be applied using inline styles. Within that list is the inline value applied.

Retrieve using the console

Use .style to retrieve inline styles for an element.

In your browser's console, run the following command.

This returns the inline style applied to the div tag.

<div class="cta" style="color: blue;">
<p>Text</p>
</div>

document.querySelector(".cta").style;

Changing CSS

Changing a single property

It does not matter where the CSS is added. This will change it.

To change the CSS of an element, first retrieve it, then append .style. = "value" to it.

For example: document.querySelector(".cta").style.color = "green"

Change using the console

This changes the CSS of the .cta class to red.

<div class="cta" style="color: blue;">
  <p>Text</p>
</div>

document.querySelector(".cta").style.color = "red";

If the property has a hyphen

If the CSS property has a hyphen in it, it becomes camelCase in JavaScript.

For example, background-color becomes backgroundColor.

<div class="cta" style="color: blue;">
  <p>Text</p>
</div>

document.querySelector(".cta").style.backgroundColor = "red";

Changing multiple properties

Use .style.cssText following by multiple statements to add multiple properties.

<div class="cta" style="color: blue;">
  <p>Text</p>
</div>

document.querySelector(".cta a").style.cssText = "padding: 1em; color: white; background-color: red;"

Assign CSS to a variable

This assigns a variable to CSS so you can easily call it in the future.

<div class="cta" style="color: blue;">
  <p>Text</p>
</div>

const message = document.querySelector('#test1');
message.style.background = '#37383d';

Get the CSS of an element

This uses getComputedStyle() to returns the CSS values for the element.

getComputedStyle(message).color

CSS custom properties (variables)

CSS variables are assigned in the root.

You cannot use .style to change the value. You must use .setProperty.

Use document.documentElement. to target the root.

:root {
  --color-primary: #5ec576;
  --color-secondary: #ffcb03;
  --color-tertiary: #ff585f;
  --color-primary-darker: #4bbb7d;
  --color-secondary-darker: #ffbb00;
  --color-tertiary-darker: #fd424b;
  --color-primary-opacity: #5ec5763a;
  --color-secondary-opacity: #ffcd0331;
  --color-tertiary-opacity: #ff58602d;
  --gradient-primary: linear-gradient(to top left, #39b385, #9be15d);
  --gradient-secondary: linear-gradient(to top left, #ffb003, #ffcb03);
}
  

document.documentElement.style.setProperty('--color-primary', 'orangered');

Inline CSS is just an attribute

Inline CSS is an attribute. Because it's an attribute, you can use the following methods to interact with the 'style' inline style.

This only works with inline CSS.

  • .hasAttribute("style"); - does element have style?
  • .getAttribute("style"); - get inline style
  • .setAttribute("style", "color:red"); - Add CSS
  • .removeAttribute("style"); - Remove inline styles
