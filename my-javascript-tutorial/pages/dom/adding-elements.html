Adding new DOM elements

This page lists different ways to add, edit, and remove DOM elements.

insertAdjacentHTML

  • insertAdjacentHTML

insertAdjacentHTML inserts nodes into the DOM at a specified location. It uses one of 4 options:

  • 'beforebegin': Before the element itself.
  • 'afterbegin': Just inside the element, before its first child.
  • 'beforeend': Just inside the element, after its last child.
  • 'afterend': After the element itself.

beforebegin()

This creats a new div element with an id of 'two'.

It is added to the DOM before the existing div.

<div id="test1">
  <p>Testing insertAdjacentHTML</p>
</div>

let getEl = document.getElementById('test1');
getEl.insertAdjacentHTML('beforebegin', '<div id="two">two</div>');

result

This shows it was added before the existing div.

<div id="two">two</div>

<div id="test1">
  <p>Testing insertAdjacentHTML</p>
</div>

afterbegin()

This creats a new div element with an id of 'two'.

It is added just after the opening tag of the existing div.

<div id="test1">
  <p>Testing insertAdjacentHTML</p>
</div>

let getEl = document.getElementById('test1');
getEl.insertAdjacentHTML('afterbegin', '<div id="two">two</div>');

result

This shows it was added after the opening existing div.

<div id="test1">
  <div id="two">two</div>
  <p>Testing insertAdjacentHTML</p>
</div>

This creats a new div element with an id of 'two'.

It is added just before the closing existing div.

  <div id="two">two</div>

It is added after the closing existing div.

appendChild()

.appendChild() Adds an HTML node to the DOM.

This example:

  • targets an existing div
  • adds the new element
  • appends text to the new element
  • then finally appends the new element to the existing div section.
<div id="test1">
  <p id="p1">Testing </p>
  <p id="p2">Testing </p>
  <p id="p3">Testing </p>
</div>
              
// target existing div to append to
const selDiv = document.querySelector('#test1');
// create an element to append
const paraElement = document.createElement("p");
// create text to add to the appended element
const paraText = document.createTextNode("new text to add");
// append text to the new element
paraElement.appendChild(paraText)
// append the new element to the existing div
selDiv.appendChild(paraElement)

Steps to add an element to the DOM

These examples first create a node.

Text or HTML is added.

It is then placed within the page.

Step 1 - create the element

createElement()

This creats a new element but does not add it to the DOM.

const message = document.createElement('div');

Step 2 - Add data to the element

This example adds information to the element using classList.add(), textContent, and .innerHTML.

const message = document.createElement('div');

message.classList.add('cookie-message');
message.textContent = 'We use cookies for analytics.';
message.innerHTML = 'We use cookies for analytics. <button class="btn btn--close-cookie">Got it!</button>';

Step 3 - Add the element to the page

prepend()

Prepend the new element to the page before the <header> tag.

This adds it as the first child.

header.prepend(message)

append()

appends the new element to the page after the <header> tag.

This adds it as the last child.

header.append(message)

Examples

Example: Adding a new DOM element

This creats a new element titled <figcaption> with text inside

If you check dev tools, you'll see the new code has been added to the DOM.

<figure class="featured-image">
  <img src="../_img/note.jpg">
</figure>

//Create a variable for the existing element
//You'll need this to append the new element
const FEATURED = document.querySelector(".featured-image");

//Create a figcaption element
//This creates a fig caption document that is not placed anywhere in the document.
//It's just sitting in the memory of the browser.
var captionElement = document.createElement("figcaption");

//Create text to add to the new figcaption element
var captionText = document.createTextNode("new text to add");

//Add the new figcaption text to the new figcaption element
captionElement.appendChild(captionText);

//Add new figcaption element to the featured image figure
FEATURED.appendChild(captionElement);

Adding to the DOM with a template literal

This adds HTML after a div tag with the class of movements.

Assign a variable to the .movements element.

Next is an Object with a nested Array namved 'movements'.

Create a Function expression that contains a forEach loop

This loop contains variables that are used in a template literal.

The template literal builds the new HTML.

Call the original variable for the DOM element. Use insertAdjacentHTML to add HTML just after the beginning of this element.

This takes 2 arguments. The 1st is position to attach the HTML (afterbegin).

The 2nd is the HTML string you are inserting.

<div class="movements"></div>
  
const containerMovements = document.querySelector('.movements');

const account1 = {
  movements: [200, 450, -400]
};

const displayMovements = function (x) {
  x.forEach(function (mov, i) {

    const type = mov > 0 ? 'deposit' : 'withdrawl'

    const html = `<div class="movements__row">
    <div class="movements__type movements__type--${type}">${i + 1} ${type}</div>     
    <div class="movements__value">${mov}</div>
    </div>
    `;
    // add this to the element you're adding to
    // 2 arguments
    // 1st is position to attach the HTML
    // 2nd is string you are inserting
    containerMovements.insertAdjacentHTML('afterbegin', html)
  });
}

displayMovements(account1.movements)
