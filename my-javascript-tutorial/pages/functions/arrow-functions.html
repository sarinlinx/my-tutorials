Arrow Functions

Shorter version of a function expression.

Good for simple one-liner functions.

return keyword can be ommited.

Also, Arrow functions do not allow the this keyword.

Never use an Arrow Function as an Object method. This will cause issues when using the 'this' keyword because it will look up to the parent scope for its 'this' keyword. If that's the Global Window Object you will get an unexpected result.

Comparison

After the =>, write what you want to return.

The return keyword must be explicity set in a normal function.

The return keyword is implicity set in an Arrow function.

// Function expression
const calcAge2 = function (birthYear) {
  return 2037 - birthYear;
}

// Arrow function
const calcAge3 = birthYear => 2037 - birthYear;
const age3 = calcAge3(1991)
console.log(age3)

More complex example

More calculations, so curly braces are needed.

return statement must also be used in this case.

const yearsUntilRetirement = birthYear => {
  const age = 2037 - birthYear;
  const retirement = 65 - age;
  return retirement;
}

console.log(yearsUntilRetirement(1991));

Multiple parameters

Must wrap in parenthesis

const yearsUntilRetirement = (birthYear, firstName) => {
  const age = 2037 - birthYear;
  const retirement = 65 - age;
  return `${firstName} retires in ${retirement} years`;
}

console.log(yearsUntilRetirement(1991, 'Jonas'));
