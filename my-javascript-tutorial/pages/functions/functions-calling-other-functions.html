Functions calling other functions

The first function cutFruitPieces() is the one that will be called.

The second function fruitProcessor() calls cutFruitPieces() twice.

You can then call fruitProcessor with two parameters.

Both functions run and the multiplied values (from the first function) are printed to the console.

function cutFruitPieces(fruit) {
  return fruit * 4;
}

function fruitProcessor(apples, oranges) {
  // call the first function
  const applePieces = cutFruitPieces(apples)
  const orangePieces = cutFruitPieces(oranges)

  const juice = `Juice with ${applePieces} piece of apples and ${orangePieces} pieces of oranges.`;
  return juice;
}

console.log(fruitProcessor(2, 3))

Output

Juice with 8 piece of apples and 12 pieces of oranges.
