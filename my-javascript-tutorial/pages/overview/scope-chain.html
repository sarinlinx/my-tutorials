Scoping

Scoping: How our program's variables are organized and accessed by the JS engine.

It asks "where do variables live" and "where can we access a cetain variable, and where not?".

Lexical scoping

Scoping is controlled by placement of functions and blocks in the code.

Scope

The space or environment in which a certain variagble is declared (variable environment in case of functions). There is global scope, function scope, and block scope

Scope of a variable

Regiion of code where a certain variable can be accessed.

Global scope

Variables declared outside a function or block. Accessible everywhere.

Function Scope

Only accessible within the function. Also called local scope.

Block scope (ES6)

Variables accessible only inside a block (everything between curly braces)

However, this ONLY APPLIES to let and const variables.

Functions are also block scoped in strict mode.

var variables are Function scoped, not Block Scoped.

The Scope Chain

Start with the Global Scope.

Then function scope.

Then block scope.

If a block scope can't find a variable, something called "Variable Lookup" occurs where the block looks up the scope for the variable.
