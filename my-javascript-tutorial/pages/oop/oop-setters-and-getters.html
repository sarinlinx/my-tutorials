Setters and Getters

every object in JavaScript can have setter and getter properties. And we call these special properties assessor properties, while the more normal properties are called data properties.

getters and setters are basically functions that get and set a value so just as the name says, but on the outside they still look like regular properties.

Getter example

Create an Object

Use the keyword get followed by a method

Call it in dot syntax, not as a method.

const account = {
  owner: 'jonas',
  movements:  [200, 300, 120, 363],

  get latest() {
    // slice creates a new array, pop() extracts it 
    return this.movements.slice(-1).pop();
  } 
}

// use it as a property 
console.log(account.latest)

Setting a property that already exists

This attempts to set a property that already exists.

You cannot use the same name as the existing property, so just add an underscore before it.

You must then create a getter for the new _fullName property.

class PersonCl {
  constructor(fullName, birthYear) {
  this.fullName = fullName;
  this.birthYear = birthYear;
  }

  // Create setter to check if text string is a full name by looking for a space
  set fullName(name) {
    if (name.includes(' ') this._fullName = name; 
    else console.log(`${name} is not a full name`)
  }
}

get fullName() {
  return this._fullName;
}
