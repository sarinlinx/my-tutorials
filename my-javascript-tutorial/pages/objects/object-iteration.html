Iterating over Object key:value pairs

You can loop through key:value pairs by using a for loop with the 'key' keyword.

var menu = {
  width:  300,
  height: 200,
  title: "Menu"
};

for(var key in menu) {
var val = menu[key];

console.log("Key: " + key + " \nValue: " + val);
}

The code above prints out the following:

Key: width
Value: 300

Key: height
Value: 200

Key: title
Value: Menu

Object.keys

Loop over property names (keys)

This first prints an Array with [thur, fri, sat]

The for loop then iterates over the properties and prints: thur fri sat

const openingHours = {
  thur: {
    open: 12,
    close: 22,
  },
  fri: {
    open: 11,
    close: 23,
  },
  sat: {
    open: 0, // Open 24 hours
    close: 24,
  },
};

const properties = Object.keys(openingHours)
console.log(properties)

for (const day of Object.keys(openingHours))
console.log(day)
}

Loop entire Object using entries

Entries are properties and values.

This returns an Array of all properties and values.

const openingHours = {
  thur: {
    open: 12,
    close: 22,
  },
  fri: {
    open: 11,
    close: 23,
  },
  sat: {
    open: 0, // Open 24 hours
    close: 24,
  },
};

const entriesHere = Object.entries(openingHours)
 console.log(entriesHere)

Destructure to print values as strings

Specify open and close property names to destructe them.

They can then be used in the console.log string.

The first destructred variable is called key, but you can name it anything.

const openingHours = {
thur: {
  open: 12,
  close: 22,
},
fri: {
  open: 11,
  close: 23,
},
sat: {
  open: 0, // Open 24 hours
  close: 24,
},
};

const entriesHere = Object.entries(openingHours)

for(const [key, {open, close}] of entriesHere) {
  console.log(`On ${key} we open at ${open} and close at ${close}`)
}
