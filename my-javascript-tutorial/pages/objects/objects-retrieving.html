Retrieving data from Objects

Retrieving the full Object

You can retrieve the full Object just like a variable.

console.log(course);

Retrieving properties using dot notation

You can use dot notation to retrieve a specific property.

The syntax is: objectName.propertyName

Dot notation works well for properly named properties.

console.log(course.title);

Retrieving properties using bracket notation

You can use bracket notation to retrieve a specific property.

The syntax is: objectName["propertyName"]

Bracket notation works well for when we might have to convert the property name into a string.

console.log(person["lastName"];);

Advantage of bracket notation

You can place any expression you like using bracket notation.

This means you do not have to write the specific string, you can compute it from some operation.

Examples

Full example how ot create and retrieve Object data

This creates an Object named myBook with data.

The data is retrieved using a string template and dot notation.

The value of 'title' is then changed manually. When checked, the new value displays.

This outputs:

  • 1984 by George Orwell
  • Animal Farm by George Orwell
let myBook = {
  title: '1984',
  author: 'George Orwell',
  pageCount: 326
}

console.log(`${myBook.title} by ${myBook.author}`)

myBook.title = 'Animal Farm'

console.log(`${myBook.title} by ${myBook.author}`)

Returning 2 Objects using a function

Instead of writing several console.log statements, you can create a function to gather the data.

This returns:

  • 1984 by George Orwell
  • A Peoples History by Howard Zinn
let myBook = {
  title: '1984',
  author: 'George Orwell',
  pageCount: 326
}

let otherBook = {
  title: 'A Peoples History',
  author: 'Howard Zinn',
  pageCount: 723
}

let getSummary = function (book) {
  console.log(`${book.title} by ${book.author}`)
}

getSummary(myBook)
getSummary(otherBook)

Create and return 2 new properties using a function

This creates two new propties within the function.

  • summary
  • pageCountSummary

This returns:

  • 1984 by George Orwell
  • 1984 is 326 pages long
  • A Peoples History is 723 pages long
  • A Peoples History by Howard Zinn
let myBook = {
  title: '1984',
  author: 'George Orwell',
  pageCount: 326
}

let otherBook = {
  title: 'A Peoples History',
  author: 'Howard Zinn',
  pageCount: 723
}

let getSummary = function (book) {
    return {
      //return different summaries by creating different properties
      //set the new summary property to a template string
      summary: `${book.title} by ${book.author}`,
      //create new pageCountSummary property
      pageCountSummary: `${book.title} is ${book.pageCount} pages long`
    }
}

//bookSummary is an Object because it returns Object data from the function
let bookSummary = getSummary(myBook)
let otherBookSummary = getSummary(otherBook)

console.log(bookSummary.summary)
console.log(bookSummary.pageCountSummary)

console.log(otherBookSummary.summary)
console.log(otherBookSummary.pageCountSummary)
