Prototypal Inheritance

Prototypal Inheritance is when you can base the functionality of an object on another object.

It gives you access to other Objects.

However, not every object can be based on another object.

Create a variable 'speak' and assign it to a function that returns the input argument.

Create another function named 'Dog'. Within that, create two variables: name, breed.

Use 'Prototypal Inheritance'. To the 'Dog' Object, add a 'speak' method. Dog.prototype.speak = speak;

Pass the method 'speak' and give it a value of 'woof'. firstDog.speak('woof');

var speak = function(saywhat) {
  console.log(saywhat);
}

var Dog = function() {
  var name, breed;
}

Dog.prototype.speak = speak;

firstDog = new Dog;
firstDog.name = "Rover";
firstDog.breed = "Doberman";
firstDog.speak('woof');

Checking output

In your console, run 'firstDog'.

You see a list of its properties. Notice how it inherited the function 'speak'.
