Object Constructors

Object Constructors are templates for creating objects that you define once. You can then use those templates again and again.

The object constructor allows you to create different objects using the same properties.

Create a constructor for an object

The constructor must be Capitalized.

This creates a constructor named 'Course'.

The 'this' keyword refers to the current object since you're making new objects and don't know the name of them yet.

Not sure what the updateViews function is supposed to do.

function Course(title,instructor,level,views) {
  this.title = title;
  this.instructor = instructor;
  this.level = level;
  this.views = views;
  this.updateViews = function() {
      return ++this.views;
    };
}

Use the constructor to create new instances of the 'Course' object.

Assign each of the properties inside the object to the matching argument.

var course01 = new Course("JavaScript Essential Training",
               "Teacher Name",
               1,
               0);

console.log(course01);

Create a new course with different information, but same properties.

var course02 = new Course("New course name", "other teacher Name", 3, 1234);

console.log(course02);
