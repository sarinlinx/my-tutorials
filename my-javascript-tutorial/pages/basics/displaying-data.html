Displaying data

JavaScript does NOT have any built-in print or display functions.

There are 4 ways to display data.

  • Writing into an alert box, using window.alert().
  • Writing into the HTML output using document.write().
  • Writing into an HTML element, using innerHTML.
  • Writing into the browser console, using console.log().

window.alert()

This creates a pop-up displaying your data.

window.alert(5 + 6);

document.write()

This displays text directly on the web page.

document.write(5 + 6);

If you use it AFTER all HTML is loaded, it will delete all existing HTML.
