Converting a String to an Array

comma

This splits the string elements into an Array.

This returns an Array: [a, b, c, d, e, f]

let str = "a,b,c,d,e,f";
let arr = str.split(",");
console.log(arr)

pipe

This creates and Array where the first index is everything before the |, and the 2nd index is everything after.

This returns an Arry: ["a,b,c", "d,e,f"]

let str = "a,b,c|d,e,f";
let arr = str.split("|");
console.log(arr)

space

This creates and Array where the first index is everything before the space, and everything after is the 2nd index.

This returns: ["a, b, c, d, e", "f"]

let str = "a,b,c,d,e f";
let arr = str.split(" ");
console.log(arr)

+ sign

This returns: [ "a", "very", "nice", "string" ]

let str = "a+very+nice+string";
let arr = str.split("+");
console.log(arr)

Destructure to create the Array

This returns: Firstname Lastname

const [firstName, lastName] = 'Firstname Lastname'.split(' ');
console.log(firstName, lastName)
