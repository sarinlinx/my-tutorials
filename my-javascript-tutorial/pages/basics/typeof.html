typeof (Data Types)

Use the typeof() method to determine the type of operator.

There are two syntaxes you can use.

typof var //more common

or

typeof(var)

Examples

This returns 'string'.

var str = "test"
alert ( typeof str );

This returns 'undefined'.

var str
alert ( typeof str );

This returns 'number'.

var str = 3
alert ( typeof str );

This returns 'boolean'.

var str = true;
alert ( typeof str );

This returns 'object'.

var person = {
    firstName:"John",
    lastName:"Doe",
    age:50,
    eyeColor:"blue"
};

alert ( typeof person );

Funcitons are objects, but this returns the typeof 'function' instead.

var str = function(){};
alert ( typeof str );

typeof cannot distinguis between types of Objects

These all return 'object'.

alert( typeof {} ) // 'object'
alert( typeof [] ) // 'object'
alert( typeof new Date ) // 'object'

Strange exceptions

This is a mistake from JS from the 90s but still exists for compatibility.

This returns 'object'.

var str = null;
alert ( typeof str );

NaN returns a type of 'number'.

var str = NaN;
alert ( typeof str );

Functions and Objects

The typeof operator returns object for both objects, arrays, and null.

The typeof operator does not return object for functions.

These return the following:

  • object
  • object
  • object
  • function
typeof {name:'john', age:34}
typeof [1,2,3,4]
typeof null
typeof function myFunc(){};
