Padding a String

Allows you to control the string size.

padStart

First value is length of string.

2nd value is what you'll pad the string with.

This returns: ++++++++++++Go to gate 23

const message = 'Go to gate 23';
console.log(message.padStart(25, '+'));

padEnd

First value is length of string.

2nd value is what you'll pad the string with.

This returns: Go to gate 23++++++++++++

const message = 'Go to gate 23';
console.log(message.padEnd(25, '+'));

Credit Card example

Convert the # to a String.

Remove last 4 #s using slice()

Use pad to add * to the beginning values.

This outputs: ************3253

const maskCreditCard = function(number) {
  // convert number to String
  // this works because if one of the operands of the + sign is a String, all operands are converted to a String
  const str = number + '';
  const last = str.slice(-4);
  return last.padStart(str.length, '*');
};

console.log(maskCreditCard(1234567891243253));
