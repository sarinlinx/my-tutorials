The Browser console

The console is a command line for the browser where you can write, manage, and monitor JavaScript on the fly.

How to open

On a PC: ctrl + shift + j

On a Mac: ctrl + opt + j

Open the console as a drawer

Opening the console as as drawer allows you to have 'dev tools' open while looking at something else (like HTML code) while having a separate window open for JavaScript.

  1. To open 'dev tools' click ctrl + shift + i.
  2. Click the three dot icon on the top right.
  3. From the dropdown choose 'Show console drawer'.
  4. If another tab is selected in the top section, the console appears as a drawer below it.
