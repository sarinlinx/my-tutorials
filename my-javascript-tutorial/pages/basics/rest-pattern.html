Rest Pattern

Packs elements into an Array/Object.

It looks like the Spread operator but does opposite of spread operator.

It depends on where ... is used as to if it will be Spread or Rest.

Collects elements that are unused in the destructuring assignment.

The Rest pattern collects all unused variables AFTER the last variable. This is by 'Pasta' is missing.

Skipped elements are NOT included.

The Rest element must be the last element.

Spread used where you would write value names separated by a comma.

Rest used where you would write variable names separated by a comma.

Rest Pattern example 1

This example deconstrcuts an Array.

2 variables are created, then the Rest pattern is used for the rest of the element.

They are placed into their own Array.

This prints: 1 2 [3, 4, 5]

const arr = [1, 2, ...[3, 4]]
const [a, b, ...others] = [1, 2, 3, 4, 5]
console.log(a, b, others)

Rest Pattern example 2

This prints: Pizza Risotto 'Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad']

The Rest pattern collects all unused variables AFTER the last variable. This is by 'Pasta' is missing.

Skipped elements are NOT included.

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto'],
}

const [pizza, , risotto, ...otherFood] = [...restaurant.mainMenu, ...restaurant.starterMenu]
console.log(pizza, risotto, otherFood)

Object example

This returns a new Object only containing Thursday and Friday.

This is because sat was already created as a variable, so the ... Rest operator took the rest of the element. 

   const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto'],

  openingHours: {
      thu: {
          open: 12,
          close: 22,
      },
      frid: {
          open: 11,
          close: 23,
      },
      sat: {
          open: 0, // Open 24 hours
          close: 24,
      },
  },
}

const {
  sat,
  ...weekdays
} = restaurant.openingHours;
console.log(weekdays)

Function example

This creates separate new Arrays based off an unknown # of arguments sent into the function.

The Rest operator is used for Rest arguments in this case.

const add = function(...numbers) {
  console.log(numbers)
}

add(2, 3)
add(5, 3, 7, 2)
add(8, 2, 5, 3, 2, 1, 4)
