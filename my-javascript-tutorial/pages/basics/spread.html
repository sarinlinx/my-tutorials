Spread operator

we can use the spread operator to basically expand an array into all its elements. So basically unpacking all the array elements at one.

This can only be used whenever you would have to write multiple values separated by commas. (like writing out all values of an Array), or passing artuments into a function.

You can only use the spread operator when building an array, or when we pass values into a function.

Create new array based off another, but add elements to beginning

Create a new Array with 2 values.

Next, add the spread operator (...) followed by the original Array name.

This expands all values from the original Array and prints they as if you wrote them all out individually.

You can use it again on the newArr to print all of those values.

 const arr = [7, 8, 9];

const newArr = [1, 2, ...arr];
console.log(newArr)
console.log(...newArr)

Pass new item into menu

Create a new Array from the old Array using the ... operator.

Add a new value at the end.

const restaurant = {
    name: 'Classico Italiano',
    location: 'Via Angelo Tavanti 23, Firenze, Italy',
    categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
    starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
    mainMenu: ['Pizza', 'Pasta', 'Risotto'],
}


// create new menu item
const newMenu = [...restaurant.mainMenu, 'Gnocci'];
console.log(newMenu)

Join Array

This creates and Array with both the starter menu and main menu.

const restaurant = {
    name: 'Classico Italiano',
    location: 'Via Angelo Tavanti 23, Firenze, Italy',
    categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
    starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
    mainMenu: ['Pizza', 'Pasta', 'Risotto'],
}

const mainMenuCopy = [...restaurant.mainMenu];

const menu = [...restaurant.starterMenu, ...mainMenuCopy, ]
console.log(menu)

Spread operator with Objects

create a new restaurant object with all the data from the original one plus some additional data.

This creates a new Object with a new restaurant name.

const restaurant = {
    name: 'Classico Italiano',
    location: 'Via Angelo Tavanti 23, Firenze, Italy',
    categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
    starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
    mainMenu: ['Pizza', 'Pasta', 'Risotto'],
}


// copy original Object
const restaurantCopy = {
    ...restaurant
};

restaurantCopy.name = 'Ristorante Roma';
console.log(restaurantCopy.name);
console.log(restaurant.name);
