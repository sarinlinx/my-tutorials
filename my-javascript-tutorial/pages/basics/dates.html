Creating a Date Object

4 ways to create a new Date Object

Date objects are created with the new Date() constructor.

There are 4 ways to create a new date object:

new Date()
new Date(year, month, day, hours, minutes, seconds, milliseconds)
new Date(milliseconds)
new Date(date string)

Date objects are static. The computer time is ticking, but date objects are not.

New Date Object with current date

By default, JavaScript will use the browser's time zone and display a date as a full text string:

Wed Nov 28 2018 18:27:38 GMT-0500 (Eastern Standard Time)

var d = new Date();
console.log(d);
Output

This example outputs:

Firefox
Date 2018-11-28T23:36:39.161Z
Chrome
Thu Nov 29 2018 17:34:47 GMT-0500 (Eastern Standard Time)

Creating a custom date Object

You can create a Date Object for any date you like.

7 numbers specify year, month, day, hour, minute, second, and millisecond

You can leave off everything except the year and month.

If only one value is specified, it's assumed it's milliseconds.

One and two digit years assume the 1900s.

var d = new Date(2018, 11, 24, 10, 33, 30, 0);;
console.log(d);

var d2 = new Date(99, 11, 24);
console.log(d2);

var d3 = new Date(9, 11, 24);
console.log(d3);
Output

This example outputs:

Mon Dec 24 2018 10:33:30 GMT-0500 (Eastern Standard Time)
Fri Dec 24 1999 00:00:00 GMT-0500 (Eastern Standard Time)
Fri Dec 24 1909 00:00:00 GMT-0500 (Eastern Standard Time)

Creating a date from a String

You can also create a date Object using a String.

var d = new Date("October 13, 2014 11:13:00");
console.log(d);
Output

This example outputs:

Mon Oct 13 2014 11:13:00 GMT-0400 (Eastern Daylight Time)

How dates are stored

JavaScript stores dates as number of milliseconds since January 01, 1970, 00:00:00 UTC (Universal Time Coordinated).

var d = new Date(0);
console.log(d);
Output

This example outputs:

(Firefox)
1970-01-01T00:00:00.000Z

Example adding 100000000000 ms

This adds 100000000000 ms from January 01, 1970, 00:00:00 UTC.

NOTE: You can also use negative values.

var d = new Date(100000000000);
console.log(d);
Output

This example outputs:

(Firefox)
1973-03-03T09:46:40.000Z

String methods

When you display a date object in HTML, it is automatically converted to a string, with the toString() method. You don't need to specify toString() in your code if you don't want to. JavaScript does it automatically.

However, you're not outputting to HTML, you'd need to specify the toString() method, otherwise it will appear different.

You can also use toUTCString() to format it differently.

The toDateString() method converts a date to a more readable format:

var d = new Date(100000000000);
console.log(d);
console.log(d.toString())
console.log(d.toUTCString())
console.log(d.toDateString())
Output

This example outputs:

(Firefox)
Date 1973-03-03T09:46:40.000Z
Sat Mar 03 1973 04:46:40 GMT-0500 (Eastern Standard Time)
Sat, 03 Mar 1973 09:46:40 GMT
Sat Mar 03 1973
