Boolean

A JavaScript Boolean represents one of two values: true or false.

Booleans are primitive values created from literals.

You can create a Boolean Object using the 'new' keyword, but this is never recommended since it slows down and complicates your code.

Example

Assigning a variable to a comparison that uses an operator makes it a Boolean variable.

This prints 'true'.

let temp = 32
let isFreezing = temp <= 32

console.log(isFreezing)

This prints 'false', then 'true'.

// Create age set to your age
let age = 65
// Calculate is child - if they are 7 or under
let isChild = age <= 7
// Calculate is senior - if they are 65 or older
let isSenior = age >= 65
// Print is child value
console.log(isChild)
// Print is senior value
console.log(isSenior)

The Boolean() function

You can use the Boolean() function to find out if an expression (or a variable) is true.

This returns: true

console.log(Boolean(10 > 9));

Everything with a value is TRUE

If there is a value at all, even without a comparison, the result is true.

This returns: true

console.log(Boolean(10));

False values using Boolean()

The following will all return false.

console.log(Boolean(0));
console.log(Boolean(-0));
console.log(Boolean(""));
var undefinedVar;
console.log(Boolean(undefinedVar));
console.log(Boolean(null));
console.log(Boolean(NaN));
