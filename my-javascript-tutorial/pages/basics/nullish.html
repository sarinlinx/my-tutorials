Setting a default value if the value doesn't exist

This checks if restaurant.numGuests exists. If so, the number is returned. If not, the default value of is returned.

This can be used instead of the ternery operator.

NOTE: This will not work if guests value is zero since that's a value. Look at nullish operator example below for solution.

 const restaurant = {
  numGuests: 1
}
              
              //ternery example
const guests1 = restaurant.numGuests ? restaurant.numGuests : 10;
console.log(guest1)

// short circuiting example
const guests2 = restaurant.numGuests || 10;
console.log(guests2)

nullish with zero

This is the same as above but uses nullish operator for the zero value.

This correctly returns zero.

This works because the nullish operator works with nullish values, not falsey values. Nullish values are null and undefined.

The DO NOT include zero or an empty string.

So it's not treating the zero as a falsey value.

 const restaurant = {
  numGuests: 1
}

              
const guests2 = restaurant.numGuests || 10;
console.log(guests2)
