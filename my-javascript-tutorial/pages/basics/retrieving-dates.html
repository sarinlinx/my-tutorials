Retrieiving Dates

Show Year only

To specifically retrieve the year, use the method of the Object: getFullYear()

var d = new Date();
console.log(d.getFullYear());
Output

This example outputs:

2018

Show Month only

To specifically retrieve the month, use the method of the Object: getMonth()

NOTE: The months count from zero, so to view the correct month, you must add 1.

var d = new Date();
console.log(d.getMonth() + 1);
Output

This example outputs:

11

Show Day only

To specifically retrieve the day of the month, use the method of the Object: getDate()

var d = new Date();
console.log(d.getDate());
Output

This example outputs:

29

Show Hours only

To specifically retrieve the hour, use the method of the Object: getHours()

This returns military time.

var d = new Date();
console.log(d.getHours());
Output

This example outputs:

17

Show Minutes only

To specifically retrieve the minute, use the method of the Object: getMinutes()

This returns military time.

var d = new Date();
console.log(d.getMinutes());
Output

This example outputs:

31

Show Seconds only

To specifically retrieve the second, use the method of the Object: getSeconds()

var d = new Date();
console.log(d.getSeconds());
Output

This example outputs:

47

Show Milliseconds only

To specifically retrieve the millisecond, use the method of the Object: getMilliseconds()

var d = new Date();
console.log(d.getMilliseconds());
Output

This example outputs:

847

Using the Number() method to display Milliseconds since 1/1/1970

The Number() method converts a date to a number.

let x = new Date("2017-09-30");
let index = Number(x);
console.log(index)
Output

This example outputs:

1506729600000

Show Day of week

To specifically retrieve the day of the week, use the method of the Object: getDay()

NOTE: In JavaScript, the first day of the week (0) means "Sunday".

var d = new Date();
console.log(d.getDay());
Output

This example outputs:

1

UTC time

To retrieve UTC time, just preface all of the examples above with UTC. For example.

  • getUTCDate()
  • getUTCHours()
  • getUTCSeconds()
