Artithmatic operators

Remainder

The modulous symbol % returns the remainder.

5 % 2; //returns 1

Exponentiation

For a natural number b, the result of a ** b is a multiplied by itself b times.

alert( 2 ** 2 );     // 4  (2 * 2)
alert( 2 ** 3 );     // 8  (2 * 2 * 2)
alert( 2 ** 4 );     // 16 (2 * 2 * 2 * 2)
alert( 4 ** (1/2) ); // 2 (power of 1/2 is the same as a square root, that's maths)
alert( 8 ** (1/3) ); // 2 (power of 1/3 is the same as a cubic root)

Increment/decrement

Increment/decrement can be applied only to a variable. An attempt to use it on a value like 5++ will give an error.

Operators ++ and -- can be placed both after and before the variable.

  • prefix form -When the operator stands before the variable: ++counter.
  • postfix form - When the operator goes after the variable: counter++.

The difference is the 'prefix form' returns the new value while the 'postfix form' returns the old value.

If you do not need the result of the increment/decrement, then it doesn't matter which form you use.

Prefix example

This returns the new value after increment.

Use this if you need to increase the value, and use the result of the operator right now.

let counter = 1;
let a = ++counter;

alert(a); // 2

Postfix example

This returns the old value prior to increment.

Use this if you need to increase the value, but use the previous result.

let counter = 1;
let a = counter++;

alert(a); // 1

Modify-in-place

Modifying-in-place shortens your notation.

let n = 2;
n += 5; // now n = 7 (same as n = n + 5)
n *= 2; // now n = 14 (same as n = n * 2)

alert( n ); // 14

Comma

This used mostly in Frameworks for more complex constructs.

It's odd and difficult to read so it should generally not be used.

Put several expressions in parenthesis and separate by a comma. Only the last expression is returned.

let a = (1 + 2, 3 + 4);

alert( a ); // 7 (the result of 3 + 4)
