If statement

Simple if statement

Type the word 'if' following by parenthesis.

Your condition statement goes in the parenthesis.

Add a section contained in braces. This is what happens as a result of the if statement.

let temp = 180

if (temp <= 32) {
    console.log('It is freezing outside!')
}

if or else statements check if the condition is true

If the condition is true, the result code is run.

If the condition is false, the code is not run.

This prints 'Welcome' since the condition is not true.

let isAccountLocked = false

if (isAccountLocked) {
    console.log('Is account locked')
} else {
    console.log('Welcome')
}

else if statement

If the 'if statement' is not true, you can add additional 'else if' and and final 'else' section to take care of those cases.

The additional else if statements can be used to look for additional conditions to check.

This prints 'Wecome' because the first if statement is NOT true, and the 'else if' statement is NOT true.

let isAccountLocked = false
let userRole = 'user'

if (isAccountLocked) {
    console.log('Is account locked')
} else if (userRole === 'admin') {
    console.log('Welcome Admin')
} else {
    console.log('Welcome')
}

boolean else if

This prints 'It is freezing outside' since the first if condition is TRUE.

let temp = 4

if (temp <= 32) {
    console.log('It is freezing outside')
} else if (temp >= 110) {
    console.log('It is hot outside')
} else {
    console.log('Go for it. It is pretty nice out')
}
