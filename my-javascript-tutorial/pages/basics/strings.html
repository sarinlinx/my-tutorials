Strings

Strings are text.

In JavaScript, all strings are in unicode, no matter which type of encoding is used.

Basics

Creating a string

Put the string in quotes.

var text = "my text string";

Adding a number and a string

The result is 45 because the number is just joined with the string.

If you check typeof, you'll see it's a string now.

var a = 4;
var b= "5";
var sum = a + b;

alert(sum);
alert(typeof sum);

Escaping quotes

To escape a quote, add a \ slash before the quote.

Escape a single quote in a single quoted string

A single quote inside a single quoted string must be escaped.

var str = 'I\'m the Walrus';
alert(str)

Escape a single quote in a double quoted string

A single quote inside a double quoted string does not need to be escaped.

var str = "I'm the Walrus";
alert(str)

Escape double quotes in a double quoted string

A double quote inside a double quoted string must be escaped.

var str = "double \"quote\" ";
alert(str)

Breaking Long Code Lines

For best readability, programmers often like to avoid code lines longer than 80 characters.

There are a few ways to do this.

Breaking after an operator

The clearest place to break is after an operator.

document.getElementById("demo").innerHTML =
"Hello Dolly!";

Using a backslash

This is not recommended since some browsers may not support it.

document.getElementById("demo").innerHTML = "Hello \
Dolly!";

Using string addition with +

Use the + sign to add two string together.

document.getElementById("demo").innerHTML = "Hello " +
"Dolly!";

Template Strings

String templates are a different option to concatentation. They make your code easier to write and read.

Use the ` character to enclose a template string.

The variable string is written as: ${varName}

This takes the place of string concatenation using the + character.

This shows the difference in the two methods.

var name="Andrew";
concatenation
console.log("Hi, my name is " + name);
String template
console.log(`Hi, my name is ${name}`);
