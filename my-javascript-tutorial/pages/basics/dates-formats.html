Date input formats

3 types of JavaScript date input format

Type Example
ISO Date "2015-03-25" (The International Standard). The ISO format follows a strict standard in JavaScript. The other formats are not so well defined and might be browser specific.
Short Date "03/25/2015" - Make sure to add a zero or the browser may throw an error.
Long Date "Mar 25 2015" or "25 Mar 2015" - Month and day can be in any order. The month can also be fully written out.

Undefined dates

The following date formats are undefined.

  • YYY/MM/DD
  • DD-MM-YYY

Date Output

Independent of input format, JavaScript will (by default) output dates in full text string format.

Tue Mar 24 2015 20:00:00 GMT-0400 (Eastern Daylight Time)

ISO Dates

ISO 8601 is the international standard for the representation of dates and times.

The ISO 8601 syntax (YYYY-MM-DD) is also the preferred JavaScript date format

NOTE: The computed date will be relative to your time zone.

var d = new Date("2015-03-25");
console.log(d.toString());
Output

This example outputs:

Tue Mar 24 2015 20:00:00 GMT-0400 (Eastern Daylight Time)

Leave out the Day

ISO dates can be written without specifying the day (YYYY-MM):

NOTE: There's no reason to do this. The day will still display, but it will default to the last day of the month.

var d = new Date("2015-03");
console.log(d.toString());
Output

This example outputs:

Sat Feb 28 2015 19:00:00 GMT-0500 (Eastern Standard Time)

Only year

You can also write a date with only the year.

However, there's no reason to do this. The month and day will still display, but it will default to the last day of the year.

Adding Hours, Minutes, and Seconds

Date and time is separated with a capital T.

UTC time is defined with a capital letter Z.

var d = new Date("2018-12-03T12:00:00Z");
console.log(d.toString());
Output

Notice how the date automatically calcualtes based on my local time which is -5hrs from UTC.

Mon Dec 03 2018 07:00:00 GMT-0500 (Eastern Standard Time)

Modify time relative to UTC

If you want to modify the time relative to UTC, remove the Z and add +HH:MM or -HH:MM instead: 

var d = new Date("2018-12-03T12:00:00-05:00");
console.log(d.toString());
Output

This example outputs:

Mon Dec 03 2018 12:00:00 GMT-0500 (Eastern Standard Time)

Time Zones

When setting or getting a date (without specifying the timezone), the browser's timezone is used.

lkjlkjlkjlkj
Output

This example outputs:

jljlkjlkjlk

Parsing a date into milliseconds

If you have a valid date string, you can use the Date.parse() method to convert it to milliseconds.

Date.parse() returns the number of milliseconds between the date and January 1, 1970:

var msec = Date.parse("March 21, 2012");
console.log(msec.toString());
Output

This example outputs:

1332302400000
