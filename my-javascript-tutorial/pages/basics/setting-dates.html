Setting Dates

Set Date methods let you set date values, rather than retrieve the current time.

Set the year

use setFullYear() to set the year.

This sets the year to 2020.

var d = new Date();
d.setFullYear(2020);
console.log(d.toString())
Output

This example outputs:

Thu Dec 03 2020 19:40:12 GMT-0500 (Eastern Standard Time)

Set the month

Use setMonth() to set the month.

This sets the month to November.

NOTE: The months start at zero. So 10 is November.

var d = new Date();
d.setMonth(10);
console.log(d.toString())
Output

This example outputs:

Sat Nov 03 2018 19:42:35 GMT-0400 (Eastern Daylight Time)

Set the day of the month

Use setDate() to set the day of the month.

This sets the day of the month to the 20th.

var d = new Date();
d.setDate(20);
console.log(d.toString())
Output

This example outputs:

Thu Dec 20 2018 19:38:11 GMT-0500 (Eastern Standard Time)

Set the hour

Use setHours() to set the hour of the day in military time.

var d = new Date();
d.setHours(13);
console.log(d.toString())
Output

This example outputs:

Mon Dec 03 2018 13:47:05 GMT-0500 (Eastern Standard Time)

Set the minutes

Use setMinutes() to set the minute of the hour.

var d = new Date();
d.setMinutes(30);
console.log(d.toString())
Output

This example outputs:

Mon Dec 03 2018 19:30:27 GMT-0500 (Eastern Standard Time)

Set the seconds

Use setSeconds() to set the seconds.

var d = new Date();
d.setSeconds(30);
console.log(d.toString())
Output

This example outputs:

Mon Dec 03 2018 19:49:30 GMT-0500 (Eastern Standard Time)

Set the Milliseconds

Use setMilliseconds() to set the seconds.

var d = new Date();
d.setMilliseconds(999);
console.log(d)
Output

This example outputs:

Date 2018-12-04T00:50:49.999Z
