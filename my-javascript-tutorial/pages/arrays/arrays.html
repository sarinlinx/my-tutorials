Overview

An array is a special variable, which can hold more than one value at a time.

This is different from arrays in some languages which represent a contiguous segment of memory, and also different from queue/stack structures based on linked-lists.

Array Syntax

There are two ways to create an Array.

  • Using the 'new' keyword' or not.

For simplicity do not use the 'new' word.

Using the 'new' keyword
const cars = new Array("Saab", "Volvo", "BMW");
Not using the 'new' keyword
const myarray = ["keys", "stuff", "things"]

The length Property

The length property of an array returns the length of an array (the number of array elements).

This returns 4

const fruits = ["Banana", "Orange", "Apple", "Mango"];
alert(fruits.length);

Viewing Array contents

Viewing the full Array contents

Output to console.log

This displays the following:

  • Array(4) [ "red", "blue", "green", "orange" ]
const pens = ["red", "blue", "green", "orange"];
console.log(pens);

Access single element

You can also access a single index.

This outputs: Saab

const cars = ["Saab", "Volvo", "BMW"];
console.log(cars[0]);

Access the last element

Use the length property and add -1.

This prints 'Mango'.

fruits = ["Banana", "Orange", "Apple", "Mango"];
const last = fruits[fruits.length - 1];
console.log(last);

Changing an Array Element

You can change an Array element by directly assigning it.

This outputs: Opel,Volvo,BMW

const cars = ["Saab",
            "Volvo",
            "BMW"];
cars[0] = "Opel";

Different types of Array elements

You can have objects in an Array. You can have functions in an Array. You can have arrays in an Array.

Difference Between Arrays and Objects

Arrays are a special kind of objects, with numbered indexes.

In JavaScript, arrays use numbered indexes.

In JavaScript, objects use named indexes.

Check if a variable is an Array

Option 1 - Array.isArray()

typeof will return Object since Arrays are Objects.

ECMAScript 5 defines a new method Array.isArray(). This returns 'true' if it's an Array.

const cars = ["Saab","Volvo","BMW"];
console.log(Array.isArray(cars));

Option 2 - Create your own function

The function always returns true if the argument is an array.

Or more precisely: it returns true if the object prototype contains the word "Array"

const cars = ["Saab","Volvo","BMW"];

function isArray(myArray) {
    return myArray.constructor.toString().indexOf("Array") > -1;
}

console.log(isArray(cars));

Option 3 - instanceof operator

The instanceof operator returns true if an object is created by a given constructor.

const cars = ["Saab","Volvo","BMW"];

console.log(cars instanceof Array);
