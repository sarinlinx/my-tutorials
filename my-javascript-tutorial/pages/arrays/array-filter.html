filter()

filter() creates a new array, but only with elements that pass a specific test in the function.

The original array is not touched.

This example creates a new array from elements with a value larger than 18

var numbers = [45, 4, 9, 16, 25];
var over18 = numbers.filter(myFunction);

function myFunction(value) {
  return value > 18;
}

console.log(over18);
Output

This example outputs:

Array [ 45, 25 ]

The index and array params

Just as above, the index and array parameters can be added or omitted.

Searching an Array Object for a string

This searches the Array named 'notes' for a string.

If found, the Object within that string is returned (from the new Array created by filter())

//Create an Array of Objects
const notes = [{
    title: 'My next trip',
    body: 'I would like to go to Spain'
}, {
    title: 'Habbits to work on',
    body: 'Exercise. Eating a bit better.'
}, {
    title: 'Office modification',
    body: 'Get a new seat'
}]


// This filters the notes Array based on a string
// notesArray is the Array param you're sending in
// query is the string you're searching for
// findNotes will contain the new Array created by filter()
const findNotes = function(notesArray, query) {
    // use filter() to return a new array based off a new function
    // 1st param is the Array Object
    // filter() returns a boolean
    // if TRUE, it is kept and sent to the new array
    return notesArray.filter(function(notez, index) {
        // notez. is the full Object
        // notez.title is a String (cycles through all)
        // You can then run the String method toLowerCase()
        // You can then 'chain' another String method of .includes since you're still working with a String
        // Pass into includes() the search text using the 'query' argument
        const isTitleMatch = notez.title.toLowerCase().includes(query.toLowerCase())
        const isBodyMatch = notez.body.toLowerCase().includes(query.toLowerCase())

        // includes returns a boolean
        // If the boolean is TRUE for either const, it is returned
        return isTitleMatch || isBodyMatch
    })
}

  // Call the function
  console.log(findNotes(notes, 'eating'))
Output

This example outputs:

[
  {
    title: 'Habbits to work on',
    body: 'Exercise. Eating a bit better.'
  }
]

Filter out negative values

This filters the movements array and returns only the positive values into a new array title deposits.

const movements = [200, 450, -400, 3000, -650, -130, 70, 1300];

// filter negative values
const deposits = movements.filter(function(mov) {
  return mov > 0;
})

console.log(deposits)
