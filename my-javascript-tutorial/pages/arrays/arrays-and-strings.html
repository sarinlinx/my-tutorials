Turn array into a string

Using join()

Convert the array into a string assigned to a new var

The join() method turns an array into a string.

This first outputs the array: Array(4) [ "Lemon", "Apple", "Orange", "Peach" ]

After the join() is used, it outputs: Lemon, Apple, Orange, Peach

var fruits = ["Lemon","Apple","Orange","Peach"];
console.log(fruits);
var str = fruits.join(', ');
console.log(str);

Convert the array into a string with a specific separator

It behaves just like toString(), but in addition you can specify the separator.

This outputs: Banana * Orange * Apple * Mango

var fruits = ["Banana", "Orange","Apple", "Mango"];
document.getElementById("demo").innerHTML = fruits.join(" * ");

Using toString()

Convert the array into a string with comma separated values

This outputs Banana,Orange,Apple,Mango

var fruits = ["Banana", "Orange", "Apple", "Mango"];
document.getElementById("demo").innerHTML = fruits.toString();

Turn String into an Array

Using split()

The split() method converts a string into an Array.

This outputs: Apple

var fruits = "Apple,Orange,Peach";
var arr = fruits.split(',');
console.log(arr[0]);
