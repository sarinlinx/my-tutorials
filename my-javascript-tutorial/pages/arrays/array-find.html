find()

Requires a callback function that returns a boolean.

This is different from filter() because filter() returns all the elements that match the condition while the Find method only returns the first one

And filter() returns a new Array where find() only returns a single element.

find() returns the value of the first array element that passes a test function.

This example finds (returns the value of ) the first element that is larger than 18.

var numbers = [4, 9, 16, 25, 29];
var first = numbers.find(myFunction);

function myFunction(value, index, array) {
    return value > 18;
}

console.log(first);
Output

This example outputs:

25

The index and array params

Just as above, the index and array parameters can be added or omitted.

Using find() in an Array of Objects

This can be used instead of indexOf() when searching an Array of Objects.

const notes = [{
    title: 'My next trip',
    body: 'I would like to go to Spain'
}, {
    title: 'Habbits to work on',
    body: 'Exercise. Eating a bit better.'
}, {
    title: 'Office modification',
    body: 'Get a new seat'
}]


// create function
// 1st argument takes in the Array of Objects
// 2nd argument is a text string you're searching for
// return note object if found
// return undefined otherwise
const findNote = function(notesArray, noteTitle) {
  // you don't need to add index unless you're using it
    // run findIndex() on the 'notes' Array. It returns the index of the first array element that passes a test function.
    // notez is the full Array of Objects

    // immediately return whatever comes back from find()
    return notesArray.find(function(notez, index) {
        return notez.title.toLowerCase() === noteTitle.toLowerCase()
    })
}

// call function
// 1st argument is the Array of Objects
// 2nd argument is the text string to search for
const note = findNote(notes, 'Office modification')
console.log(note)
Output

This example outputs:

{ title: 'Office modification', body: 'Get a new seat' }

const account2 = {

};

const account3 = {
  owner: 'Steven Thomas Williams',
  movements: [200, -200, 340, -300, -20, 50, 400, -460],
  interestRate: 0.7,
  pin: 3333,
};

const account4 = {
  owner: 'Sarah Smith',
  movements: [430, 1000, 700, 50, 90],
  interestRate: 1,

const accounts = [account1, account2, account3, account4];

const account = accounts.find(acc => acc.owner === 'Jessica Davis')
