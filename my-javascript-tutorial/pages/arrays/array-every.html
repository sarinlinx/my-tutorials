every()

every() method check if all array values pass a test.

It returns true or false.

var numbers = [45, 4, 9, 16, 25];
var allOver18 = numbers.every(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

console.log(allOver18);
Output

This example outputs:

false

The index and array params

Just as above, the index and array parameters can be added or omitted.

Example

Returns false since all values are not over 0

const movements = [200, 450, -400, 3000, -650, -130, 70, 1300];
const anyDeposits = movements.every(mov => mov > 0)
console.log(anyDeposits)
