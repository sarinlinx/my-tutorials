reduce()

Used to boil down all the elements of the original array into one single value. And an example of this can be to add all the elements of an array together.

In this way, it has 'reduced' the Array to a single value.

Calculate sum of all values.

reduce() takes 3 params: current Element, Index, Array

The first param is called the accumulator.

0 is the initial value of the accumulator in the first iteration.

const movements = [200, 450, -400, 3000, -650, -130, 70, 1300];

// calculate sum of all values
const balance = movements.reduce(function (acc, cur, i, arr) {
  // loop of Array
  return acc + cur
}, 0);
console.log(balance)
