flat()

Take a 1 Level Deep nested Array and combine them into a single Array.

This takes no arguments.

const arr = [[1,2,3], [4,5,6], 7,8];
// put sub elements into single Array
console.log(arr.flat())

flat() with depth

Same thing but 2 levels of Nested Arrays.

You can then add the numerical param to go deeper.

const arrDeep = [[[1,2],3], [4,[5,6]], 7,8];
console.log(arrDeep.flat(2))

flat() with depth

It's common to use .map with .flat.

But, it's better for performance if you use .flatMap() which combines them.

It's a map method that flattens the result.

NOTE: This only goes 1 level deep and you CANNOT change it.

const account1 = {
  owner: 'Jonas Schmedtmann',
  movements: [200, 450, -400, 3000, -650, -130, 70, 1300],
  interestRate: 1.2, // %
  pin: 1111,
};

const account2 = {
  owner: 'Jessica Davis',
  movements: [5000, 3400, -150, -790, -3210, -1000, 8500, -30],
  interestRate: 1.5,
  pin: 2222,
};

const account3 = {
  owner: 'Steven Thomas Williams',
  movements: [200, -200, 340, -300, -20, 50, 400, -460],
  interestRate: 0.7,
  pin: 3333,
};

const account4 = {
  owner: 'Sarah Smith',
  movements: [430, 1000, 700, 50, 90],
  interestRate: 1,
  pin: 4444,
};

const accounts = [account1, account2, account3, account4];


const overalBalance = accounts
.flatMap(acc => acc.movements)
.reduce((acc, mov) => acc + mov, 0);

console.log(overalBalance)
