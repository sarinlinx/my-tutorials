map()

Perform a function on each array element

map() method creates a new array by performing a function on each array element.

The map() method does NOT change the original array.

The array must have values for this to work.

This example multiplies each array value by 2, creates a new array, then prints that new array.

var numbers1 = [45, 4, 9, 16, 25];
//Create new array using map()
//Run the function to create the new array
var numbers2 = numbers1.map(myFunction);

document.getElementById("demo").innerHTML = numbers2;

function myFunction(value, index, array) {
    return value * 2;
}
Output

This example outputs:

90,8,18,32,50

The index and array params

Just as above, the index and array parameters can be added or omitted.

Map Array example

This maps the first Array and uses the value of eurToUsd to multiply all values into the new Array.

const movements = [200, 450, -400];

const eurToUsd = 1.5;

const movementsUSD = movements.map(function(curEl){
  return curEl * eurToUsd;
})

console.log(movements)
console.log(movementsUSD)
Output

This example outputs:

[200, 450, -400];
[300, 675, -600]

Compute a username with a map

This extracts the first letter of each name.

This outputs: stw

// usernames are initials
const user = 'Steven Thomas Williams';
// split() created a new Array. Loop over Array with .map
const username = user.toLowerCase().split(' ').map(function(name) {
  // return first letter
  return name[0];
}).join('');

console.log(username)

Same thing but rewritten with an Arrow Function

The individual methods are also put on different lines.

const user = 'Steven Thomas Williams';
const username = user
.toLowerCase()
.split(' ')
.map(name => name[0])
.join('');

console.log(username)

Same thing but everything is inside a Function

const createUsernames = function (user) {

const username = user
  .toLowerCase()
  .split(' ')
  .map(name => name[0])
.join('');
return username;
}

console.log(createUsernames('Steven Thomas Williams'));
