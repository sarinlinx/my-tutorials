Searching with indexOf()

Searching with indexOf() using the syntax: Syntax: arrayName.indexOf("termToSearch", indexToStartFrom);

This searches for item with word 'Orange' starting at index 2.

This outputs: The index position is: 2

NOTE: If you started searching after the index number where it's located, it would return -1

The search term is also Case Sensitive.

var fruits = ["Kiwi", "Banana", "Orange", "Lemon", "Apple", "Mango"];
var result = fruits.indexOf("Orange", 2);
console.log("The index position is: ", result);

Using findIndex()

Overview

indexOf() will not work on an Array of Objects. You need to sue findIndex()

The findIndex() method returns the index of the first element in the array that satisfies the provided testing function.

Otherwise, it returns -1, indicating no element passed the test.

Example

First, create a function that takes a param.

In that function, check the argument.PROPERTYNAME against a string.

Run the findIndex() method on the Array name: ARRAY.findIndex()

console.log that to view the index value where that string appears.

This returns: 1

function myFunction(something) {
  return something.title === 'Habbits to work on'
}

let indexValue = notes.findIndex(myFunction)
console.log(indexValue)

Another way to write it without naming the function

Here's another way to write it.

const index = notes.findIndex(function (param) {
  return param.title === 'Habbits to work on'
})

console.log(index)

Search for a string, then return the note Object using find()

This searches for a string, but if found, the Object is returned instead of the index.

This returns the object that matches the string.

This returns: Object { title: "Office modification", body: "Get a new seat" }

//create a function with 2 params
//param1 is the notes Array you're searching through
//noteString is the string you're looking for in the Array
const findNote = function (param1, noteString) {
    //Search the note Object with find(). This takes a param which is the
    //value of the current element. In this case, it's the
    //Objects within the 'notes' Array.
    //Return the note Object if found, otherwise undefined
    return param1.find(function (paramx) {
        //Checks all the titles of all Objects
        //to see if any equal the string input
        //If one does, it's returned
        return paramx.title.toLowerCase() === noteString.toLowerCase()
    })
}

//Run findNote() with 2 params
//notes is the name of the Array you're searching through
//string you're looking for
const note = findNote(notes, 'Office modification')
console.log(note)

This does the same as above, but uses findIndex()

This searches for a string, but if found, the Object is returned instead of the index.

This returns the object that matches the string.

This returns: Object { title: "Office modification", body: "Get a new seat" }

//param1 is the notes Array you're searching through
//noteString is the string you're looking for in the Array
const findNote = function (param1, noteString) {
     //Search ARRAY.findIndex()
     //param2 is the value of the current element. In this case, it's the
     //Objects within the 'notes' Array.
     //'index' stores the Object matche, otherwise -1 is returned
     const indexVal = param1.findIndex(function (param2) {
         //Checks all the titles of all Objects
         //to see if any equal the string input
         //If one does, it's returned
         return param2.title.toLowerCase() === noteString.toLowerCase()
     })
     //return the Object title
     //Return the note Array and the indexVal variable you created above
     //to store the result of the search
     return notes[indexVal]
 }

//Run findNote() with 2 params
//notes is the notes you're searching through in the Array
//string you're looking for
const note = findNote(notes, 'Office modification')
console.log(note)
