fill()

Fill an empty Array

Create an empty Arry with 7 indexes

Use fill() to fill up the Array with a value.

const x = new Array(7)
x.fill(1)
console.log(x)

Start filling at index 3

Similar to slice() add an index to start at.

This start filling at index 3

const x = new Array(7)
x.fill(1, 3)
console.log(x)

Start filling at index 3 and end at 5

add an index to start and end at.

const x = new Array(7)
x.fill(1, 3, 5)
console.log(x)

Mutate existing Array

Fill existing Array with values

const arr = [1,2,3,4,5,6,7]
console.log(arr.fill(23,4,6))

Array.from()

This uses .from() on an Array constructor

Array is a function. On that object you run the method .from()

You also use this to create Array from other things like iterables

const y = Array.from({length: 7}, () => 1)
console.log(y)

Select all elements with a class name

This uses .querySelectroAll() and .from to place all HTMl elements into an Array.

const x = Array.from(document.querySelectorAll('.className'));
console.log(x);
