Editing Array items

The splice() method

The splie() method can be used to add or remove items to the Array.

The first parameter (2) defines the position where new elements should be added (spliced in).

The second parameter (0) defines how many elements should be removed.

The rest of the parameters ("Lemon" , "Kiwi") define the new elements to be added.

Removing items from an Array

splice() deletes an index, then all other indexes shift their value.

This removes one element starting at index zero.

This outputs the word: to home

var myArray = ["Go", "to", "home"]

//starting at index 0, remove 1 element
myArray.splice(0, 1)

//"to" became first element
console.log(myArray[0] + " " + myArray[1])

Adding items

This example positions at index 2.

It then is told zero items should be removed.

The items to add are then listed.

They are inserted at index 2.

This outputs: Array(6) [ "Banana", "Orange", "Lemon", "Kiwi", "Apple", "Mango" ]

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2, 0, "Lemon", "Kiwi");
console.log(fruits);

You can also use negative values

Removing items

The pop() method

Removing the last item

The pop() method removes the last element from an array.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
// just run the method to remove the last item
fruits.pop();

"Mango" is removed. You can then assign it to a variable.

This assigns a variable x to the last itme in the array.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
// the value of x is "Mango"
var x = fruits.pop();

The shift() method

Removing the first item

Shifting is equivalent to popping, working on the first element instead of the last.

The shift() method removes the first array element and "shifts" all other elements to a lower index.

"Banana" is removed. The index values of the remaining items are shifting down by 1.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
// Removes the first element "Banana" from fruits
fruits.shift();
console.log(fruits);

Adding items

Using the push() method

Adding an item to the end

The push() method adds a new element to an array.

This prints: Array(5) [ "Banana", "Orange", "Apple", "Mango", "Kiwi" ]

var fruits = ["Banana", "Orange", "Apple", "Mango"];

// Adds a new element ("Kiwi") to fruits
fruits.push("Kiwi");
console.log(fruits);

The length property

Adding using the length property

This adds a new element at the next index value.

This prints: 4
Array(5) [ "Banana", "Orange", "Apple", "Mango", "Lemon" ]

var fruits = ["Banana", "Orange", "Apple", "Mango"];
//This prints 4, the new index value, which is also the total number of items
console.log(fruits.length);
// adds a new element (Lemon) to fruits
fruits[fruits.length] = "Lemon";
console.log(fruits);

Adding by specifying the index value

This adds a new element at a specific index value.

Note that this will create a gap in the index values.

This prints: Array(10) [ "Banana", "Orange", "Apple", "Mango", <5 empty slots>, "Lemon" ]

unshift()

Adding using the unshift() method

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements.

This prints: Array(5) [ "Lemon", "Banana", "Orange", "Apple", "Mango" ]

All index values are shifted 1 value higher due to the new item being inserted at the beginning.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
// Adds a new element "Lemon" to fruits
fruits.unshift("Lemon");
console.log(fruits);

Changing an element

Change an item by specifying its index number and its new value.

This outputs: Array(4) [ "Kiwi", "Orange", "Apple", "Mango" ]

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits[0] = "Kiwi";
// Changes the first element of fruits to "Kiwi"
console.log(fruits);

Delete an element

Since JavaScript arrays are objects, elements can be deleted by using the JavaScript operator delete.

However, using delete may leave undefined holes in the array. So you should use pop() or shift() instead.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
// Changes the first element in fruits to undefined
delete fruits[0];

Joining Arrays

The concat() method creates a new array by concatenating two arrays.

This outputs: Array(5) [ "Cecilie", "Lone", "Emil", "Tobias", "Linus" ]

var myGirls = ["Cecilie", "Lone"];
var myBoys = ["Emil", "Tobias","Linus"];
// .concat joins the 2 arrays
var myChildren = myGirls.concat(myBoys);
console.log(myChildren);

Slicing an Array using slice()

Slicing from a single index value to the end

The slice() method slices out a piece of an array into a new array.

This example slices out a part of an array starting from array element 1 ("Orange") and goes to the end of the array.

This outputs: Array(4) [ "Orange", "Lemon", "Apple", "Mango" ]

var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
var citrus = fruits.slice(1);
console.log(citrus);

Slicing from a single index value toa specific index value

The slice() method can take two arguments. For example: slice(1,3).

The method then selects elements from the start argument, and up to (but not including) the end argument.

This starts at item 1 and goes to item 3. BUT, item 3 is NOT included.

This outputs: Array [ "Orange", "Lemon" ]

var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
var citrus = fruits.slice(1,3);
console.log(citrus);

Slicing with a negative value

Using a negative value extract the last indexes from the Array.

This returns: Apple, Mango.

var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
var citrus = fruits.slice(-2);
console.log(citrus);

Slicing with 2 values where the 2nd is negative

This starts at position 1 and goes to position -2.

This returns: Orange, Lemon.

var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
var citrus = fruits.slice(-2);
console.log(citrus);
