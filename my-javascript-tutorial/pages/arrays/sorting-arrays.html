Sorting Arrays with sort()

sort() automatically sorts an Array.

It does not require a function as the first argument, but it can take one.

The optional compare function will allow you to specify the sorting behavior.

Alphabetical

It works as expected with alphabetical strings.

Numbers

It does not work well with numbers and requires more work to put the values in correct order.

Alphabetical sorting

sort() puts alphabetical strings in alpabetical order.

Uppercase letters come before lower case letters.

This prints: Apple,Banana,Mango

NOTE: The original Array is altered when using sort()

var fruits = ["Banana", "Apple", "Mango"];
console.log(fruits.sort());

Numeric sorting

This outputs: Array(3) [ 1, 15, 2 ]

Notice something strange? The order is 1, 15, 2.

Thatâ€™s because sort() converts everything to string and uses lexicographical order by default.

var arr = [ 1, 2, 15 ]
arr.sort()
console.log(arr);

Sorting in correct order using a comparison function

To make it smarter, we need to pass in the custom comparison function.

It should accept two arguments and return 1, 0 or -1.

The purpose of the compare function is to define an alternative sort order.

The compare function should return a negative, zero, or positive value, depending on the arguments

Syntax

Example of a comparison function

function(a, b){return a-b}

Find the Highest (or Lowest) Array Value

Sorting

Sorting a whole array is a very inefficient method if you only want to find the highest (or lowest) value.

There are no built-in functions for finding the max or min value in an array.

However, after you have sorted an array, you can use the index to obtain the highest and lowest values.

var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){
  return a-b
});

console.log(points[0]);

Using Math.max()

Find highest value

Use Math.max.apply to find the highest number in an array

var points = [40, 100, 1, 5, 25, 10];

function myArrayMax(arr) {
    return Math.max.apply(null, arr);
}

console.log(myArrayMax(points));

Best solution - custom

Displays highest value

The fastest solution is to use a "home made" method.

This function loops through an array comparing each value with the highest value found: 

var points = [40, 100, 1, 5, 25, 10];
document.getElementById("demo").innerHTML = myArrayMax(points);

function myArrayMax(arr) {
    var len = arr.length;
    var max = -Infinity;
    while (len--) {
      if (arr[len] > max) {
        max = arr[len];
      }
    }
    return max;
}

      }
    }
    return min;
}

Sorting Object Arrays

Even if objects have properties of different data types, the sort() method can be used to sort the array.

The solution is to write a compare function to compare the property values

This sorts the car objects by year. This prints:

  • Saab 2001
  • BMW 2010
  • Volvo 2016
var cars = [
{type:"Volvo", year:2016},
{type:"Saab", year:2001},
{type:"BMW", year:2010}]

displayCars();

function myFunction() {
    cars.sort(function(a, b){return a.year - b.year});
    displayCars();
}

function displayCars() {
  document.getElementById("demo").innerHTML =
  cars[0].type + " " + cars[0].year + "<br>" +
  cars[1].type + " " + cars[1].year + "<br>" +
  cars[2].type + " " + cars[2].year;
}

Another example of sorting Object Arrays

This example sorts the objects by their title in Alpha order.

NOTE: Uppercase comes before lowercase in Alpha order. So you should always use the toLowerCase() function

const notes = [{
    title: 'my next trip',
    body: 'I would like to go to Spain'
}, {
    title: 'Habbits to work on',
    body: 'Exercise. Eating a bit better.'
}, {
    title: 'Office modification',
    body: 'Get a new seat'
}]


const sortNotes = function (notesArray) {
    //Sort the Array by passing a comparion function
    //a and b are Objects from the Array
    notesArray.sort(function (a, b) {
        //if a should come before b, return FALSE so nothing happens
        if (a.title.toLowerCase() < b.title.toLowerCase()) {
            return -1
        //if b should come before a, return true
        } else if (b.title.toLowerCase() < a.title.toLowerCase()) {
            return 1
        //if their order is the same
        } else {
            return 0
        }
    })
}

sortNotes(notes)
console.log(notes)

Sort todo list by incomplete items first

This example sorts the objects by their title in Alpha order.

NOTE: Uppercase comes before lowercase in Alpha order. So you should always use the toLowerCase() function

const todos = [{
    text: 'Order cat food',
    completed: false
}, {
    text: 'Clean kitchen',
    completed: true
}, {
    text: 'Buy food',
    completed: true
}, {
    text: 'Do work',
    completed: false
}, {
    text: 'Exercise',
    completed: true
}]

const sortTodos = function (todos) {
    todos.sort(function (a, b) {
        if (!a.completed && b.completed) {
            return -1
        } else if (!b.completed && a.completed) {
            return 1
        } else {
            return 0
        }
    })
}


sortTodos(todos)
console.log(todos)
Output

This example outputs the Objects in Alpha order by their title:

  { text: 'Do work', completed: false },
