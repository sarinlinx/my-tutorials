Running a method on an Array

reverse()

Use this syntax to run a method on an Array: ArrayName.method();

NOTE: This mutates the original Array.

myarray.reverse();

This runs a method named reverse() on the array myarray.

This first sorts in correct alpha order, then reverse it.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
console.log(fruits.sort());
console.log(fruits.reverse());

concat()

Combine 2 Arrays.

1st Array is one that is added to.

This DOES NOT mutate the original Array.

Another way of writing this is with the spread operator: [...arr, ...arr2]

const arr = ['a', 'b', 'c'];
const2 arr = ['d', 'e', 'f'];

const letters = arr.concat(arr2);
console.log(letters);
