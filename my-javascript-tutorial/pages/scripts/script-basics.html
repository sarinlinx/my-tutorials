if statement

if statement

if (condition) {
  block of code to be executed if the condition is true
}

Example

if (hour < 18) {
  greeting = "Good day";
}

if else statement

if statement

if (condition) {
  block of code to be executed if the condition is true
} else {
  block of code to be executed if the condition is false
}

Example

if (hour < 18) {
  greeting = "Good day";
} else {
  greeting = "Good evening";
}

else if statement

if statement

if (condition1) {
  block of code to be executed if condition1 is true
} else if (condition2) {
  block of code to be executed if the condition1 is false and condition2 is true
} else {
  block of code to be executed if the condition1 is false and condition2 is false
}

Example

if (time < 10) {
  greeting = "Good morning";
} else if (time < 20) {
  greeting = "Good day";
} else {
  greeting = "Good evening";
}

Ternary operator

Syntax to shorten code.

var = condition ? value_if_true : value_if_false

For example, this returns false since 1 is less than 10.

var value = ( 1 > 10 ) ? true : false

You can also use it with text. This returns the string "correct".

var value = ( 1 < 10 ) ? "correct" : "incorrect"

You can also use it with numbers. This returns the number 555.

var value = ( 1 < 10 ) ? 555 : 777

Loops

while

While the condition is true, the code from the loop body is executed.

var i = 0;

while(i<10) {
  // do something
}

do while

The loop will always execute once, then it checks the condition. If the condition is true, the code is exectued again until the condition is not true.

var i = 0;

do {
  alert(i);
  i++;
}

while (i < 5);

for

Most common loop type. All parts are put into the same location.

The variable is set: i=0

The condition to check is set: i <10

The value is altered: i++

for( i=0; i<10; i++) {
  alert(i)
}

Break

Use 'break' to exit a loop at any time you choose.

This example breaks the loop when i is equal to 3.

var i;

for (i = 0; i < 10; i++) {
  if (i === 3) { break; }
  alert(i);
}

Continue

Continue stop the current iteration and forces the loop to start the next.

This counts 0, 1, 2, 3 and stops. The negative values are skipped.

var i = -3

while( i<3 ) {
  i++

  if ( i<0 ) continue

  alert(i)
}

Labels

Break

Lables are used for nested loops. This allows you to break out of an inner loop and outer loop at the same time. Otherwise, you'd only break out of the inner loop.

This example displays the following, but breaks out before the second outer loop can begin.

  • Outer Loop 0 Inner Loop 0
  • Outer Loop 0 Inner Loop 0
  • Outer Loop 0 Inner Loop 2
  • Done!
outer:
for (var i = 0; i < 3; i++) {
  for (var j = 0; j < 3; j++) {

    alert("Outer Loop " + i + " Inner Loop " + j);

    if (j === 2) break outer;

  }
}

alert('Done!');

Continue

You can also use 'continue' with labels.

This example will display the following since the value of 1 continues the statement (skips the next value of 2).

  • Outer Loop 0 Inner Loop 0
  • Outer Loop 0 Inner Loop 1
  • Outer Loop 1 Inner Loop 0
  • Outer Loop 1 Inner Loop 1
  • Outer Loop 2 Inner Loop 0
  • Outer Loop 2 Inner Loop 1
  • Done!
outer:
for (var i = 0; i < 3; i++) {
  for (var j = 0; j < 3; j++) {

    alert("Outer Loop " + i + " Inner Loop " + j);

    if (j === 1) continue outer;

  }
}

alert('Done!');

Switch

The Switch operator checks the argument against each case. It then executes the code below the match until it meets the break operator.

This example outputs 2 and 3 because it starts at case2 and proceeds through case3.

var x = 2

switch(x) {
case 1:
alert('never executes')

case 2:
alert('x === 2')

case 3:
alert('x === 3')
}

This example outputs 2, but then breaks out of the Switch, so Case3 is never ran.

var x = 2

switch(x) {
case 1:
alert('never executes')

case 2:
alert('x === 2')
break

case 3:
alert('x === 3')
}

Switch uses === to check equality.

In this example, if you enter 0, 1, or 2, a pop-up displays your values normally.

However if you enter 3, Switch skips to the default case and states the value is unknown. This is because there are no quotes around the '3' for the case 3 statement meaning it's a string, not a number.

var arg = prompt("Enter arg?")
switch(arg) {
case '0':
case '1':
alert('One or zero')

case '2':
alert('Two')
break

case 3:
alert('Never happens')

case false:
alert('False')
break

default:
alert('Unknown value: '+arg)
}
