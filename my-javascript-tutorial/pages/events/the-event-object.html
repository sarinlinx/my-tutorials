The Event Object

Once you register and capture an event, you'll receive an event object back from the browser. The object you get might look different depending on the version of the browser you're using.

This is important to remember, because you won't always be able to use all this information.

View the Event Object

  1. First, register the event using addEventListener
  2. Pass the Event Object to a function using function(e)
  3. Load the page.
  4. Click the text 'pink'. This triggers the event.
  5. View the output in the console.log
  6. The console displays the Event Object: click { target: img#pink, buttons: 0, clientX: 14, clientY: 30, layerX: 14, layerY: 30 }
  7. You can then click the Event Object to view more if its information. 
<img src"pink.gif" "alt="pink" id="pink">

//Register an event to receive the Event Object
//Pass the Event Object to a function(e)
  document.getElementById('pink').addEventListener('click', function(e) {
    //Output the Event Object
    console.log(e);
  } , false);

type - This is just simply the type of event that you're getting. So in our case this would be a click event.

defaultPrevented - prevent default behavior from happening

Targeting Information

currenTarget - element the event was assigned to. And that's not necessarily the event you clicked on because you can assign the event to be tracked by another item.

target - the element that the event is originated from, which may differ from the element the event was assigned to.

srcElement - The source element is the actual element that fired the event.

fromElement - from and to element are kind of related to each other and they deal with mouse over and mouse out events.
