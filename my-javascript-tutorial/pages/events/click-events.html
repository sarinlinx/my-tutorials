mouse enter

mouse enter fires when the mouse enters an element

Hover over an H1 tag to view

const h1 = document.querySelector('h1');

h1.addEventListener('mouseenter', function(e){
  console.log('addEventListern: Great! You are reading the heading')
});

Attach eventListener to an element using 'on' (old way)

Use 'onmouseenter' and set it to the function.
