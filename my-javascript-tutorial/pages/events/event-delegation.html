Event delegation

Useful when elements are not already on the page at runtime. (EX: buttons added dynamically while using the app.)

This allows you to add event handlers on elements that do not already exist.

Smooth scrolling example

This adds a single even handler to the parent element of multiple sub-elements.

It then determines on what sub-element the click event occurred on.

<ul class="nav__links">
  <li>
    <a href="#section--1">Features</a>
  </li>
  <li>
    <a href="#section--2">Operations</a>
  </li>
  <li>
    <a href="#section--3">Testimonials</a>
  </li>
</ul>

<section class="section" id="section--1"></section>
<section class="section" id="section--2"></section>
<section class="section" id="section--3"></section>

/* EVENT DELEGATION */
// put eventListener on a common parent element when the event bubbles up
// 1. Add event listener to common parent element
// 2. Determine what element originated the event
document.querySelector('.nav__links').addEventListener('click', function (e) {
  // prevent hyperlink jump or URL change
  e.preventDefault();

  // e.tareget is where the event occurred
  if (e.target.classList.contains('nav__link')) {
    // obtain the specific anchor tag
    const id = e.target.getAttribute('href');
    document.querySelector(id).scrollIntoView({ behavior: 'smooth' });
  }
});
