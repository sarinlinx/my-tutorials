Maps

A Map is used to map values to keys.

Data is stored in Key Value pairs.

Keys can have any type (this different from an Object where the Keys are Strings

Creating a Map

Use new Map() to create the Map

Use .set to add values.

each new .set method updates and returns the map. You can see this in the console.log output.

//  Create map
const rest = new Map();
// add values
rest.set('name', 'Clasico Italiano');
// each new .set method updates and returns the map
rest.set(1, 'Fireze, Italy')
console.log(rest.set(2, 'Lisbon, Portugal'))

Chaining a Map

Because each new .set method returns the map, you can chain the .set method.

This example uses .set several times to continue to add data.


const rest = new Map();
rest.set('categories', ['Italian', 'Pizzeria', 'Vegetarian', 'Organic']).set('open',11).set('close', 23)
console.log(rest)

Read data from a Map

Use .get to retrieve data.

Make sure the data type is the same one you're retrieving (String, Boolean).

This returns: Clasico Italiano and We are open.

const rest = new Map();       
rest.set('name', 'Clasico Italiano');       
rest.set(true, 'We are open')

console.log(rest.get('name'))
console.log(rest.get(true))

Use Boolean to return open or closed status

This uses .get to check the time value.

It then returns true or false which prints the value text.

const rest = new Map();       
rest.set(true, 'We are open')
rest.set(false, 'We are closed')

const time = 21;
console.log(rest.get(time > rest.get('open') && time < rest.get('close')));

Check if Map contains a key

Use .has to check if a value exists.

This returns true.

const rest = new Map();       
rest.set('categories', ['Italian', 'Pizzeria', 'Vegetarian', 'Organic']).set('open', 11).set('close', 23)
console.log(rest.has('categories'))

Remove a key

Use .delete based on the key

This removes the key titled 'name'.

const rest = new Map();
rest.set('name', 'Clasico Italiano');

rest.set(true, 'We are open')
rest.set(false, 'We are closed')

rest.delete('name')
console.log(rest)

Create Map by passing in an Array

This uses an Array of Arrays to populate the Map.

This output is the same as using Object.entries(property).

This means there's an easy way to convert from Objects to Maps.

const question = new Map([
  ['question', 'What is the best programming language?'],
  [1, 'C'],
  [2, 'Java'],
  [3, 'JavaScript'],
  ['correct', 3],
  [true, 'Correct'],
  [false, 'Try again']
])

console.log(question)

Convert from Object to Map

This converts the Object to a Map.

This output is the same structure above as an Array of Arrays.

const openingHours = {
  thu: {
    open: 12,
    close: 22,
  },
  fri: {
    open: 11,
    close: 23,
  },
  sat: {
    open: 0, // Open 24 hours
    close: 24,
  },
}; 
const hoursMap = new Map(Object.entries(openingHours))

Iteration

Loop using for of

This uses a for of loop to print the values to the console

The Map containers keys and values.

Destructure those into 2 variables named key and value.

Only print if key is a number

const question = new Map([
  ['question', 'What is the best programming language?'],
  [1, 'C'],
  [2, 'Java'],
  [3, 'JavaScript'],
  ['correct', 3],
  [true, 'Correct'],
  [false, 'Try again']
])

// question
console.log(question.get('question'))

// answers
for (const [key, value] of question) {
  if (typeof key === 'number')
  console.log(`Answer ${key}: ${value}`)
}

Create a question and check its answer

Building on above, add a prompt.

Use .get to check if the answer matches the 'correct' property.

const question = new Map([
  ['question', 'What is the best programming language?'],
  [1, 'C'],
  [2, 'Java'],
  [3, 'JavaScript'],
  ['correct', 3],
  [true, 'Correct'],
  [false, 'Try again']
])

// question
console.log(question.get('question'))

// answers
for (const [key, value] of question) {
  if (typeof key === 'number')
  console.log(`Answer ${key}: ${value}`)
}

// answer
const answer = Number(prompt('Your answer'));
console.log(answer)

console.log(question.get(question.get('correct') === answer));
