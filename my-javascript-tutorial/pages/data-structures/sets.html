Sets

A Set is a collection of unique values.

They CANNOT contain duplicate info.

They can contain mixed data types.

Order is elements does not matter.

Creating a Set

Create a Set by writing: new Set()

Pass into it an Array.

This prints: Pasta, Pizza, Risotto.

The 2nd Pizza is removed since a Set cannot contain duplicates.

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
console.log(ordersSet)

Set String example

This returns: Set(6) [ "M", "y", "N", "a", "m", "e" ]

console.log(new Set(`MyName`))

Get Set size

Use .size to return the size of the Set.

This returns 3.

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
console.log(ordersSet.size)

Check if an element exists in the Set

Use .has to check if an element exists.

This returns true.

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
console.log(ordersSet.has('Pizza'))

Add an element to a Set

Use .add to check if an element exists.

This prints: Pasta, Pizza, Risotto, Bread

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
  console.log(ordersSet.add('Bread'))

Delete an element from a Set

Use .delete to remove an element.

This prints: Pasta, Pizza

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
ordersSet.delete('Risotto')
console.log(ordersSet)

Delete all elements from a Set

Use .clear to delete all elements.

This prints: Pasta, Pizza

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
ordersSet.clear()

Retrieve specific element from a Set

It's not possible to retrieve data from a Set

This is because all values are unique and not in order.

You would use an Array if you need to retrieve a specific value.

Iterating over a Set

Use for of loop.

const ordersSet = new Set(['Pasta', 'Pizza', 'Pizza', 'Risotto']);
for(const x of ordersSet)
console.log(x)

Example

Most common reason to use a Set is to remove elements from an Array.

Return only unique positions.

const staff = ['Waiter', 'Chef', 'Waiter', 'Manager', 'Chef', 'Waiter'];
// Only list the different positions
const staffUnique = new Set(staff);
console.log(staffUnique)

Add the unique values into an Array by using the spread operator

Srround Set in brackets to create an Array

Use the ... operator to unpack all values.

const staff = ['Waiter', 'Chef', 'Waiter', 'Manager', 'Chef', 'Waiter'];

const staffUnique = [...new Set(staff)];
console.log(staffUnique)
