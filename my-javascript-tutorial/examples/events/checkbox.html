Checkbox functionality

Checkbox prints true or false to console

Add label to checkbox. This allows you to click any part of the text to check it.

Look for the 'change' event which fires immediately.

Open console to view state of Checkbox. CHECKED=TRUE

<label>
  <input id="for-fun" type="checkbox"> Check me for fun
</label>
              
// look for 'change' event. This should be used for checkboxes since it fires immediately
document.querySelector('#for-fun').addEventListener('change', function (e) {
  // Display the status of the event argument's target of 'checked'
  // boolean, true if checked
  console.log(e.target.checked)
})
