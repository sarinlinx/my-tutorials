Dropdowns

Clicking the dropdown displays the value assigned to it in the console.

Checking the value prevents issues if the text is changed due to a language plugin.

<select id="filter-by">
<!-- value is used so JavaScript consistently knows which option was selected 
this allows the text of the dropdown to be adjusted in a different language and still function -->
  <option value="byEdited">Sort by last edited</option>
  <option value="byCreated">Sort by recently created</option>
  <option value="alphabetical">Sort alphabetically</option>
</select>
                  
//target dropdown and listen for its change event
document.querySelector('#filter-by').addEventListener('change', function (e) {
  // print value property of the dropdown (not the text of it since this can be changed by language plugins)
  console.log(e.target.value)
})
