Change text of button when clicked

Target button by its id

Clicking the button changes the text based off the id of the button.

Use 'querySelector' for IDs

<button id="create-note">Create Note</button>

// select the button with an ID of create-note
// (e) is the callback argument that represents the event. Info about what event happened and why
// it can be called anything, but the standard is (e)
document.querySelector('#create-note').addEventListener('click', function (e) {
    // target is a representation of the DOM element
    e.target.textContent = 'The button was clicked'
})

Target button by its CLASS

Clicking the button changes the text based off the class of the button.

Use 'querySelectorAll' for classes

<p class="note">This application was created by Andrew Mead</p>
<p class="note">Some note text</p>

<button id="remove-all">Remove All Notes</button>

//when remove button is clicked, all notes are removed
document.querySelector('#remove-all').addEventListener('click', function () {
    document.querySelectorAll('.note').forEach(function(x) {
        x.remove()
    })
})
