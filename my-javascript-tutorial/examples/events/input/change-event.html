The 'change' event

Print text to console using the 'change' event

Type text into the input field

Press the ENTER key.

The console prints all the text you just inputted.

<input id="search-text" type="text" placeholder="Filter todos">

// locate #search-text element, look for change event
// the change event is triggered after the ENTER key is pressed
document.querySelector('#search-text').addEventListener('change', function(e) {
  // print the target's 'value'. 'value' is similar to 'text-content'
  console.log(e.target.value)
})
