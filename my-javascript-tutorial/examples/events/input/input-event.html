The 'input' event

Print text to console using the 'input' event

Type text into the input field

The console immediately prints all the text you enter.

<input id="search-text" type="text" placeholder="Filter todos">

// locate #search-text element, look for input event
// input event fires on every character entered. No need to press the ENTER button
document.querySelector('#search-text').addEventListener('input', function(e) {
  // print the target's 'value'. 'value' is similar to 'text-content'
  console.log(e.target.value)
})
