Adding a new HTML element to a page

3 steps to add a new element

  1. Create a new element using document.createElement()
  2. Give the element text content using textContent
  3. Add the element to the page by using querySelector for the body, then appendChild to place it at the bottom
  1. Create a new element
  2. Give the element text content
  3. Add the element to the page
// 3 steps to add a new element
  
// create new element
const newParagraph = document.createElement('p')

// Give the new element some text content
newParagraph.textContent = 'This is a new element from JavaScript'

// add the new element to the page
// querySelector targets the body
// appenChild adds element to bottom of body
document.querySelector('body').appendChild(newParagraph)
