Scrolling to section coordinates

Create variables for the button and the section to scroll to.

Add an event listener for the click of the button.

Create a variable to store the coordinates of the section.

Use window.scrollTo() to scroll to the left and top position of that section.

You must also add window.pageXOffset and window.pageYOffset since the coordintes change as you scroll. This ensures the value stays the same.

<button class="btn--text btn--scroll-to">Learn more ↓</button>

<section class="section" id="section--1">

const btnScrollTo = document.querySelector('.btn--scroll-to');
const section1 = document.querySelector('#section--1');
  
// add event listener to button
btnScrollTo.addEventListener('click', function(e) {
  // get coordinates of element to scroll to
  const s1coords = section1.getBoundingClientRect();
  console.log(e.target.getBoundingClientRect())
  // scrolling: 1st arg is left position of s1coords: 2nd arg is top position of s1coords
  window.scrollTo(
    s1coords.left + window.pageXOffset,
    s1coords.top + window.pageYOffset);
})

Add smooth scroll

Update the scrollTo() method to the following to smoothly scroll.

window.scrollTo({
  left: s1coords.left + window.pageXOffset,
  top: s1coords.top + window.pageYOffset,
  behavior: 'smooth',
});

Modern smooth scroll

Remove the window.scrollTo() section and use scrollIntoView()

The first part is the element to scroll to.

It only needs a behavior of 'smooth'.

section1.scrollIntoView({
  behavior: 'smooth'
})
