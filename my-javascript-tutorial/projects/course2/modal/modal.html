Add JS to open modal

This code shows the modal when a button is clicked.

Add JS variables at top of script.

Use querySelectorAll to select all buttons with the same class name.

You can then create a for loop to loop over the Node List of classes found from querySelectorAll.

The openModal function is then run.

  • Modal example
'use strict';

const modal = document.querySelector('.modal');
const overlay = document.querySelector('.overlay');
const btnCloseModal = document.querySelector('.close-modal');
const btnsOpenModal = document.querySelectorAll('.show-modal');


for (let i = 0; i < btnsOpenModal.length; i++)
  btnsOpenModal[i].addEventListener('click', openModal);


const openModal = function () {
  modal.classList.remove('hidden');
  overlay.classList.remove('hidden');
};

Add JS to close modal

Below the code above, add class to add back hidden CSS whenever the overlay or close button is clicked.

  • Modal example
btnCloseModal.addEventListener('click', closeModal);
overlay.addEventListener('click', closeModal);

const closeModal = function () {
  modal.classList.add('hidden');
  overlay.classList.add('hidden');
};

Add code so ESC key closes modal

Below the code above, add if statement to close modal when ESC key is clicked.

This listens for the keydown event.

  • Modal example
document.addEventListener('keydown', function (e) {
  // console.log(e.key);

  if (e.key === 'Escape' && !modal.classList.contains('hidden')) {
    closeModal();
  }
});
